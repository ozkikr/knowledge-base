# Installation Container
## Summary
This admin guide covers running SearXNG in containerized environments with Docker or Podman. It outlines prerequisites, image sourcing, basic run commands, and lifecycle management. The page also documents volume mounts, environment variables, custom certificates, and custom image builds.
## Key Points
- Explains prerequisites and user/group setup for Docker-based deployments.
- Shows how to pull official images from Docker Hub or GHCR mirrors.
- Provides a baseline `docker run` setup with persistent config/cache volumes.
- Lists operational commands for shell access, stop/remove, and inspection.
- Documents key environment variables and extension points for custom builds.
