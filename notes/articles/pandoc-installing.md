# Pandoc - Installing pandoc
## Summary
Comprehensive installation guide for Pandoc across major operating systems and package managers, with notes on PDF toolchains and caveats.

## Key Points
- Covers Windows, macOS, Linux, BSD, Chrome OS, Conda Forge, Docker, GitHub Actions, and CI usage.
- Lists package-manager paths (Chocolatey, winget, Homebrew, MacPorts, distro packages, conda/mamba/pixi).
- Documents binary installer/tarball options and source-build fallback paths.
- Calls out limitations of statically linked binaries for Lua filters requiring C modules.
- Provides guidance for PDF generation dependencies (LaTeX distributions and related tooling).
