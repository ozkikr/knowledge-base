# Getting Started | Promptfoo
## Summary
Setup guide for Promptfoo covering initialization, YAML configuration, provider setup, test cases, assertions, and evaluation/view commands for prompt benchmarking.

## Key Points
- Initialize with a starter example or interactive setup from scratch.
- Define prompts, providers, and test variables in promptfooconfig.yaml.
- Use assertions (including rubric and JS scoring) for automated quality checks.
- Run matrix evaluations across prompts/models/tests and inspect results in web UI.
- Supports cloud APIs, local models, and custom providers/scripts.
