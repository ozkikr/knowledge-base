# Sourcebot Agents Overview

## Summary
Documentation for Sourcebot's experimental agents feature. Agents are automations that leverage indexed code to perform tasks like PR review, building on Sourcebot's code search and navigation capabilities.

## Key Points
- Experimental feature with potential breaking changes in non-major releases
- Agents leverage Sourcebot's indexed code for specific tasks
- Review Agent: AI agent that reviews PRs to identify issues
- Requires Sourcebot setup first, then agent configuration
