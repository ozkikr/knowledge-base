# AG-UI Integration - AgentPool
## Summary
Technical documentation on using AG-UI with AgentPool, focusing on consuming remote AG-UI agents, configuration, and client-side tool execution.

## Key Points
- Introduces AG-UI as an HTTP/SSE-based protocol for frontend-agent interaction.
- Describes AGUIAgent as a client for remote AG-UI servers within AgentPool.
- Shows YAML and Python setup for endpoints, auth headers, and managed startup commands.
- Explains client-side tool execution flow (tool call events, local execution, continuation).
- Demonstrates combining native and remote agents in teams for hybrid orchestration.
