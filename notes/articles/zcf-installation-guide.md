## Summary
Installation and onboarding guide for ZCF, including environment requirements, interactive setup flow, and tooling options around Claude Code and related utilities.

- Guide covers environment checks and platform-specific notes (Linux/WSL/Windows).
- Interactive menu supports full initialization, workflow import, API/MCP config, and model defaults.
- Doc explains backup/merge strategies for existing config and step-by-step first-run prompts.
