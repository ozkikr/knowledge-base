# Coding Agents & Complexity Budgets

## Summary
Lee Robinson describes migrating cursor.com from a headless CMS back to raw code and Markdown in 3 days using $260 in tokens and hundreds of coding agents. The article argues that with AI coding agents, the cost of abstractions like CMSes has never been higher.

## Key Points
