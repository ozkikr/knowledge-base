# Generating Content and Performing Tasks with Foundation Models

## Summary
Apple's documentation for the FoundationModels framework, which enables on-device AI content generation and task execution on Apple platforms. The framework provides APIs for leveraging Apple's built-in foundation models directly within apps without requiring cloud services.

## Key Points
- Part of Apple's FoundationModels framework for on-device inference
- Enables text generation and task completion using built-in models
- Runs locally on Apple devices without cloud dependency
- Integrated into the Apple developer ecosystem via Swift APIs
- Focuses on privacy-preserving AI capabilities
