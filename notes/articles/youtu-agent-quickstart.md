## Summary

- Covers repository setup using `uv`, virtual environment creation, and `.env` key configuration.
- Shows command-line usage for simple agents and multi-agent orchestration examples.
- Documents evaluation harness commands for full experiments, re-judging, and result export.
- Includes optional database/tracing configuration (e.g., PostgreSQL, Phoenix).
- Provides a pattern for custom agent config and minimal Python integration script.
