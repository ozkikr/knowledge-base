# OpenCode - Models Documentation

## Summary
Documentation for configuring LLM providers and models in OpenCode, an AI coding tool. Uses AI SDK and Models.dev to support 75+ LLM providers including local models. Recommended models include GPT 5.2, Claude Opus 4.5, Claude Sonnet 4.5, and Gemini 3 Pro.

## Key Points
- Supports 75+ providers via AI SDK and Models.dev
- `/models` command to select models; `/connect` for provider credentials
- Recommended: GPT 5.2, GPT 5.1 Codex, Claude Opus 4.5, Sonnet 4.5, Gemini 3 Pro
- Model config uses `provider_id/model_id` format (e.g., `opencode/gpt-5.1-codex`)
- Per-model options configurable (reasoning effort, thinking budget, etc.)
