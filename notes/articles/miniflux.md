# Miniflux - Minimalist and Opinionated Feed Reader

## Summary
Miniflux is a minimalist, opinionated RSS/Atom feed reader focused on readability and privacy. It's a single static binary with no external dependencies, offering paid hosting at $15/year or self-hosted deployment via Docker/RPM/Debian packages.

## Key Points
- Optimized for readability with clean layout; content-first design
- Automatically fetches full article contents from summary-only feeds
- Privacy-focused: removes pixel trackers, strips tracking params, uses media proxy, no telemetry
- Single static binary deployment; also available as Docker image
- Keyboard-driven navigation for productivity
