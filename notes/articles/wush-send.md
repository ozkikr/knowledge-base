# Wush - Send, Receive, Access

## Summary
Wush is a WireGuard-powered peer-to-peer file transfer and remote access tool. It supports unlimited file sizes with end-to-end encryption and works across command line and browser interfaces.

## Key Points
- Peer-to-peer file transfer using WireGuard protocol
- End-to-end encrypted transfers
- No file size limits
- Works between CLI and browser
- Zero-setup P2P connectivity
