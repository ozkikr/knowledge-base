# Jobkit - Docling
## Summary
This usage guide explains how to run Docling conversion as distributed jobs with Docling Jobkit. It covers local CLI execution, pipeline options, and connector-based I/O. The page also documents practical Google Drive setup steps for authenticated import/export workflows.
## Key Points
- Supports local, Ray, and Kubeflow execution pipelines.
- Uses configuration files to define conversion options, sources, and targets.
- Includes connectors for HTTP endpoints, S3, and Google Drive.
- Documents OAuth setup and token management for Google Drive access.
- Enables end-to-end processing from external storage to converted output sinks.
