# dev MCP - the best MCP server for coding with AI agents
## Summary
A tool reference for Cased Kit MCP covering package research, repository ingestion, symbol extraction, semantic/regex search, AST matching, and AI-assisted diff review.

## Key Points
- The docs define a broad toolbox for source analysis across local and remote repositories.
- Search capabilities combine lexical grep, semantic retrieval, and AST-level querying.
- Workflow tools support opening repos, extracting symbols, tracing usages, and reviewing diffs.
- The interface is designed for AI-agent orchestration through structured tool signatures.
