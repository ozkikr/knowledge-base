# Announcing Magika 1.0: Now Faster, Smarter, and Rebuilt in Rust

## Summary
Google announces Magika 1.0, a stable release of their AI-powered file type detection system. Rebuilt from the ground up in Rust, it now supports 200+ file types (up from ~100), with improved accuracy for text-based formats and native CLI, Python, and TypeScript clients.

## Key Points
- Identifies 200+ content types including data science (Jupyter, PyTorch, ONNX), modern languages (Swift, Kotlin, TypeScript), and DevOps formats (Dockerfile, TOML, HCL)
- Engine completely rewritten in Rust for maximum speed and security
- Over 1 million monthly downloads since initial open source release
- Improved accuracy for challenging text-based formats like code and config files
- Available as Rust CLI, Python module, and TypeScript module
