# CS294/194-196: Large Language Model Agents (Berkeley Fall 2024)

## Summary
A UC Berkeley graduate course on LLM agents covering reasoning, planning, tool use, infrastructure, and applications. Features guest lectures from researchers at Google DeepMind, OpenAI, AutoGen-AI, LlamaIndex, and others.

## Key Points
- Covers LLM foundations, reasoning, planning, tool use, and agent infrastructure
- Guest lecturers include Denny Zhou (DeepMind), Shunyu Yao (OpenAI), Chi Wang (AutoGen-AI)
- Applications: code generation, robotics, web automation, medical, scientific discovery
- Includes topics on privacy, safety, ethics, and multi-agent collaboration
- Lectures available on YouTube with slides and reading lists
