# Unlock AI Agent Collaboration: Convert ADK Agents for A2A

## Summary
A Google Cloud blog post providing a step-by-step guide for converting a standalone ADK agent (using MCP tools) into a fully A2A-compatible component. Uses a MultiURLBrowser agent with Firecrawl MCP tool as a practical example.

## Key Points
- Step-by-step conversion of ADK agents to A2A-compatible components
- Shows how to define core agent with MCP toolset (Step 1)
- Demonstrates establishing public identity via Agent Cards (Step 2)
- Uses Google ADK (Agent Development Kit) with Gemini models
- Bridges MCP tools and A2A protocol for multi-agent ecosystems
