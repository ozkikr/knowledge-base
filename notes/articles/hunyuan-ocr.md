# HunyuanOCR

## Summary
HunyuanOCR is Tencent's end-to-end OCR expert Vision Language Model built on Hunyuan's native multimodal architecture. With only 1B parameters, it achieves state-of-the-art results across multiple OCR benchmarks including multilingual document parsing, text spotting, information extraction, video subtitle extraction, and photo translation.

## Key Points
- Lightweight 1B parameter design achieving multiple SOTA benchmarks
- Supports complex multilingual document parsing
- Practical applications: text spotting, open-field info extraction, video subtitles, photo translation
- Built on Hunyuan's native multimodal architecture
- Available via HuggingFace Transformers with custom integration
