# Go Quickstart for ADK (Agent Development Kit)

## Summary
Official quickstart guide for Google's Agent Development Kit (ADK) in Go. Walks through creating a basic agent project that uses Gemini models and built-in Google Search tool, with step-by-step setup instructions.

## Key Points
- Requires Go 1.24.4+ and ADK Go v0.2.0+
- Uses `google.golang.org/adk` packages for agent, model, and tool definitions
- Demonstrates creating an LLM agent with Google Search tool capability
- Configures Gemini API via `GOOGLE_API_KEY` environment variable
- Provides a launcher pattern for running agents from CLI
