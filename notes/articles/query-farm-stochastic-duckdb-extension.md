# Stochastic DuckDB Extension
## Summary
Reference-style documentation for Query.Farm's DuckDB community extension that adds statistical distribution functions for probabilistic analysis directly in SQL.

## Key Points
- Installation via `INSTALL stochastic FROM community; LOAD stochastic;`.
- Provides broad continuous/discrete distribution support (normal, beta, poisson, binomial, etc.).
- Includes APIs for sampling, PDF/PMF, CDF, quantiles, hazard functions, and distribution properties.
- Documents parameter conventions and practical SQL examples for analytics and simulation.
- Enables synthetic data generation and probability workflows within DuckDB queries.
