# AssistantSidebar
## Summary
A short component doc describing a resizable side-panel layout for copilots: app content on the left, threaded chat on the right, with customizable panel sizing and behavior.

## Key Points
- Provides a two-panel co-pilot layout with resizable split panes.
- Left panel hosts application UI; right panel renders a chat Thread component.
- Designed as copied source code so teams can modify defaults and styling.
- Useful for embedding contextual AI assistance into existing product interfaces.
