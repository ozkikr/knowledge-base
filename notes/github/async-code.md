# Async Code

## Summary
Async Code lets you run multiple Claude Code or Codex CLI tasks in parallel with a Codex-style UI. It acts as a personal background coding agent with a web interface for managing concurrent tasks.

## Key Points
- Parallel task execution with Claude Code / Codex CLI
- Codex-style web UI for task management
- Docker support with dedicated Claude automation container
- Server + web frontend architecture
