# VibeTunnel

## Summary
VibeTunnel turns any browser into a terminal, letting you monitor AI agents and access your Mac terminal remotely. It proxies terminal sessions to a web browser with zero-friction setup, ideal for checking on long-running builds or agent tasks from mobile.

## Key Points
- Browser-based terminal access to your Mac
- Monitor AI agents and long-running processes remotely
- Native macOS app with menu bar integration
- Git follow mode and terminal title management
- Authentication support for secure remote access
