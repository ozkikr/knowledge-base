# Paperless-ngx

## Summary
Paperless-ngx is a community-supported document management system that transforms physical documents into a searchable online archive. It's the official successor to Paperless and Paperless-ng, with easy Docker Compose deployment and an active community.

## Key Points
- Scans, indexes, and archives documents into a searchable online system
- Official successor to Paperless and Paperless-ng projects
- Easy deployment via Docker Compose with an install script
- Live demo available at demo.paperless-ngx.com
- Sponsored by DigitalOcean, community-driven development
