# tree-sitter-markdown

## Summary
A Markdown parser for tree-sitter designed to read markdown according to the CommonMark spec with optional extensions (GFM, task lists, strikethrough, pipe tables). Primarily aimed at providing syntax highlighting for editors like Neovim and Helix.

## Key Points
- CommonMark-compliant markdown parsing for tree-sitter
- Toggleable extensions: GFM, task lists, strikethrough, pipe tables
- Available on npm, crates.io, and PyPI
- Primary use case is syntax highlighting in editors (Neovim, Helix)
- Not recommended where strict correctness is required
