# ov - Feature-rich Terminal Pager

## Summary
ov is a feature-rich terminal-based text viewer (pager) written in Go. It serves as a modern alternative to less/more with advanced features like column mode, section navigation, header support, and mouse interaction.

## Key Points
- Supports column mode with rainbow coloring, header pinning, and vertical headers
- Follow mode for tailing files with section-aware following
- Built-in search with regex patterns and filtering capabilities
- Mouse support with text selection and wheel scrolling
- Available via Homebrew, apt, nix, winget, and more
