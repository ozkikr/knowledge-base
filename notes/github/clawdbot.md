# clawdbot
## Summary
clawdbot (now redirected to openclaw/openclaw) is a personal AI assistant platform intended to run on user-owned devices. It provides a gateway/control plane with integrations across messaging channels, voice, and local tooling. The project emphasizes local-first operation, extensibility, and multi-model support.
## Key Points
- Supports many communication channels (chat apps, web chat, and extension channels).
- Includes CLI onboarding, daemonized gateway runtime, and tool-driven automation.
- Designed for personal single-user assistant workflows with configurable models.
- Documents security defaults around DM pairing and trusted sender controls.
