# Snitch

## Summary
Snitch is a friendlier alternative to `ss`/`netstat` for inspecting network connections. It provides a clean TUI and styled table output, written in Go with support for multiple themes and installation methods.

## Key Points
- Replaces ss/netstat with a human-friendly TUI for network connection inspection
- Installable via Homebrew, Go, Nix, AUR, and shell script
- Supports multiple color themes (Catppuccin, Gruvbox, Dracula, Nord, Tokyo Night, etc.)
- Home-manager module available for NixOS users
- Configurable refresh interval and DNS resolution
