# Open-Agent

## Summary
Open-Agent is an open-source alternative to Claude Agent SDK, ChatGPT Agents, and Manus. It provides a multi-agent collaboration framework where frontier models (OpenAI, Claude, Gemini) work together to complete tasks, with spec & context engineering for structured planning.

## Key Points
- Multi-agent collaboration across frontier models (OpenAI, Claude, Gemini, open-source)
- Self-hostable via Docker Compose with simple config
- Focuses on decision-making over prompt-chasing with structured planning
- Open source under Apache 2.0 license
- Web-based chat interface at open-agent.io
