# Blackfriday

## Summary
Blackfriday is a Markdown processor implemented in Go. It's paranoid about input safety, fast, supports common extensions (tables, smart punctuation), and handles all UTF-8 input. Originally translated from the C Sundown library.

## Key Points
- Safe for user-supplied input with paranoid parsing
- Supports common extensions: tables, smart punctuation, etc.
- v2 offers cleaned-up API, AST via separate Parse call, and extensible rendering
- HTML output with Smartypants extensions
- Widely used Go markdown library
