# fx

## Summary
fx is a terminal JSON viewer and processor. It provides an interactive TUI for exploring and manipulating JSON data directly in the terminal, making it easy to navigate large JSON structures and apply transformations.

## Key Points
- Terminal-based interactive JSON viewer
- JSON processing and transformation capabilities
- TUI interface for navigating complex JSON structures
- Lightweight CLI tool
- Useful for API debugging and data exploration
