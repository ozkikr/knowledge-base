# Youtu-Agent

## Summary
Youtu-Agent is a flexible, high-performance framework from Tencent for building, running, and evaluating autonomous agents. It achieves SOTA on WebWalkerQA (71.47%) and GAIA (72.8%) using open-weight models, and supports automated agent generation, experience-based learning, and end-to-end RL training.

## Key Points
- SOTA results on GAIA and WebWalkerQA with open-source models (DeepSeek-V3)
- Automated agent generation: Workflow mode and Meta-Agent mode
- Training-Free GRPO for continuous experience learning without parameter updates
- End-to-end reinforcement learning pipeline for agent training
- Over 81% tool synthesis success rate
