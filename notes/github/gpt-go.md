# gpt-go

## Summary
A tiny GPT implementation from scratch in pure Go, trained on Jules Verne books. Designed as an educational companion to Karpathy's "Neural Networks: Zero to Hero" course, with progressive git tags showing model evolution from naive to full.

## Key Points
- Simple GPT in pure Go, no dependencies on ML frameworks
- Trains in ~40 minutes on MacBook Air M3
- Progressive learning via git tags: naive, bigram, multihead, block, residual, full
- Includes detailed test file with explanations from basic neurons to self-attention
- Supports chat mode after training
