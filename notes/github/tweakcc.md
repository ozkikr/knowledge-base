# tweakcc

## Summary
A CLI tool that customizes the Claude Code experience. Allows modifying system prompts, creating custom toolsets, adding themes, custom thinking verbs/spinners, input pattern highlighters, and UI personalization. From the team behind Piebald AI.

## Key Points
- Customize all of Claude Code's system prompts
- Create custom toolsets usable via `/toolset` command
- Custom themes with graphical HSL/RGB color picker
- Input pattern highlighting with custom colors
- Manual session naming with `/title` or `/rename`
- Supports ad-hoc patches via sandboxed scripts and remote config URLs
- Has an API (npm package) since v4.0.0
