# ext-apps
## Summary
This is the official specification and SDK repository for MCP Apps extensions. It defines how MCP servers can deliver interactive UI components inside compatible chat hosts.

## Key Points
- Contains protocol docs and SDK code for MCP Apps.
- Targets embedded UI experiences in conversational AI clients.
- Documents stable and draft specification versions.
- Important reference repo for MCP ecosystem implementers.
