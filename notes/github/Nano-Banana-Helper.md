# Nano-Banana-Helper
## Summary
Nano Banana Helper is a macOS app for batch image editing workflows powered by Google's Gemini models. It focuses on orchestrating large edit jobs with queueing, cost estimation, and project-based organization.
## Key Points
- Built with Swift/SwiftUI and modern async concurrency patterns.
- Supports multi-image staging, prompt reuse, and history tracking.
- Includes cost controls such as model-tier and batch-tier settings.
- Targets high-throughput creative workflows on local desktop.
