# Planning with Files

## Summary
A Claude Code skill implementing Manus-style persistent markdown planning. It provides structured task planning via markdown files with session recovery, supporting multiple AI coding agents including Claude Code, Gemini CLI, and OpenCode.

## Key Points
- Manus-inspired persistent planning workflow using markdown files
- Session recovery when context fills up and `/clear` is used
- Supports Claude Code, Gemini CLI, Kiro, OpenCode, and more
- Active community with multiple forks adding features (multi-project, parallel execution)
- `/plan` command for easy autocomplete and status tracking
