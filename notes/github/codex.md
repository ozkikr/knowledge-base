# codex
## Summary
openai/codex is the official terminal-based coding agent from OpenAI. It provides local agent execution for code generation, editing, and workflow automation from the command line.

## Key Points
- Official CLI implementation for OpenAI's Codex coding agent experience.
- Designed for local usage in terminal-centric developer workflows.
- Distributed through npm, Homebrew, and GitHub releases.
- Contains multi-language tooling and platform-specific components.
