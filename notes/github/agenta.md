# Agenta

## Summary
Agenta is an open-source LLMOps platform providing prompt playground, prompt management, LLM evaluation, and observability in one place. It helps engineering and product teams build reliable LLM applications faster.

## Key Points
- Interactive LLM playground for comparing prompts side by side
- Prompt management with versioning for production deployment
- Built-in evaluation framework for LLM outputs
- Observability and tracing for production LLM applications
- Self-hostable with cloud option available
