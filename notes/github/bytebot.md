# Bytebot

## Summary
A self-hosted AI desktop agent that automates computer tasks through natural language commands. Unlike browser-only agents, Bytebot operates within a full containerized Linux desktop environment, enabling it to use any application, manage files, and complete multi-step workflows.

- Supports OpenAI and Gemini as AI providers
