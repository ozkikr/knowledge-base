# NeuTTS - On-Device Text-to-Speech by Neuphonic

## Summary
NeuTTS is a collection of open-source, on-device TTS speech language models with instant voice cloning. Built on LLM backbones, it delivers natural-sounding speech with real-time performance, available in multiple sizes (Air, Nano) and languages, with GGUF quantizations for edge deployment.

## Key Points
- Best-in-class realism for model size; optimized for on-device deployment
- GGUF quantizations (Q4, Q8) for phones, laptops, Raspberry Pi
- Instant voice cloning capability
- Multilingual: English, French, German, Spanish models available
- NeuTTS-Air (English) and NeuTTS-Nano (multilingual collection) variants
