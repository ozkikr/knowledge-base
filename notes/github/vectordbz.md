# VectorDBZ - Vector Database Desktop App

## Summary
A modern desktop application for exploring, managing, and analyzing vector databases. Supports multiple vector DB backends including Qdrant, Weaviate, Milvus, ChromaDB, Pinecone, pgvector, and Elasticsearch.

## Key Points
- Supports 7+ vector databases (Qdrant, Weaviate, Milvus, ChromaDB, Pinecone, pgvector, Elasticsearch)
- Desktop GUI for vector DB management and exploration
- Unified interface across different vector database providers
- Visual analysis capabilities for vector data
- Minimum version requirements documented per database
