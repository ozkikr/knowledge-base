# Worktrunk

## Summary
A CLI for git worktree management designed for running AI agents in parallel. Makes git worktrees as easy as branches with three core commands, plus quality-of-life features like hooks for automating local workflows.

## Key Points
- Simplifies git worktree UX to three core commands
- Designed for scaling 5-10+ parallel AI coding agents
- Each agent gets its own working directory to avoid conflicts
- Built in Rust, from the creator of PRQL (10k stars)
- Includes hooks for automating local workflows
