# ocrserver - Simple OCR API Server

## Summary
A simple OCR API server built on gosseract, easily deployable via Docker or Heroku. Provides HTTP endpoints for OCR processing of uploaded images.

## Key Points
- Built on gosseract Go OCR package
- One-command Docker deployment
- Heroku-compatible for cloud hosting
- REST API for image-to-text conversion
- 762 stars, lightweight and focused
