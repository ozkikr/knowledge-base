# SurfSense - OSS Alternative to NotebookLM, Perplexity, and Glean

## Summary
SurfSense is an open-source AI research agent that connects any LLM to internal knowledge sources. It integrates with search engines, Google Drive, Slack, Teams, Linear, Jira, Notion, GitHub, and many more, providing cited answers and team collaboration with RBAC.

## Key Points
- Connects any LLM to 20+ external sources (SearxNG, Google Drive, Slack, Jira, etc.)
- Cited answers like Perplexity with natural language interaction
- Supports 50+ file formats for knowledge base uploads
- Works with local LLMs (vLLM, Ollama) for privacy
- Team collaboration with role-based access control (Owner, Admin, Editor, Viewer)
