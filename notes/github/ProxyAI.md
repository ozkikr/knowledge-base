# ProxyAI

## Summary
ProxyAI is the leading open-source AI copilot plugin for JetBrains IDEs. It connects to any model (OpenAI, Anthropic, Azure, Mistral, or self-hosted) and provides chat, auto-apply, image support, web search, and git history referencing.

## Key Points
- Auto Apply: streams AI code changes into editor with diff preview
- Supports images, file/folder references, web docs, and git history in chat
- Works with OpenAI, Anthropic, Azure, Mistral, and self-hosted models
- Full offline experience possible with local models
- Alternative to GitHub Copilot for JetBrains users
