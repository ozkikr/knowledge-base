# Agent Deck - Terminal Session Manager for AI Coding Agents

## Summary
Agent Deck is a TUI (Terminal User Interface) command center for managing multiple AI coding agent sessions. It supports Claude Code, Gemini, OpenCode, Codex and more from a single interface, solving the problem of managing too many terminal tabs across projects.

## Key Points
- Single TUI to manage multiple AI coding agent sessions
- Supports Claude Code, Gemini, OpenCode, Codex, and more
- Session forking, tracking running/waiting/done states
- MCP pooling support
- Built in Go with Claude Code skill integration
