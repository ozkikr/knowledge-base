# DevDocs
## Summary
DevDocs is a UI-driven documentation ingestion platform that crawls technical docs and prepares structured outputs for AI usage. It is positioned as a way to reduce documentation research time and support MCP-based querying workflows. The stack includes frontend/backend services and Docker-first setup paths.
## Key Points
- Provides depth-controlled crawling with child URL discovery and selective extraction.
- Exports cleaned outputs in Markdown/JSON for downstream AI workflows.
- Includes MCP-oriented integration paths for local assistant tooling.
- Ships with scripts for Docker orchestration, health checks, and service debugging.
