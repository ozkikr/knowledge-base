# jjui - Jujutsu TUI

## Summary
A terminal user interface for the Jujutsu (jj) version control system. Provides interactive rebase, squash, file restore/split, bookmark management, operation log viewing, and revset auto-complete with signature help.

## Key Points
- TUI for Jujutsu VCS with keyboard-driven workflow
- Interactive rebase and squash operations
- Revset input with auto-complete and signature help
- Bookmark management, op log viewing/restoring
- File-level operations: restore, split, diff viewing
