# LlamaIndex

## Summary
The leading data framework for building LLM-powered agents over your data. Provides core indexing/retrieval capabilities with 300+ integration packages on LlamaHub for LLMs, embeddings, and vector stores.

## Key Points
- Namespaced Python library: core vs integration packages
- 300+ integrations on LlamaHub (loaders, vector stores, LLMs, embeddings)
- Starter package or customized core + plugins approach
- TypeScript version available (LlamaIndex.TS)
- Large ecosystem: LlamaHub, LlamaLab for AGI projects
