# Agentic Developer MCP

## Summary
An MCP server that wraps OpenAI's Codex CLI, making it accessible through the TeaBranch/open-responses-server middleware. Scales development into controllable agentic, recursive flows for building features bottom-up.

## Key Points
- Wraps OpenAI Codex CLI as an MCP (Model Context Protocol) server
- Three components: Codex CLI, MCP Wrapper Server, open-responses-server middleware
- Supports stdio and SSE transport modes
- Requires Node 22 for Codex compatibility
- Docker-based deployment available
