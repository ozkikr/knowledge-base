# OpenRT

## Summary
OpenRT is an open-source red teaming framework for multimodal LLMs with 37+ attack methods. It features a modular, plugin-based architecture supporting both black-box and white-box attacks across text and image vectors, with YAML-driven experiment configuration.

## Key Points
- 35+ attack methods covering black-box and white-box approaches
- Multi-modal support: text and image attack vectors
- Modular plugin-based architecture with component registry
- YAML configuration for experiment definition
- Comprehensive evaluation via keyword matching and LLM Judge
