# Hugging Face AI Sheets

## Summary
AI Sheets is an open-source tool by Hugging Face for building, enriching, and transforming datasets using AI models with no code. It can be deployed locally or on the Hub, leveraging thousands of open models via Inference Providers or local models.

## Key Points
- No-code spreadsheet interface for AI-powered dataset operations
- Access thousands of models from Hugging Face Hub
- Deploy locally via Docker/pnpm or use the hosted Space
- Supports data generation at scale via HF Jobs
- Works with open models including GPT-OSS from OpenAI
