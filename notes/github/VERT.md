# VERT
## Summary
VERT is an open-source file conversion app that performs conversion locally on-device via WebAssembly, with an optional self-hostable daemon for video workflows.

## Key Points
- Converts images, audio, documents, and video across 250+ formats.
- Prioritizes privacy by default through local conversion rather than cloud upload.
- Built with Svelte + TypeScript and includes Docker/self-hosting documentation.
- Offers an accessible web UI and a public hosted instance at vert.sh.
