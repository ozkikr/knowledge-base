# Skip
## Summary
Skip is a Swift-based cross-platform development stack aimed at building apps that run on both iOS and Android. The repository contains the Skip Xcode/SwiftPM plugin package and companion tooling hooks. It is positioned as an opinionated path for Swift developers targeting dual-platform delivery.

## Key Points
- Enables dual-platform app development using Swift and SwiftUI-centric workflows.
- Ships plugin infrastructure that works with the Skip toolchain ecosystem.
- Includes onboarding workflows (`skip checkup`, `skip create`) for faster project setup.
- Connects to broader documentation and sample projects in the Skip ecosystem.
- Designed to fit naturally into Xcode and Swift Package Manager workflows.
