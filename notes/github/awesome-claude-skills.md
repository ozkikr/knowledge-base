# Awesome Claude Skills

## Summary
A curated awesome-list of Claude Skills, resources, and tools for customizing Claude AI workflows. Skills are specialized folders containing instructions, scripts, and resources that Claude dynamically discovers and loads when relevant to tasks, using a progressive disclosure architecture.

## Key Points
- Skills use progressive disclosure: metadata loading (~100 tokens) then full instructions on demand
- Works with Claude.ai web interface, Claude Code CLI, and Claude API
- Includes official skills for documents (docx, pdf, pptx) and more
- Community-contributed skills and resources
- Inspired by awesome-list format
