# LearnChain - Learn from AI Coding Sessions

## Summary
LearnChain is a terminal-based learning tool that analyzes Codex or Claude session logs and generates interactive quizzes to reinforce coding concepts. Built with Ratatui for a polished TUI experience, distributed via npm.

## Key Points
- Parses AI-assisted coding session logs (Claude, Codex) to generate quizzes
- AI-powered quiz creation based on actual coding patterns
- Built with Ratatui (Rust) for interactive terminal UI
- Distributed via npm for cross-platform installation
- Uses OpenAI API for quiz generation
