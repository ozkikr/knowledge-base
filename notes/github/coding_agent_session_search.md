# coding_agent_session_search
## Summary
Unified CLI/TUI for indexing and searching coding-agent session history across many providers.

## Key Points
- Aggregates local session logs from tools like Codex, Claude, Gemini, Cursor, and others.
- Provides a single searchable timeline for past agent interactions.
- Useful for retrieval of prior solutions, prompts, and debugging context.
