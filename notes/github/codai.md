# Codai

## Summary
Codai is an AI coding agent for the terminal written in Go. It supports multiple LLMs (OpenAI, Anthropic, Gemini, DeepSeek, etc.) and provides context-aware code completions, refactoring, bug fixes, and documentation generation across multiple programming languages.

## Key Points
- Supports variety of LLMs including OpenAI, Anthropic, Gemini, Grok, DeepSeek
- Uses Tree-sitter for full project context summarization
- Can modify multiple files simultaneously
- Tracks token consumption per request
- Works with C#, Go, Python, Java, JavaScript, TypeScript
