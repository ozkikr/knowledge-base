# next-ai-draw-io
## Summary
next-ai-draw-io is a Next.js application that combines AI chat workflows with draw.io diagram generation/editing, enabling natural-language diagram creation and refinement.

## Key Points
- Supports diagram generation from prompts, image replication, and PDF/text-driven diagram extraction.
- Integrates with multiple model providers (OpenAI, Anthropic, Gemini, Bedrock, etc.).
- Includes an MCP server package for Claude Code, Cursor, VS Code, and related agent clients.
- Offers web demo usage plus local, Docker, and cloud deployment options.
