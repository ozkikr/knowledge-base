# Blink Shell

## Summary
Blink is a professional-grade terminal app for iOS built on Mosh and SSH. It provides stable, always-on connections with fast rendering via Chromium's HTerm, full keyboard configurability, and features like SplitView.

## Key Points
- Mosh-based for stable mobile connections that survive network changes
- Fast rendering using Chromium's HTerm engine
- Fully configurable keyboard mappings (Caps as Esc/Ctrl, etc.)
- Custom themes and fonts support
- Swipe navigation between connections, pinch to zoom, SplitView
