# mcp-trino
## Summary
Go-based MCP server that exposes Trino query and metadata capabilities to AI assistants via standardized tools.

## Key Points
- Supports STDIO and HTTP transports with optional OAuth/JWT flows.
- Provides tools for query execution plus catalog/schema/table introspection.
- Includes deployment and security guidance for production environments.
