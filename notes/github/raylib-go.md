# raylib-go
## Summary
raylib-go provides Go bindings for the raylib game development library, enabling graphics and interactive app development from Go. It bundles raylib C sources and supports cross-platform workflows with documented build requirements. The project includes examples, build tags, and guidance for desktop, Android, and related targets.

## Key Points
- Offers idiomatic Go access to raylib APIs for graphics and game programming.
- Includes many runnable examples mapped to raylib modules and use cases.
- Supports multiple backend/build-tag configurations (OpenGL, SDL, DRM, etc.).
- Documents platform-specific dependencies and cross-compilation strategies.
- Can run with or without cgo in certain Windows scenarios.
