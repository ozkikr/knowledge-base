# awesome-ai-apps
## Summary
awesome-ai-apps is a large curated collection of practical LLM and agent projects, covering starter apps through advanced workflows. It organizes examples across MCP integrations, memory systems, RAG pipelines, and framework-specific templates. The repository serves as a hands-on catalog for learning and prototyping AI application patterns.

## Key Points
- Aggregates dozens of runnable examples across agent frameworks and use cases.
- Categorizes projects into starter agents, MCP agents, memory agents, and RAG applications.
- Includes tutorials and course content to support structured learning.
- Useful as a reference hub for comparing implementation patterns across tools.
