# RAG-Anything: All-in-One Multimodal RAG Framework

## Summary
RAG-Anything is a comprehensive multimodal document processing RAG system built on LightRAG. It handles text, images, tables, equations, charts, and multimedia in a unified pipeline, eliminating the need for multiple specialized tools.

## Key Points
- All-in-one multimodal RAG supporting text, images, tables, equations
- Built on top of LightRAG for graph-based retrieval
- VLM-Enhanced Query mode for integrating images into visual language models
- Context configuration module for intelligent contextual information integration
- Supports multimodal queries across diverse document formats (PDFs, images, Office docs)
