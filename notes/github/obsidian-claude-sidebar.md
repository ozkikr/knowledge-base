# Obsidian Claude Sidebar

## Summary
An Obsidian plugin that embeds a terminal in the sidebar running Claude Code. It auto-launches Claude, supports multiple tabs, folder context menus, and a YOLO mode for skipping permissions.

## Key Points
- Embedded terminal in Obsidian sidebar with auto-launching Claude Code
- Multiple Claude instances via tabs
- Right-click folder context menu to open Claude in that directory
- YOLO mode (`--dangerously-skip-permissions`) via right-click
- Works on macOS, Linux, and Windows; requires Python 3
