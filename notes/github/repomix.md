# repomix
## Summary
Repomix packs an entire repository into a single AI-friendly artifact for LLM analysis and generation workflows. It supports multiple output formats, token counting, filtering, and security-oriented checks to make large codebases easier to share with AI tools. The project is available via CLI, website, browser extension, and container workflows.
## Key Points
- Generates consolidated repository outputs in XML, Markdown, JSON, and plain text styles.
- Includes token estimation and compression features to help fit model context limits.
- Supports remote repositories, include/exclude patterns, stdin file lists, and git metadata options.
- Integrates with common JS ecosystem tooling and provides Docker-based execution paths.
