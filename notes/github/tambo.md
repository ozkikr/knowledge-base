# tambo
## Summary
Tambo is an open-source toolkit for building agent-driven generative UI in React applications. It provides component registration, streaming prop updates, conversation state management, and MCP integration through a unified developer experience. Teams can use hosted cloud orchestration or self-host the backend stack.
## Key Points
- Lets agents choose and render registered React components with typed schemas.
- Supports both generative and persistent interactable component patterns.
- Provides `TamboProvider`, thread hooks, tool registration, and streaming state primitives.
- Includes MCP integration and support for multiple LLM providers.
