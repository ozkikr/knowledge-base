# repere
## Summary
repere is a local-first SQL data explorer for large tabular datasets that exceed spreadsheet comfort zones. It combines DuckDB-powered querying with visual pipeline and charting workflows in web and desktop modes. The tool emphasizes privacy and portability by avoiding mandatory cloud upload.

## Key Points
- Works with CSV, Parquet, XLSX, JSON, and JSONL sources.
- Provides SQL querying, transformation DAG views, and chart dashboards.
- Supports exports to multiple formats, including shareable HTML artifacts.
- Uses DuckDB-WASM on web and native DuckDB in desktop/Tauri builds.
