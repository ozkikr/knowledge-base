# AutoGen

## Summary
AutoGen is Microsoft's framework for creating multi-agent AI applications that can act autonomously or alongside humans. It supports AgentChat for conversational agents, MCP server integration, and includes AutoGen Studio for a no-code GUI experience.

## Key Points
- Python 3.10+, pip installable (autogen-agentchat + extensions)
- Supports OpenAI, Anthropic, and many other model providers
- MCP server integration (e.g., Playwright for web browsing agents)
- AutoGen Studio provides no-code GUI for building agents
- Note: Microsoft now recommends new users check out Microsoft Agent Framework; AutoGen continues receiving maintenance
