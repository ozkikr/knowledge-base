# Rainfrog

## Summary
Rainfrog is a terminal-based database tool written in Rust. It supports Postgres (tier 1), MySQL, SQLite, Redshift (tier 2), and experimental DuckDB and Oracle support, with vim-like keybindings and mouse controls.

## Key Points
- Vim-like keybindings and mouse controls for efficient navigation
- Query editor with keyword highlighting, session history, and favorites
- Multi-database support: Postgres, MySQL, SQLite, Redshift, DuckDB, Oracle
- Cross-platform including Android via Termux
- Currently in beta; name inspired by "frogs find refuge in elephant tracks"
