# Holmes (holmes-go)
## Summary
Holmes is a local diff-checking utility focused on privacy-sensitive comparisons. It supports text and structured formats so users can inspect changes without uploading data to external services. The project targets secure workflows for confidential configs and logs.

## Key Points
- Runs entirely locally with no required outbound sharing.
- Compares text, JSON, and XML with structure-aware behavior.
- Includes a browser UI and Docker-based deployment option.
- Positions itself as a security-first alternative to online diff tools.
