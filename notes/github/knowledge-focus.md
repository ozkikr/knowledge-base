# knowledge-focus
## Summary
KnowledgeFocus is a privacy-first desktop agent platform that turns local files into a searchable, conversational knowledge system. It combines a TypeScript/React frontend, Rust/Tauri system layer, and Python AI backend to support multimodal retrieval and document chat.
## Key Points
- Scans and monitors local files to keep a personal knowledge base updated.
- Supports multimodal retrieval (text, images, audio, video) and semantic search.
- Provides document chat, PDF co-reading, and knowledge card generation.
- Uses a plugin-capable desktop-agent architecture for extensibility.
