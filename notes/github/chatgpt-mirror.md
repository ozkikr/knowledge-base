# chatgpt-mirror
## Summary
chatgpt-mirror is a Go implementation of a ChatGPT mirror/proxy service oriented toward team or shared-account usage. It focuses on account isolation, user management, and administrative controls around token usage. The project presents itself as a self-hosted interface with deployment guidance and operations features.

## Key Points
- Implements user/account isolation to separate conversation history and access.
- Supports admin workflows for token management and user provisioning.
- Adds usage-control concepts (limits, windows, package-like configurations).
- Built around a Go backend with web UI and Docker-based deployment support.
