# mcpc
## Summary
MCPC is an SDK for composing and building agentic MCP servers from existing tools. It focuses on reusable tool composition, fine-tuning tool interfaces, and multi-agent orchestration through standard MCP mechanisms. The repository also includes tracing, skills support, and multiple execution modes.
## Key Points
- Provides a composition-first approach for building MCP-native agent tools.
- Supports modifying tool metadata, wrapping outputs, and workflow specialization.
- Includes built-in logging/tracing support and skill-oriented extensions.
- Targets portability and interoperability across MCP-capable clients.
