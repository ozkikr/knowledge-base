# Bleve

## Summary
Bleve is a modern text/numeric/geo-spatial/vector indexing and search library for Go. It supports a wide range of field types and query types, including approximate k-nearest neighbors via vector search and hybrid search with score fusion.

## Key Points
- Supports text, number, datetime, boolean, geopoint, geoshape, IP, and vector fields
- Query types: term, phrase, match, prefix, regexp, wildcard, fuzzy, geo spatial, vector
- Hybrid search with RRF and RSF score fusion
- tf-idf and BM25 scoring models
- Faceting support for terms, numeric ranges, and date ranges
