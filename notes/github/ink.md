# Ink

## Summary
React for interactive command-line apps. Ink provides component-based UI building using React's renderer for the terminal, with Flexbox layouts powered by Yoga. Used by Claude Code, Gemini CLI, GitHub Copilot CLI, Cloudflare Wrangler, and more.

## Key Points
- Full React renderer for terminal — all React features supported
- Flexbox layouts via Yoga with CSS-like properties
- Used by Claude Code, Gemini CLI, GitHub Copilot CLI, Cloudflare Wrangler
- Component-based: Text, Box, and custom components with hooks
- npm install ink react — drop-in React experience for CLIs
