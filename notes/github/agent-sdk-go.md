# Agent SDK Go - AI Agent Framework for Go

## Summary
A Go framework for building production-ready AI agents with multi-LLM support (OpenAI, Anthropic, Google Vertex AI), memory management, tool execution, and enterprise features. Includes MCP integration, guardrails, observability, and declarative YAML configuration.

## Key Points
- Multi-model support: OpenAI, Anthropic, Google Vertex AI (Gemini)
- Advanced memory management with buffer and vector-based retrieval
- MCP integration via HTTP and stdio transports
- Built-in guardrails, tracing, and multi-tenancy for enterprise use
- Declarative YAML agent configuration with auto-bootstrapping from prompts
