# clopus-watcher
## Summary
This link currently resolves to A2W's "Lucas" repository, a Kubernetes SRE agent that can run in report-only or autonomous remediation modes. It combines in-cluster diagnostics, Slack-driven workflows, and a lightweight dashboard for operational visibility.
## Key Points
- Supports both interactive and scheduled cluster health investigations.
- Offers autonomous-fix vs report-only operation modes.
- Integrates Slack command workflows for incident response.
- Stores run/session data in SQLite with dashboard visibility.
