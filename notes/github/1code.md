# 1code
## Summary
1Code is a desktop orchestration/UI layer for Claude Code workflows with local and remote agent execution. It combines agent sessions, git worktree isolation, and visual diff/review tooling in a Cursor-like interface. The project focuses on safer parallel coding workflows without committing directly on the main branch.

## Key Points
- Provides visual UX around Claude agent execution and tool activity.
- Runs each chat in isolated git worktrees for branch safety.
- Supports background execution and integrated terminal/git operations.
- Includes plan mode with clarifying questions and structured previews.
- Ships cross-platform support for macOS, Linux, and Windows.
