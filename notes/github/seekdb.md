# OceanBase seekdb

## Summary
An AI-native search database that unifies relational, vector, text, JSON, and GIS in a single engine. Supports hybrid search combining vector, full-text, and relational queries. Runs embedding, reranking, and LLM inference inside the database for complete RAG workflows.

## Key Points
- Unifies vector, text, structured, and semi-structured data in one engine
- Hybrid search: vector + full-text + relational in a single statement
- AI inside: embedding, reranking, LLM inference in-database
- MySQL compatible, supports embedded and single-node modes
- Apache 2.0 licensed, built on OceanBase engine
