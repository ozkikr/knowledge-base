# CodeWebChat

## Summary
An open-source, privacy-first AI coding toolkit for VS Code that helps construct prompts for chatbot smart workspaces and APIs. It supports context selection, prompt caching optimization, and works with 19+ chatbot platforms including ChatGPT, Claude, Gemini, and Copilot.

## Key Points
- Designed for prompt caching: reduces latency up to 80%, input costs up to 90%
- Supports 19+ chatbot platforms via browser extension autofill
- Context-aware prompt construction from VS Code
- Supports both remote and local (Ollama) model providers
- Available on VS Code Marketplace and Open VSX
