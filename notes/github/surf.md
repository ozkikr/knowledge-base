# Deta Surf

## Summary
Deta Surf is an open-source AI notebook that brings files and the web into your stream of thought. Built with Svelte, TypeScript, and Rust, it runs on macOS/Windows/Linux, stores data locally in open formats, and supports multi-media research with AI features.

## Key Points
- Personal AI notebook for simultaneous research and thinking across media types
- Local-first data storage using SFFS (Surf Flat File System) in open formats
- Supports YouTube notes, PDF notes, web search, and app generation
- Open source with open model choice including local models
- Cross-platform: macOS, Windows, Linux
