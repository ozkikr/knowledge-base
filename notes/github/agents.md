# LiveKit Agents - Realtime Voice AI Agent Framework

## Summary
LiveKit Agents is a Python framework for building realtime, programmable voice AI participants that run on servers. It supports conversational, multi-modal agents that can see, hear, and understand, with integrations for STT, LLM, TTS, and telephony via WebRTC.

## Key Points
- Flexible STT/LLM/TTS/Realtime API integrations ecosystem
- Built-in job scheduling and dispatch APIs for connecting users to agents
- Telephony integration for making/receiving phone calls
- Semantic turn detection using transformer models to reduce interruptions
- Native MCP support and built-in test framework with judges
