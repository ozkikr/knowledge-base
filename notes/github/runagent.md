# RunAgent

## Summary
RunAgent simplifies serverless deployment of AI agents with a powerful CLI, multi-language SDK support (Python, JS, Rust, Go, Dart, C#), and built-in agent invocation and streaming. It also offers RunAgent Pulse for scheduling and orchestration.

## Key Points
- Multi-language SDKs: Python, JavaScript, Rust, Go, Dart, C#
- Serverless agent deployment at scale
- RunAgent Pulse: self-hosted scheduling service ("Google Calendar for AI agents")
- Built-in streaming support for agent responses
- Integration with PaperFlow for arXiv monitoring workflows
