# pgsqlite

## Summary
A PostgreSQL protocol adapter for SQLite databases written in Rust. Allows PostgreSQL clients (psql, pgAdmin, DataGrip) to connect to and query SQLite databases using the PostgreSQL wire protocol. Experimental project.

## Key Points
- Enables PostgreSQL tools to work with SQLite databases transparently
- Great for rapid testing/CI-CD, feature branch deployments, and local dev
- Useful for AI agent development with isolated sandbox environments
- Supports in-memory databases for testing
- Built in Rust; available as pre-built binaries
