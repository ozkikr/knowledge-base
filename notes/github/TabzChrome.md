# TabzChrome
## Summary
TabzChrome is a Chrome extension that gives you persistent sidebar terminals connected to your local machine. It is aimed at developers who want tmux-backed shell access next to browser workflows and AI tooling.
## Key Points
- Runs real Linux shell sessions in a browser sidebar via WebSocket connectivity.
- Built for managing tmux sessions and long-running terminal workflows.
- Useful for running coding assistants (Claude Code, Codex, etc.) alongside web context.
- Focuses on persistent, always-available terminal UX inside Chrome.
