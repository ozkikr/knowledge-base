# lazymake

## Summary
Lazymake is a terminal UI for browsing and running Makefile targets. It provides fuzzy search, dependency graph visualization, variable inspection, syntax highlighting, and safety warnings for destructive commands.

## Key Points
- Interactive TUI with fuzzy search for Makefile targets
- Dependency graph visualization to understand build chains
- Variable inspector for debugging complex expansions
- Safety warnings for destructive commands (configurable)
- Installable via Homebrew or `go install`
