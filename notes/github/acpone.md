# ACPone - Multi-Agent ChatBot

## Summary
ACPone is a chatbot based on the Agent Client Protocol (ACP) that dispatches tasks to multiple AI agents (Claude Code, Codex, Gemini CLI, Kimi CLI) through a web UI. Features Vue 3 frontend with Go backend communicating via JSON-RPC.

## Key Points
- Multi-agent support with @mention routing to specific agents
- Vue 3 + TypeScript frontend with Go backend
- SSE streaming responses with session persistence
- Permission confirmation mechanism for agent actions
- Dark/light theme and i18n (Chinese/English)
