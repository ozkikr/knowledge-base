# AutoAgent: Fully-Automated and Zero-Code LLM Agent Framework

## Summary
AutoAgent enables creating tools, agents, and workflows using natural language alone. It features an agent editor for standalone agent creation and a workflow editor for multi-step agent pipelines, all through conversational interfaces without writing code.

## Key Points
- Natural language customization for creating agents, tools, and workflows
- Agent editor mode for standalone agent creation
- Workflow editor mode for multi-step agent pipelines
- Docker-based containerized agent-interactive environment
- Supports multiple LLM providers (OpenAI, DeepSeek, Anthropic, Gemini, etc.)
