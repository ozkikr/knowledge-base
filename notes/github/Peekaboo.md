# Peekaboo

## Summary
Peekaboo is a macOS CLI and optional MCP server that enables AI agents to capture screenshots, interact with UI elements, and control applications. It provides comprehensive macOS automation including clicking, typing, window management, and menu interaction.

## Key Points
- Screenshot capture with optional visual question answering via local/remote AI
- Full UI automation: click, type, press, scroll, swipe, drag elements by ID or coordinates
- Window, app, space, menu, dock, and dialog management commands
- Works as both standalone CLI and MCP server for AI agent integration
- Supports retina capture and JSON output
