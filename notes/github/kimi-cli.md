# Kimi Code CLI

## Summary
Kimi Code CLI is a terminal-based AI coding agent by MoonshotAI. It can read/edit code, execute shell commands, search/fetch web pages, and autonomously plan actions. It features a built-in shell mode and integrates with VS Code and ACP-compatible IDEs.

## Key Points
- AI agent that runs in terminal for software development and terminal operations
- Shell command mode (Ctrl-X) for running commands without leaving the agent
- VS Code extension and Agent Client Protocol (ACP) support for IDE integration
- Works with Zed and JetBrains IDEs via ACP
- Available on PyPI as `kimi-cli`
