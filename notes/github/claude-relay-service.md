# Claude Relay Service

## Summary
CRS is a self-hosted relay/proxy service for Claude Code that enables subscription sharing ("拼车") among friends. It provides multi-account management, usage tracking, and a web admin panel for transparent cost splitting.

## Key Points
- Self-hosted proxy for Claude Code API access
- Multi-account rotation with automatic failover
- Per-user API key management and token usage tracking
- Web monitoring dashboard
- Supports HTTP/SOCKS5 proxies for region-restricted access
- Security advisory: v1.1.248 and below had admin auth bypass vulnerability
