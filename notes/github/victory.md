# Victory

## Summary
Victory is a collection of composable React components for building interactive data visualizations. Maintained by Formidable Labs, it has 11.3k stars and over 8,600 commits, making it a mature and well-established charting library.

## Key Points
- Composable React components for charts and data viz
- Supports both web and React Native
- Highly customizable with a declarative API
- Large community with 11.3k GitHub stars
- Monorepo with multiple packages
