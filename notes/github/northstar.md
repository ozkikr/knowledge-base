# Northstar

## Summary
Northstar is a boilerplate for creating real-time hypermedia applications using Datastar and NATS. The stack includes Go, NATS messaging, Datastar for hypermedia, Templ for templating, and Tailwind/DaisyUI for styling, with live reload via Air + esbuild.

## Key Points
- Stack: Go + NATS + Datastar + Templ + Tailwind/DaisyUI
- Live reload out of the box with Air + esbuild
- Debugging support via Delve
- VS Code integration with launch.json and settings.json
- Docker support for deployment
