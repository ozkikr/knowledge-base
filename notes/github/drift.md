# drift
## Summary
Drift is a codebase intelligence project aimed at helping AI tools detect conventions and retain engineering decisions across sessions. The repository currently points users to a newer v2 architecture while keeping a deprecated v1 implementation. It targets IDE and CLI-based assistant workflows with memory-oriented context support.
## Key Points
- Focuses on extracting conventions and long-term project knowledge for AI usage.
- Includes MCP-server positioning and offline CLI usage scenarios.
- Repo is split between deprecated v1 and active v2 codebases.
- Current direction centers on Rust/TypeScript monorepo development.
