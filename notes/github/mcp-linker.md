# mcp-linker
## Summary
MCP Linker is a desktop app for discovering, adding, and syncing MCP server configurations across AI clients. It emphasizes quick setup and cross-client compatibility so users can manage tool integrations from one place.

## Key Points
- Provides one-click MCP server installation and configuration flows.
- Supports multiple clients including Claude, Cursor, VS Code, Windsurf, and Codex.
- Includes a built-in MCP marketplace and optional cloud sync features.
- Implemented primarily with TypeScript and Rust.
