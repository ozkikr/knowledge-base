# chess-3
## Summary
chess-3 is a UCI-compatible chess engine written in Go and intended for use with external chess GUIs. It is a continuation of earlier chess engine iterations by the same author. The repository is focused on engine logic, search/evaluation implementation, and competitive play compatibility.
## Key Points
- Implements a standalone command-line engine with UCI protocol support.
- Designed to run through third-party GUIs and tournament tooling.
- Contains dedicated modules for move generation, evaluation, and search.
- Documents competitive rating context and version progression.
