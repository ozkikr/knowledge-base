# GrantFlow.AI

## Summary
GrantFlow.AI is an intelligent grant management platform that automates discovery, planning, and application workflows for researchers and institutions. It combines RAG-powered content generation, real-time CRDT collaboration, and vector search across documents.

## Key Points
- Automated grant opportunity monitoring with targeted notifications
- Real-time collaborative document editing via Y.js CRDT synchronization
- RAG pipeline with Kreuzberg for context-aware grant application generation
- Full-stack monorepo: Next.js 15 frontend, Litestar API, PostgreSQL + pgvector
- Multi-tenant with role-based access; Discord/webhook integrations
