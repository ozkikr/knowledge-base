# Data Formulator

## Summary
An AI-powered data visualization tool by Microsoft that lets users explore data with visualizations powered by AI agents. Supports loading diverse data sources, agent-driven exploration, interactive verification of AI-generated results, and report creation.

## Key Points
- AI agent mode for hands-off data exploration or interactive control
- Connects to live data (URLs, databases) with auto-refresh
- Supports multiple chart types including US Map and Pie Chart
- Community data loaders: BigQuery, MySQL, Postgres, MongoDB
- Generates verifiable reports grounded in data
