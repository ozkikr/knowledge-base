# Browser Use

## Summary
Python framework that makes websites accessible for AI agents, enabling automated browser interaction. Supports cloud-hosted stealth browsers, sandbox deployments, and a CLI for persistent browser sessions. Integrates with any LLM provider.

## Key Points
- AI agent-driven browser automation with Chromium
- Cloud option for stealth, scalable browser sessions
- CLI for persistent browser control (open, click, type, screenshot)
- Claude Code skill available for direct integration
- Sandbox deployment with built-in persistence, auth, and cookies
