# Quests

## Summary
Quests is an open-source desktop app builder that uses any LLM to build and run applications locally on your machine. It supports BYOK (bring your own key) with 20+ AI providers and includes a coding agent with real-time linting.

## Key Points
- BYOK support: OpenAI, Anthropic, Google, OpenRouter, Ollama, and 20+ more
- Built-in version control with restoration
- Multiple full-stack projects simultaneously
- Evals system for comparing model outputs
- Templates for React, Angular, Astro, Svelte, and more
- Apache 2.0 licensed
