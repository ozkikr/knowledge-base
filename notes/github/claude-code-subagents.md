# Claude Code Subagents Collection

## Summary
A comprehensive collection of 100+ production-ready specialized subagents for Claude Code. Each subagent is an expert in a specific domain (languages, frameworks, DevOps, etc.) and is automatically invoked based on context or explicitly called.

## Key Points
- 100+ specialized subagents covering programming languages, frameworks, and tools
- Domain experts: Python, Go, Rust, Java, TypeScript, Bash, and many more
- Configured with specific Claude models based on task complexity
- Context-aware automatic invocation or explicit calling
- Covers full development lifecycle: coding, testing, DevOps, security
