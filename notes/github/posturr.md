# Posturr
## Summary
Posturr is a macOS posture assistant that blurs the screen when slouching is detected. It supports both camera-based tracking and AirPods motion-sensor tracking, with all processing performed locally. The app is designed as a lightweight, privacy-conscious ergonomic reminder.

## Key Points
- Tracks posture in real time using Vision-based camera input or AirPods motion data.
- Applies progressive blur as behavioral feedback when posture degrades.
- Supports multi-display setups and menu bar control workflows.
- Includes calibration and sensitivity settings for personalized behavior.
- Prioritizes local processing with no account or cloud dependency.
