# CodeRunner

## Summary
CodeRunner is an MCP server that executes AI-generated code in a sandboxed environment on Mac using Apple's native containers. It enables local file processing with remote LLMs without uploading data to the cloud.

## Key Points
- MCP server for sandboxed code execution on macOS with Apple Silicon
- Process local files (videos, images, docs) without uploading to cloud
- Pre-configured Python environment with auto-install of dependencies
- Integrates with Claude Desktop and other MCP-compatible tools
- LLM generates code that runs locally, returns results directly
