# Flux (photon-ai)

## Summary
Flux is an open-source CLI tool for deploying LangChain agents to iMessage in seconds. Developers can build agents, test locally, and deploy to production with a single command, enabling iMessage-based interaction with AI agents.

## Key Points
- Deploy LangChain agents to iMessage with one command
- Local testing mode via terminal before production deployment
- Phone number authentication via iMessage
- Automatic agent validation in CLI
- Available via npx (no install needed) or global npm install
