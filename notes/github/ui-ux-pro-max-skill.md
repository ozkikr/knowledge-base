# ui-ux-pro-max-skill
## Summary
ui-ux-pro-max-skill is an AI-oriented design intelligence package focused on generating UI/UX direction and design systems for different product types. It combines style libraries, palette recommendations, typography pairings, and industry-oriented heuristics. The repo positions itself as a reusable skill layer for coding assistants and design-aware workflows.
## Key Points
- Provides structured style, color, typography, and layout recommendations.
- Introduces rule-based “reasoning” for industry-specific design guidance.
- Supports integration paths for multiple coding assistants and CLI tooling.
- Includes curated anti-pattern checks and delivery quality checklists.
