# Trieve

## Summary
Trieve is an all-in-one platform for search, recommendations, RAG, and analytics offered via API. It combines semantic dense vector search, typo-tolerant full-text/neural search, and sub-sentence highlighting with self-hosting options for AWS, GCP, Kubernetes, and Docker Compose.

## Key Points
- Semantic vector search via OpenAI/Jina embeddings + Qdrant, plus neural sparse-vector search via SPLADE
- Sub-sentence highlighting for enhanced search UX
- Recommendation engine for finding similar chunks or grouped files
- Self-hostable on AWS, GCP, Kubernetes, or Docker Compose
- MCP server available for AI tool integration
