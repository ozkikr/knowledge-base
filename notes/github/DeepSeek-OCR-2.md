# DeepSeek-OCR-2
## Summary
DeepSeek-OCR-2 is a document OCR model project introducing a visual causal flow approach for structured extraction tasks. The repository includes environment setup and inference examples for both vLLM and Transformers paths. It is aimed at converting documents to markdown and handling layout-aware OCR workflows.
## Key Points
- Provides model artifacts and references to paper and arXiv publication.
- Supports both vLLM and Hugging Face Transformers inference pipelines.
- Includes prompt patterns for layout-aware markdown extraction and free OCR modes.
- Documents environment requirements and benchmarking-oriented execution scripts.
