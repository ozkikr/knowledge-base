## Summary
wcgw is an MCP server with tightly integrated shell and code editing tools, empowering chat applications to code, build, and run on your local machine. It provides a fully interactive shell experience with keystroke control and a VS Code extension.

## Key Points
- Fully interactive shell with keystroke support (including Ctrl sequences)
- VS Code extension attaches agent's shell in your editor
- Modes: architect, code-writer, and full wcgw mode
- Syntax checking on file writes and edits
- Context saving tool for task checkpoints and knowledge transfer
