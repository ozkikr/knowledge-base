# Clipper.js

## Summary
Clipper is a Node.js CLI tool for clipping web content and converting it to Markdown. It uses Mozilla's Readability and Turndown libraries, offering both single-page clipping and site crawling capabilities.

## Key Points
- Converts HTML to Markdown from URLs, files, or directories
- Crawl mode for scraping entire sites (uses Playwright)
- Output formats: Markdown or JSON/JSONL for dataset creation
- Can also convert PDFs to Markdown via poppler integration
- Install globally via npm (`@philschmid/clipper`)
