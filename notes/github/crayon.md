# Crayon

## Summary
Generative UI SDK for building agentic UI interfaces beyond text. Provides extensible React components, lightweight state management, and hooks for creating rich AI-powered interfaces that integrate with any backend.

## Key Points
- React-based UI framework for AI agent interfaces
- Template system for mapping agent responses to custom components
- Packages: `react-core` (state/hooks) and `react-ui` (components)
- MIT licensed, by Thesys
- Designed for agentic UIs beyond chat/text
