# lazygit

## Summary
A simple terminal UI for git commands. Hugely popular open-source project that provides a keyboard-driven interface for all common git operations including staging, committing, branching, rebasing, and more. Widely sponsored and actively maintained.

## Key Points
- Terminal UI for git with intuitive keyboard shortcuts
- Covers staging, committing, branching, merging, rebasing
- Large community with many contributors and sponsors
- Cross-platform (macOS, Linux, Windows)
- One of the most popular git TUI tools
