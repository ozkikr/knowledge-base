# DeepChat

## Summary
A feature-rich open-source AI agent platform that unifies models, tools, and agents. It supports multi-LLM chat, MCP tool calling, and ACP agent integration as a desktop application.

## Key Points
- Multi-LLM chat supporting OpenAI, Gemini, Anthropic API formats
- MCP tool calling and ACP agent integration
- Open-source desktop application (Electron-based)
- Supports multiple model providers
- Active community with rapid development
