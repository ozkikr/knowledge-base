# runvnc/mindroot
## Summary
MindRoot is a plugin-based Python framework and web UI for building and deploying AI agents, tools, and app-like workflows. It combines an extensible plugin registry with configurable agents and operational controls through an admin interface. The platform supports both interactive web use and API/SDK integration for programmatic agent execution.

## Key Points
- Provides a customizable UI and plugin architecture for agent capabilities.
- Supports installation and management of model, tool, and knowledgebase plugins.
- Includes admin features for agent command toggles and system configuration.
- Offers REST API and a Python SDK for integration into external applications.
- Positions itself as a flexible framework for composing full AI web apps.
