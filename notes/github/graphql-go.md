# graphql-go

## Summary
graphql-go is a Go GraphQL server library focused on ease of use. It provides full support for the October 2021 GraphQL specification with idiomatic Go packages, including schema type-checking against resolvers, parallel execution, and subscription support.

## Key Points
- Full October 2021 GraphQL spec support
- Minimal API with resolvers matched by method sets (interfaces or structs)
- Parallel execution of resolvers with panic handling
- Built-in support for OpenTelemetry and OpenTracing
- Field selection inspection to avoid N+1 query problems
