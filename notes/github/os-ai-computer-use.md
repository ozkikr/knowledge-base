# OS AI Computer Use

## Summary
AI-powered desktop application that lets Claude control your OS through mouse and keyboard automation. Cross-platform Flutter app with a chat-based interface, real-time cost tracking, and secure API key storage in system keychain.

## Key Points
- Cross-platform desktop app (macOS, Windows, Linux, Web) built with Flutter
- Uses Anthropic Claude API for computer automation
- Chat-based interface with visual feedback
- Secure API key storage in system keychain
- Real-time cost tracking for API usage
