# Tailsnitch

## Summary
Tailsnitch is a security auditor for Tailscale configurations. It scans your tailnet for 50+ misconfigurations, overly permissive access controls, and security best practice violations, with an interactive fix mode.

## Key Points
- Scans for 50+ Tailscale misconfigurations and security issues
- Severity filtering (high/medium/low)
- Interactive fix mode (`--fix`) for remediation
- Supports OAuth client and API key authentication
- Checks ACL policy, devices, DNS, and auth keys
