# shot-scraper

## Summary
shot-scraper is a command-line utility by Simon Willison for taking automated screenshots of websites. Built on Playwright, it can capture full pages or specific elements and is useful for documentation, testing, and web scraping workflows.

## Key Points
- CLI tool for automated website screenshots powered by Playwright
- Can target specific CSS selectors or full pages
- Supports JavaScript execution before screenshot capture
- Multi-shot mode for batch screenshots from YAML config
- Apache-2.0 licensed; 2.3k+ stars
