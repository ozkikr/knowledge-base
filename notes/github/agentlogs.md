# agentlogs
## Summary
AgentLogs is an open-source platform for capturing and analyzing coding-agent transcripts, with team dashboards and git-linked traceability. It helps organizations make AI-assisted development workflows visible, shareable, and auditable. The project supports cloud and self-hosted deployment options.

## Key Points
- Captures transcripts from several coding agents and syncs them into a shared workspace.
- Links sessions to commits to improve provenance and code-review context.
- Provides team-level observability on model/agent usage and activity patterns.
- Supports plugin/CLI workflows for multiple agent ecosystems.
- Can be self-hosted with documented local and Cloudflare deployment paths.
