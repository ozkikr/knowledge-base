# Retain
## Summary
Retain is a native macOS app for consolidating conversations from multiple AI tools into one searchable local knowledge base. It focuses on preserving user preferences and corrections across tools by extracting learnings from chat history. The project is local-first and includes export workflows to help assistants retain persistent context.
## Key Points
- Aggregates conversations from CLI tools and web UIs (including Claude, Codex, Cursor, and ChatGPT).
- Provides full-text search over large conversation histories with local SQLite storage.
- Extracts preferences/corrections and can export approved learnings to CLAUDE.md.
- Includes menu bar support and Sparkle-based auto-update flow for macOS distribution.
