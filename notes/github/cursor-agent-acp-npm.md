# cursor-agent-acp-npm
## Summary
cursor-agent-acp-npm is a full Agent Client Protocol adapter for Cursor CLI, intended for ACP-compatible editors like Zed. It bridges Cursor agent capabilities through standard stdio transport with strong schema compliance. The implementation emphasizes production readiness, test coverage, and secure tool execution paths.
## Key Points
- Implements ACP session lifecycle, prompt handling, and streaming updates.
- Uses official ACP SDK types for strict protocol compatibility.
- Includes filesystem/terminal tooling, validation, and security controls.
- Supports integration patterns for Zed and other editor environments.
