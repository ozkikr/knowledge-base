# Katana - Next-Generation Web Crawling and Spidering Framework

## Summary
Katana by ProjectDiscovery is a fast, configurable web crawling and spidering framework written in Go. It supports both standard and headless browser modes, JavaScript parsing, automatic form filling, and flexible scope control for security reconnaissance.

## Key Points
- Standard and headless (Chrome) crawling modes with JavaScript parsing
- Customizable automatic form filling and scope control (regex/preconfigured)
- Flexible I/O: STDIN/URL/LIST input, STDOUT/FILE/JSON output
- Available via Go install, Docker, and prebuilt binaries
- Part of the ProjectDiscovery security toolchain
