# Gemini CLI

## Summary
Gemini CLI is Google's open-source AI agent that brings Gemini models directly into the terminal. It offers a generous free tier (60 req/min, 1000 req/day) with access to Gemini 3 models and a 1M token context window.

## Key Points
- Free tier with personal Google account: 60 requests/min, 1,000 requests/day
- Built-in tools: Google Search grounding, file operations, shell commands, web fetching
- MCP (Model Context Protocol) support for custom integrations
- Installable via npx, npm, Homebrew, MacPorts, or Conda
- Apache 2.0 licensed
