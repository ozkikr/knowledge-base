# NotebookLM-Chrome
## Summary
NotebookLM-Chrome (FolioLM) is a browser extension for collecting and querying information from tabs, bookmarks, and browsing history with AI. It is designed to create notebook-style source collections with chat and transformation workflows.

## Key Points
- Collects sources from browser context and organizes them into notebooks.
- Supports AI chat over collected sources with citation-oriented workflows.
- Includes transformation templates for study, analysis, and content drafting.
- Implements a modern extension stack centered on TypeScript and web UI assets.
