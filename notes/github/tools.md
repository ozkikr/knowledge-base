# tools
## Summary
simonw/tools is a large collection of practical utility tools, many generated or assisted by LLMs. It spans lightweight web tools, scripts, and experiment-style utilities with an emphasis on usefulness and iteration speed. The repository acts as a living sandbox for applied AI-assisted software creation.
## Key Points
- Contains a wide variety of single-purpose tools implemented across HTML/JS and Python.
- Demonstrates pragmatic LLM-assisted development workflows at scale.
- Organized for discoverability with per-tool docs and generated pages.
- Useful as inspiration for rapid tool prototyping and prompt-to-utility workflows.
