# SkyPilot

## Summary
SkyPilot is a system for running, managing, and scaling AI workloads on any AI infrastructure. It provides a unified interface for AI teams to run jobs across Kubernetes, 20+ clouds, or on-prem, while giving infra teams a control plane with advanced scheduling, scaling, and orchestration.

## Key Points
- Supports 20+ cloud providers and Kubernetes with a single unified interface
- Features multi-cloud pools for running batch inference on managed warm workers across clouds
- Enables training and serving of large models (GPT-OSS, Llama, DeepSeek, etc.) with simple commands
- Managed jobs with automatic failure recovery and spot instance support
- Active development with features like RL training support (SkyRL, veRL) and large-scale LLM training (TorchTitan)
