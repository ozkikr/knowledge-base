# Async Server

## Summary
An open-source developer tool combining AI coding with task management and code review. Integrates Claude Code + Linear + GitHub PRs into one opinionated workflow, executing code changes in isolated cloud environments.

## Key Points
- Automatically researches coding tasks before execution, asks clarifying questions
- Executes in isolated Google Cloud Run environments, not local
- Breaks work into reviewable subtasks with stacked diffs
- Full workflow from GitHub issue to merged PR
- Tech stack: FastAPI, Claude Code, GCP, Firebase Firestore
