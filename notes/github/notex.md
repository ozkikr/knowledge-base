# Notex

## Summary
Notex is a privacy-first, open-source alternative to Google's NotebookLM. It's an AI-powered knowledge management app that creates intelligent notebooks from documents, supporting multiple transformations like summaries, FAQs, quizzes, mindmaps, and podcast scripts.

## Key Points
- Supports PDFs, text, Markdown, DOCX, and HTML document sources
- AI-powered chat for Q&A based on uploaded sources
- Generates summaries, FAQs, study guides, timelines, glossaries, quizzes, mindmaps, infographics, and podcasts
- Local SQLite storage with optional cloud backends for full privacy
- Multi-model support: OpenAI, Ollama, and compatible APIs
