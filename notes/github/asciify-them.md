# asciify-them
## Summary
asciify-them is a Python package and CLI for converting images into ASCII renderings in the terminal. It supports both color and black-and-white output, optional edge detection, and configurable character sets. The project is aimed at terminal-native image rendering and creative text-mode workflows.
## Key Points
- Works as both a library API and a command-line tool.
- Provides presets and custom charset control for different visual styles.
- Includes optional Sobel/Canny edge-aware rendering modes.
- Supports saving output to files for reuse in other tools like ansee.
