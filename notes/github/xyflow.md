# xyflow (React Flow / Svelte Flow)

## Summary
Powerful open-source libraries for building node-based UIs with React or Svelte. The mono repo contains React Flow 12, React Flow 11, Svelte Flow, and a shared helper library. Ready out-of-the-box and infinitely customizable, widely used for workflow builders, diagram editors, and visual programming interfaces.

## Key Points
- Two frameworks: React Flow (@xyflow/react) and Svelte Flow (@xyflow/svelte)
- Shared core library (@xyflow/system) powers both implementations
- MIT licensed with optional Pro sponsorship for commercial use
- Includes built-in components: MiniMap, Controls, Background
- Supports custom nodes, edges, and interactions
