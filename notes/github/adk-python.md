# Google Agent Development Kit (ADK) - Python

## Summary
Google's open-source, code-first Python framework for building, evaluating, and deploying sophisticated AI agents. While optimized for Gemini, it's model-agnostic, deployment-agnostic, and compatible with other frameworks.

## Key Points
- Flexible, modular framework applying software dev principles to AI agent creation
- Optimized for Gemini but model-agnostic and deployment-agnostic
- Features: custom service registration, session rewind, code execution sandbox
- Available in Python, Java, Go, and Web variants
- Supports MCP toolsets and A2A protocol integration
