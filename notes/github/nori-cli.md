# Nori CLI

## Summary
Nori is a local AI coding agent CLI that lets you switch between Claude, Gemini, and Codex from a single native TUI. Built in Rust with Ratatui for fast incremental rendering, it reuses existing auth from other CLI tools and supports session persistence and MCP integration.

## Key Points
- Multi-provider: Anthropic Claude Code, Google Gemini, OpenAI Codex from one CLI
- Built in Rust with Ratatui for fast TUI rendering
- Reuses existing authenticated sessions from other CLI tools
- Planned features: sandboxed execution, MCP integration, multi-agent orchestration
- Install via npm (`nori-ai-cli`) or GitHub releases
