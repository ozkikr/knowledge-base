# mcpc (MCP CLI)

## Summary
A universal CLI client for the Model Context Protocol (MCP) that maps MCP operations to intuitive commands. Supports persistent sessions, stdio/HTTP transport, OAuth 2.1, JSON output for scripting, and proxy mode for AI sandboxes.

## Key Points
- Works with any MCP server over Streamable HTTP or stdio
- Persistent sessions with multiple simultaneous server connections
- Full OAuth 2.1 support with OS keychain credential storage
- JSON code mode for integration with jq and scripting
- MCP proxy server for secure AI sandbox access
