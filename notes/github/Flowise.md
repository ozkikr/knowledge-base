# Flowise

## Summary
Flowise is a visual drag-and-drop tool for building AI agents and LLM workflows. It provides a low-code interface built on a Node.js backend with a React frontend, supporting Docker deployment and extensive third-party integrations.

## Key Points
- Visual drag-and-drop UI for building AI agents and chains
- Quick start with `npx flowise start`
- Monorepo with server, UI, components, and API docs modules
- Docker and Docker Compose support
- Self-hostable with cloud option available
