# quotio
## Summary
Quotio is a native macOS app for managing multiple AI provider accounts and monitoring usage quotas for coding-agent workflows. It serves as a control surface for CLIProxyAPI, helping users route requests and configure agent tools. The project emphasizes account orchestration, failover strategies, and quick operational visibility from the menu bar.
## Key Points
- Aggregates multiple AI providers and account auth modes in one local app.
- Adds real-time quota and traffic visibility for CLI coding workflows.
- Supports one-click configuration for common coding-agent CLIs.
- Includes menu bar controls, notifications, and fallback/routing modes.
