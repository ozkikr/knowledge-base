# mcp-google-sheets

## Summary
A Python-based MCP server that bridges MCP-compatible AI clients (like Claude Desktop) with the Google Sheets API. It enables creating and modifying spreadsheets through AI-driven automation workflows using Google Drive and Google Sheets integration.
