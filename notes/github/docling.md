# Docling

## Summary
Docling simplifies document processing by parsing diverse formats (PDF, DOCX, PPTX, images, audio, etc.) with advanced PDF understanding and seamless integrations with the gen AI ecosystem including LangChain, LlamaIndex, and Crew AI.

## Key Points
- Parses multiple formats: PDF, DOCX, PPTX, XLSX, HTML, images, audio, LaTeX
- Advanced PDF understanding: layout, reading order, tables, code, formulas
- Unified DoclingDocument representation with exports to Markdown, HTML, JSON
- Local execution for sensitive/air-gapped environments
- MCP server, CLI, OCR support, and Visual Language Model integration
