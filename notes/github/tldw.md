# tldw - Too Long, Didn't Watch

## Summary
tldw_server is an open-source, API-first platform for ingesting media, transcribing, and analyzing content using 16+ LLM providers. It serves as a personal research multi-tool and open-source alternative to NotebookLM.

## Key Points
- API-first media analysis and research platform
- Supports 16+ LLM providers and OpenAI-compatible APIs for chat, embeddings, and evals
- Processes video, audio, and text media with transcription
- Features chatbook tools and resource governor configuration
- PostgreSQL content mode with Docker Compose deployment
