# sanity
## Summary
Sanity is the open-source codebase for Sanity Studio, a customizable content workspace for structured content systems. It is designed for teams building composable digital experiences with flexible schemas and plugin-driven extensions. The project pairs a React-based studio with Sanity's hosted content infrastructure.
## Key Points
- Powers Sanity Studio for structured content authoring workflows.
- Highly customizable via JavaScript/TypeScript and plugin architecture.
- Supports advanced editing interfaces, validation, and theming.
- Integrates with Sanity Content Lake for real-time hosted content.
- Used as a foundation for headless/composable CMS setups.
