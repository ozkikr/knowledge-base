# qq15725/modern-screenshot
## Summary
modern-screenshot is a frontend library for generating images from DOM nodes using HTML5 canvas and SVG techniques. It is positioned as a fast, modernized alternative/fork of html-to-image. The project exposes convenient APIs for exporting PNG, SVG, JPEG, WebP, blobs, and canvas/image objects.

## Key Points
- Converts DOM elements into multiple output formats including PNG, SVG, JPEG, WebP, Blob, and Canvas.
- Supports browser, module, and CDN usage patterns.
- Includes reusable context/worker APIs for repeated high-frequency captures.
- Exposes configurable conversion options and progress/debug hooks.
- Designed for practical screenshot/export workflows in web apps.
