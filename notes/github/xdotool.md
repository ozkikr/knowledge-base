# xdotool - X11 Automation Tool

## Summary
Command-line tool for simulating keyboard input, mouse activity, and managing windows on X11. Uses X11's XTEST extension and Xlib functions. Also provides libxdo, a C library for programmatic use.

## Key Points
- Simulates keyboard input and mouse activity
- Window management: move, resize, hide, modify properties
- Desktop switching and window-to-desktop movement
- Available in most Linux/BSD/macOS package managers
- Does NOT work on Wayland (X11 only)
- Includes libxdo C library for integration
