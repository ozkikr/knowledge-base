# Attu

## Summary
Attu is the premier GUI administration tool for the Milvus vector database by Zilliz. It provides visual schema design, data management with smart filters, interactive vector similarity search, RBAC management, and real-time monitoring.

## Key Points
- Visual schema designer for Milvus collections
- Smart data operations with filtering, inline editing, import/export
- Interactive vector similarity search with visualization
- Enterprise features: RBAC, privilege management, slow query analysis
- Available as Docker, Kubernetes, desktop app, or behind nginx proxy
