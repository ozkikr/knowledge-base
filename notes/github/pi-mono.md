# Pi Monorepo
## Summary
pi-mono is an AI agent toolkit monorepo centered on coding-agent workflows and multi-provider LLM access. It bundles packages for agent runtime, terminal/web UI components, Slack integration, and GPU pod operations. The project is aimed at teams building practical agent systems from shared components.
## Key Points
- Contains @mariozechner/pi-* packages for LLM APIs, agent core, coding CLI, and chat UI.
- Includes a Slack bot integration that delegates tasks to the coding agent.
- Adds tooling for managing vLLM deployments via pi-pods.
- Provides unified repo scripts for build, checks, and tests across packages.
