# Wegent

## Summary
Wegent is an open-source AI-native operating system for defining, organizing, and running intelligent agent teams. It features a chat agent with multi-model support and a code agent, with capabilities like group chat, sandboxed execution, MCP tools, and long-term memory via mem0.

## Key Points
- Multi-model support: Claude, OpenAI, Gemini, DeepSeek, GLM
- Chat agent with group chat, attachments, follow-up mode, and error correction
- Code agent for sandboxed command execution (E2B compatible)
- Long-term memory via mem0 integration
- Extensible with custom prompts, MCP tools, and Skills
