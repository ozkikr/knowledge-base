# Data Copilot
## Summary
Data Copilot is a workflow repo for building analysis using BigQuery INFORMATION_SCHEMA and GitHub Copilot premium models. It helps teams bootstrap analysis contexts, pull schema/query metadata, and iterate with AI assistance in IDEs. The project focuses on practical analyst workflows rather than a general-purpose framework.
## Key Points
- Uses environment-driven setup for project and region configuration.
- Includes helper targets to initialize sample datasets and sync query structures.
- Designed for natural-language prompting with Copilot agent mode and Claude models.
- Includes prompt examples to accelerate exploratory analysis.
