# go-critic

## Summary
A highly extensible, opinionated Go source code linter for code audit. Provides almost 100 diagnostics for bugs, performance, and style issues, with support for dynamic rules without recompilation.

## Key Points
- ~100 diagnostics covering bugs, performance, and style
- Extensible with dynamic rules (ruleguard) without recompilation
- Integrates with golangci-lint
- Available via Homebrew, MacPorts, or `go install`
- Self-documented with `go-critic doc` command
