# AgentOps

## Summary
A Python SDK for AI agent monitoring, LLM cost tracking, benchmarking, and debugging. Provides session replay analytics, integrates natively with CrewAI, Agno, OpenAI Agents SDK, LangGraph, and more. Open source and self-hostable.

## Key Points
- 2-line integration for automatic LLM call analytics
- Step-by-step agent execution graph replays
- LLM cost management across foundation model providers
- Native integrations with major agent frameworks (CrewAI, AG2, Agno, LangGraph)
- Fully self-hostable (Dashboard + API backend)
