# dcv - Docker Container Viewer

## Summary
DCV is a TUI tool for monitoring Docker containers and Docker Compose applications. It provides vim-style keybindings and supports viewing containers, images, networks, volumes, browsing files inside containers, executing shell commands, and real-time log streaming.

## Key Points
- TUI for Docker containers, images, networks, and volumes
- Real-time log streaming (last 1000 lines + follow)
- File browsing and shell execution inside containers
- Docker-in-Docker (dind) container management
- Vim-style key bindings and command-line interface
