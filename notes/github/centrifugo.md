# Centrifugo - Scalable Real-Time Messaging Server

## Summary
Centrifugo is an open-source, language-agnostic real-time messaging server supporting WebSocket, HTTP-streaming, SSE, gRPC, and WebTransport. It acts as a user-facing PUB/SUB server, enabling chat apps, live comments, multiplayer games, and collaborative tools with any backend.

## Key Points
- Language-agnostic PUB/SUB with channel subscriptions
- Supports WebSocket, HTTP-streaming, SSE, gRPC, WebTransport
- Self-hosted alternative to Pubnub, Pusher, Ably, Socket.io
- Official client SDKs for browser and mobile; unidirectional mode for simple cases
- Mature, production-ready with scalability for many concurrent connections
