# Coolify

## Summary
Coolify is an open-source, self-hostable PaaS alternative to Heroku, Netlify, and Vercel. It lets you manage servers, applications, and databases on your own hardware via SSH, supporting VPS, bare metal, and Raspberry Pi deployments.

## Key Points
- One-line install: `curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash`
- Supports static sites, databases, full-stack apps, and 280+ one-click services

- Manages multiple servers via SSH
- Also offers a paid cloud version at app.coolify.io
