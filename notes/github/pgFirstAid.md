# pgFirstAid
## Summary
pgFirstAid provides a PostgreSQL function/view that returns prioritized health and performance recommendations. It is designed to be simple to deploy and useful for both DBAs and general developers. The output focuses on actionable remediation steps with severity levels and references.
## Key Points
- Single SQL-based health-check system with minimal setup.
- Prioritizes findings by severity from CRITICAL to INFO.
- Covers common issues like missing keys, bloat, and stale statistics.
- Includes recommended actions and links to relevant PostgreSQL docs.
- Inspired by SQL Server First Responder-style diagnostics.
