# Rhesis - Collaborative Testing for LLM & Agentic Apps

## Summary
Rhesis is an open-source platform and SDK for testing LLM and agentic applications. It enables teams to define expected behavior, generate test scenarios from requirements, simulate conversations, and evaluate with 60+ metrics using OpenTelemetry tracing.

## Key Points
- AI-powered test generation from plain language requirements
- Single-turn and conversation flow simulation
- 60+ evaluation metrics with OpenTelemetry integration
- Adversarial behavior detection and edge case generation
- Knowledge-aware via MCP connections (Notion, GitHub, Jira, Confluence)
