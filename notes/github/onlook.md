# onlook
## Summary
Onlook is an AI-first visual editor for building and editing React apps. It combines prompt-driven generation with browser-based visual editing for design-to-code workflows.

## Key Points
- Positions itself as a visual-first coding environment for designers.
- Supports direct editing of Next.js and Tailwind-based interfaces.
- Includes collaborative and iterative app-building workflows.
- Open-source project with active community-driven development.
