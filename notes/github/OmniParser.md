# OmniParser (Microsoft)

## Summary
OmniParser is a comprehensive screen parsing tool that converts UI screenshots into structured elements, enabling vision-based GUI agents. It significantly enhances the ability of models like GPT-4V to generate accurately grounded actions on user interfaces.

## Key Points
- Parses UI screenshots into structured, easy-to-understand elements for GUI agents
- V2 achieves state-of-the-art 39.5% on ScreenSpot Pro grounding benchmark
- OmniTool: control a Windows 11 VM with OmniParser + vision model (OpenAI, DeepSeek, Qwen, Anthropic)
- Supports local trajectory logging for building training data pipelines
- Fine-grained icon detection and improved interactable region identification in latest versions
