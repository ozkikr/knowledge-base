# Jennifer - Code Generator for Go

## Summary
Jennifer is a Go library for generating Go source code programmatically. It provides a fluent API for constructing Go code including functions, types, imports, and control flow, making it ideal for code generation tools and metaprogramming.

## Key Points
- Fluent API for building Go source code programmatically
- Handles imports, identifiers, operators, control flow, generics
- Renders to files or strings with proper formatting
- Used in real-world projects like zerogen and go-contentful-generator
- Stable, well-documented library
