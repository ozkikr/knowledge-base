# Agent Development Kit (ADK) for Go

## Summary
Google's open-source, code-first Go toolkit for building, evaluating, and deploying AI agents. Optimized for Gemini but model-agnostic, it leverages Go's concurrency strengths for cloud-native agent applications with multi-agent orchestration.

## Key Points
- Idiomatic Go design with code-first development approach
- Rich tool ecosystem with pre-built and custom tools
- Modular multi-agent systems with composable specialized agents
- Deploy anywhere, optimized for Google Cloud Run
- Companion SDKs available in Python and Java
