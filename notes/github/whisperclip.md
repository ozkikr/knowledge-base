# WhisperClip

## Summary
WhisperClip is a privacy-first voice-to-text macOS app with AI text enhancement. All processing runs locallyâ€”no cloud services. It supports two speech engines (Parakeet and WhisperKit), local LLM enhancement, and AI meeting notes with speaker separation.

## Key Points
- 100% local processing with Parakeet (Apple Neural Engine) and WhisperKit engines
- AI-powered text enhancement using local LLMs (Gemma, Llama, Qwen, Mistral)
- AI meeting notes with speaker separation, summaries, and action items
- Global hotkey, hold-to-talk, auto-copy/paste, menu bar integration
- Supports audio file transcription (MP3, WAV, M4A, FLAC)
