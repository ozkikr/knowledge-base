# Enhanced DCA Bot

## Summary
A cryptocurrency trading bot implementing enhanced Dollar Cost Averaging with multiple technical indicators (RSI, MACD, Bollinger Bands, SMA), risk management, and real-time monitoring via Prometheus/Grafana.

## Key Points
- Multi-indicator approach with dynamic position sizing based on signal strength
- Risk management: max position limits, balance restrictions, minimum thresholds
- Prometheus metrics + Grafana dashboards for monitoring
- Telegram integration for trade alerts
- Backtesting engine for historical analysis and strategy optimization
