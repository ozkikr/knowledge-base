# GoSkills

## Summary
GoSkills is a Go-based command-line tool for parsing, managing, and executing Claude Skill packages. It supports multiple LLM backends (OpenAI, Claude, compatible APIs), provides a web chat interface, and includes built-in tools for shell commands, Python execution, file operations, web fetching, and search.

## Key Points
- Manages and executes Claude Skills with multi-LLM support (OpenAI, Claude, DeepSeek)
- Web interface with real-time updates, session replay, and artifact rendering (PPT, podcasts)
- Built-in tools: shell, Python, file ops, web fetch, search
- MCP (Model Context Protocol) server integration
- Deep research mode with recursive analysis and self-correction
