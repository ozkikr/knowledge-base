# stash
## Summary
stash is a CLI tool that bidirectionally syncs Markdown files with Apple Notes. It tracks note mappings via front matter and supports both pushing local files and pulling remote edits. The project aims for a simple, scriptable workflow without complex configuration.
## Key Points
- Syncs Markdown notes to and from Apple Notes.
- Uses front-matter identifiers to maintain note/file linkage.
- Exposes straightforward `push` and `pull` command flows.
- Designed around shell usage with minimal setup overhead.
- Built for people who write in Markdown but consume in Apple Notes.
