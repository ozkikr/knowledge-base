# Claude Code + Aider Integration

## Summary
A seamless integration that brings Aider's comprehensive codebase mapping capabilities to Claude Code through custom slash commands. Type `/map` in Claude Code to get a full repository overview.

## Key Points
- Combines Claude Code's interactive AI coding with Aider's codebase mapping
- Provides `/map` slash command for instant repository mapping
- Works with both Anthropic Claude and OpenAI GPT models
- Includes fallback support for graceful API failure handling
- Rich, colored output for better readability
