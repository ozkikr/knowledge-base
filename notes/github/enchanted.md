# Enchanted

## Summary
Enchanted is an open-source iOS, macOS, and visionOS app for chatting with privately hosted language models via Ollama. It provides a ChatGPT-like UI that connects to your own Ollama server, enabling unfiltered, secure, private, and multimodal experiences across Apple devices.

## Key Points
- Native Apple ecosystem app supporting macOS, iOS, and Vision Pro
- Connects to self-hosted Ollama servers for private LLM inference
- ChatGPT-like UI with multimodal support
- Available on the App Store (requires your own Ollama server)
- Supports models like Llama, Mistral, Vicuna, Starling, and more
