# Puzld.ai
## Summary
Puzld.ai is a multi-LLM orchestration framework for agentic execution with memory and training-data-oriented workflows. It is aimed at coordinating different model providers in a single operational layer.
## Key Points
- Orchestrates multiple LLM backends (e.g., Claude, Gemini, Codex ecosystems).
- Supports agent-oriented execution patterns and memory-aware flows.
- Distributed through npm with JS/TS-first integration.
- Designed for teams experimenting with model routing and composable AI workflows.
