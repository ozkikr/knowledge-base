# fractalbits-main
## Summary
High-performance S3-compatible object storage system designed for AI and analytics workloads, with low latency and atomic rename support.

## Key Points
- Uses a Fractal ART metadata engine and full-path approach for scalable directory semantics.
- Targets small-object performance with NVMe + S3 two-tier storage architecture.
- Includes local, Docker, and BYOC deployment flows for rapid testing and benchmarking.
