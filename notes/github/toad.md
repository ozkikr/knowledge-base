# Toad

## Summary
Toad is a unified terminal interface for AI coding agents (Claude, Gemini, Codex, OpenHand, etc.) by Will McGugan. It blends a traditional shell workflow with agentic AI through an intuitive TUI, featuring a fully integrated shell with persistent state.
