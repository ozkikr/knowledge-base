# Fresh — Terminal-Based IDE & Text Editor

## Summary
Fresh is a modern, zero-configuration terminal text editor with familiar VS Code/Sublime keybindings, full mouse support, and IDE-level features. It handles multi-gigabyte files with negligible memory overhead and consistently low-latency input.

## Key Points
- VS Code-like keybindings and UX in the terminal — no learning curve
- Command palette, fuzzy finder, multi-cursor editing, split views
- Handles multi-gigabyte files with minimal memory overhead
- Built-in file explorer, integrated terminal, and theme browser
- No modes or complex shortcuts — standard editor paradigm
