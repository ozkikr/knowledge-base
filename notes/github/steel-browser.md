# Steel Browser - Open Source Browser API for AI Agents

## Summary
Steel Browser is an open-source browser API that provides a batteries-included browser sandbox for AI agents and apps. It manages sessions, pages, and browser processes, allowing programmatic web interaction via Puppeteer, Playwright, or Selenium without building infrastructure from scratch.

## Key Points
- Full browser control via CDP with Puppeteer/Playwright/Selenium compatibility
- Session management maintaining cookies, local storage across requests
- Built-in proxy chain management, anti-detection stealth plugins, fingerprint management
- Browser tools for converting pages to markdown, readability, screenshots, PDFs
- Automatic cleanup and browser lifecycle management
