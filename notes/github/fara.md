# Fara-7B

## Summary
Microsoft's first agentic small language model (7B parameters) designed specifically for computer use. Operates visually by perceiving webpages and taking actions like scrolling, typing, and clicking on predicted coordinates, achieving state-of-the-art performance in its size class.

- Integrates with Magentic-UI for a visual interface
- Served via vLLM with CLI and UI interfaces
