# PR-Agent

## Summary
Open-source AI-powered code review agent by Qodo (formerly Codium). The original open-source PR reviewer that automates code reviews on GitHub, GitLab, Bitbucket, and Azure DevOps.

## Key Points
- Instant test: comment `@CodiumAI-Agent /improve` on any public GitHub repo
- GitHub Action integration with simple workflow file
- CLI usage: `pip install pr-agent` for local development
- Supports multiple platforms: GitHub, GitLab, BitBucket, Azure DevOps
- Community-maintained legacy project; Qodo offers a separate free tier
