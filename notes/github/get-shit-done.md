# get-shit-done
## Summary
Lightweight meta-prompting and spec-driven workflow system for Claude Code, OpenCode, and Gemini CLI focused on preventing context rot.

## Key Points
- Provides a command-driven workflow for project setup, planning, execution, and verification.
- Emphasizes context engineering, structured prompts, and subagent orchestration for reliability.
- Distributed as an npm package (`get-shit-done-cc`) with local/global installation modes.
