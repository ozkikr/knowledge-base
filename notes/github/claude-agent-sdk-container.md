# Claude Agent SDK Container

## Summary
A containerized deployment of the Claude Agent SDK for cloud providers (AWS, GCP, Azure) using standard Anthropic API keys. Provides both a web-based CLI and REST API interface, with GitHub OAuth authentication and multi-agent collaboration examples.

## Key Points
- Deploys Claude Agent SDK via Docker to any cloud platform
- Dual access: web CLI with real-time streaming and REST API
- GitHub OAuth authentication with org/user allowlisting and JWT sessions
- Built-in multi-agent example (Canadian/Australian agents discussing requests)
- Includes prompt injection security considerations
