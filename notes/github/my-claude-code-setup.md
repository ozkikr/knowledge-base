# my-claude-code-setup
## Summary
my-claude-code-setup is a starter template repository for Claude Code projects, including CLAUDE.md memory-bank patterns, hooks, and slash-command conventions. It is designed to help users bootstrap repeatable local setups and context-retention workflows. The repo also includes onboarding guidance and alternate README formats for different experience levels.
## Key Points
- Provides reusable starter files for Claude Code configuration and project memory organization.
- Promotes a memory-bank approach to preserve context across sessions.
- Includes recommendations for tooling, optional integrations, and dev-container usage.
- Offers multiple documentation styles to support different learning preferences.
