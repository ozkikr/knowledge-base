# LyteNyte Grid

## Summary
LyteNyte Grid is a high-performance React data grid built on modern web technologies, designed for enterprise-scale datasets. Available in free Core and paid PRO editions with features like sorting, filtering, row grouping, and async data loading.

## Key Points
- Performance-first: handles large datasets with native browser scrolling
- Core (free): sorting, filtering, row grouping, aggregation, column auto-sizing
- PRO: async data loading, tree data, column manager, data exporting
- Tiny bundle size with modern React-aligned architecture
- One-way data binding and declarative rendering
