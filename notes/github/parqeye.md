# parqeye

## Summary
parqeye is a terminal-based tool for inspecting Parquet files. It provides interactive data visualization, schema exploration, file metadata viewing, and row group statistics â€” all from the terminal with a tab-based interface.

## Key Points
- Interactive table view with keyboard navigation for browsing Parquet data
- Schema explorer for column types, nested structures, and field definitions
- View file-level and row group-level metadata and statistics
- Installable via Cargo, Homebrew, or direct download
- Inspired by csvlens; planned features include cloud file support (s3://)
