# Poe the Poet

## Summary
Poe the Poet is a Python task runner designed to work well with Poetry or uv. It provides a simple way to define and run project tasks from pyproject.toml, serving as a cross-platform alternative to Makefiles for Python projects.

## Key Points
- Define tasks in pyproject.toml under `[tool.poe.tasks]`
- Supports shell, script, and sequence task types
- Works as a Poetry plugin or standalone CLI
- Cross-platform task execution without Makefiles
- Integrates with uv and other Python packaging tools
