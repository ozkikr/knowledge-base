# MCPJam Inspector

## Summary
MCPJam Inspector is a local development client for ChatGPT apps, MCP ext-apps, and MCP servers. It provides a full widget emulator, LLM playground, and OAuth debugger, eliminating the need for ngrok or ChatGPT subscriptions during development.

## Key Points
- Local development for ChatGPT Apps SDK with full windows.openai API support
- MCP ext-apps (Claude) support with JSON-RPC message types
- OAuth debugger supporting all protocol versions (03-26, 06-18, 11-25)
- Available via npx, Docker, or Mac/Windows desktop apps
- LLM playground for chatting with any LLM against your MCP server
