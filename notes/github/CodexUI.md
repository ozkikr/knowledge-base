# CodexUI
## Summary
Native macOS graphical interface for OpenAI Codex CLI with session management and Xcode-aware context capture.

## Key Points
- Runs Codex CLI as a subprocess with streaming responses and resume support.
- Includes configurable sandbox/approval profiles and settings integration.
- Provides optional voice input path via CodeWhisper integration.
