# plexe

## Summary
Plexe lets you build machine learning models by describing them in natural language. It uses 14 specialized AI agents across a 6-phase workflow to analyze data, select metrics, search for the best model, and package it for deployment. YC-backed.

## Key Points
- Describe your ML task in plain language, provide a dataset, get a trained model
- Multi-agent architecture with 14 specialized agents across 6 phases
- Supports XGBoost, CatBoost, and Keras for tabular data
- Accepts Parquet, CSV, ORC, and Avro datasets
- Available as open-source library and managed cloud service
