# structtag

## Summary
A Go library for parsing and manipulating struct field tags. Used by tools like gomodifytags and widely adopted in the Go ecosystem for programmatic struct tag manipulation.

## Key Points
- Parse, modify, and serialize Go struct field tags
- Used by gomodifytags and many other Go tools
- Simple API: Parse tags, get/set/delete individual keys
- Supports sorting and reordering of tags
- Lightweight with no external dependencies
