# SwiftTerm

## Summary
VT100/Xterm terminal emulator library for Swift, embeddable in macOS and iOS apps. Used in commercial SSH clients like Secure Shellfish and La Terminal, and open-source projects like CodeEdit.

## Key Points
- UI-agnostic terminal engine with AppKit (macOS) and UIKit (iOS) frontends
- Used in production SSH clients (Secure Shellfish, La Terminal, CodeEdit)
- Advanced UTF/Unicode/grapheme cluster handling
- Ported from XtermSharp, originally based on xterm.js
- Companion apps: SwiftTermApp (iOS SSH client), Pane (terminal multiplexor)
