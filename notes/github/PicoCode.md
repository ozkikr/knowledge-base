# PicoCode
## Summary
PicoCode is a local codebase assistant that indexes project files and enables semantic Q&A over source code. It includes a production-oriented backend and a simple web interface, with additional PyCharm/IntelliJ plugin support. The design emphasizes keeping data local while allowing customizable inference providers.

## Key Points
- Uses embeddings + vector search to support natural-language questions over codebases.
- Exposes backend capabilities through a REST API and lightweight web UI.
- Supports per-project data isolation and asynchronous analysis workflows.
- Ships with IDE plugin support for integrated developer experience.
