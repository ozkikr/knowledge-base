# QueryWeaver

## Summary
QueryWeaver is an open-source Text2SQL tool that converts natural language questions into SQL using graph-powered schema understanding. It provides REST API and MCP interfaces, powered by FalkorDB's graph database.

## Key Points
- Converts plain English to SQL using graph-based schema understanding
- Available as Docker image, REST API, and MCP server
- Powered by FalkorDB graph database for schema representation
- Supports multiple database connections via environment configuration
- Open-source with cloud option at app.queryweaver.ai
