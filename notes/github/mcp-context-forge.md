# MCP Context Forge (IBM)

## Summary
ContextForge MCP Gateway is IBM's feature-rich gateway and registry that federates MCP and REST services. It unifies discovery, auth, rate-limiting, observability, and virtual servers into one endpoint for AI clients, supporting multiple transport protocols (stdio, SSE, Streamable HTTP).

## Key Points
- Converts REST API endpoints to MCP protocol
- Composes virtual MCP servers with security and observability layers
- Supports multi-transport: stdio, SSE, Streamable HTTP
- Deployable via PyPI or Docker, scales on Kubernetes with Redis-backed federation
- Includes optional Admin UI for management
