# Consult LLM MCP

## Summary
An MCP server that lets Claude Code consult stronger AI models (GPT-5.2, Gemini 3.0 Pro, DeepSeek Reasoner) when Sonnet is insufficient. Supports multi-turn conversations, file context, git change analysis, and cost estimation.

## Key Points
- Query powerful reasoning models (GPT-5.2, Gemini 3.0 Pro, DeepSeek Reasoner) from within Claude Code
- Include relevant files and git changes as context
- Multi-turn conversation support
- Comprehensive logging with cost estimation
- Gemini CLI mode for free quota usage
