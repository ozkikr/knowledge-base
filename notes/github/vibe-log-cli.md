# vibe-log-cli

## Summary
vibe-log-cli is an open-source CLI tool that analyzes Claude Code and Cursor AI-driven coding sessions to extract productivity insights and generate reports. All analysis runs locally using Claude Code's sub-agents for parallel processing.

## Key Points
- Generates daily standup summaries from Claude Code session data in 2-3 minutes
- Produces comprehensive productivity reports with local-only processing (no data leaves your machine)
- Uses Claude Code's sub-agent capabilities for parallel session analysis
- Includes a statusline for real-time session monitoring
- Simple usage via `npx vibe-log-cli@latest`
