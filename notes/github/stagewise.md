# stagewise

## Summary
stagewise is a frontend coding agent for production codebases that works directly in the browser. You click on elements to provide context and tell the agent what to change — it modifies your codebase using real-time browser-powered context.

## Key Points
- Click elements in browser to provide context, describe changes in natural language
- Works out of the box via `npx stagewise@latest` — no install required
- Compatible with Cursor, GitHub Copilot, Windsurf, Cline, Roo Code, and more
- Framework-agnostic with plugin extensibility
- AGPL licensed, open source
