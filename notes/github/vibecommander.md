# Vibe Commander - Terminal IVE for AI Coding Assistants

## Summary
Vibe Commander is a terminal-based Integrated Vibe Environment (IVE) inspired by Norton Commander. It's designed for developers using AI coding assistants like Claude Code, Gemini CLI, or Codex, providing file browsing, code viewing, git operations, and AI integration in one TUI.

## Key Points
- Norton Commander-style dual-panel file browser with vim navigation and git status indicators
- Syntax-highlighted code viewer with regex search and inline diff view
- Git panel for staging/unstaging, committing (with GPG support)
- AI integration with Claude Code, Gemini CLI, and Codex
- Fuzzy search, compact indent toggle, and keyboard-driven workflow
