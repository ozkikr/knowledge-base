# RAGFlow

## Summary
RAGFlow is a leading open-source Retrieval-Augmented Generation engine that combines cutting-edge RAG with agent capabilities. It provides a streamlined workflow for enterprises, powered by a converged context engine and pre-built agent templates.

## Key Points
- Open-source RAG engine with agent capabilities
- Converged context engine for high-fidelity data processing
- Pre-built agent templates for rapid deployment
- Enterprise-scalable with Docker deployment
- Active community with demo available at demo.ragflow.io
