# Terraform GUI

## Summary
A web application that provides a graphical interface for learning and working with Terraform. Targets developers and DevOps beginners by offering a visual cell-based representation of cloud resources, form-based HCL templating, and downloadable Terraform file generation.

## Key Points
- Visual GUI with cell representation of provider resources
- Form-based HCL file templating and generation
- Generates provider, resource, variable, and output configurations
- Downloads ZIP solutions ready for `terraform plan` and `apply`
- Stack: ReactJS frontend + Symfony API backend with Docker deployment
