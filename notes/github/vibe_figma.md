# vibe_figma
## Summary
CLI and API toolchain that converts Figma designs into React components, with Tailwind-first output and optional optimization.

## Key Points
- Supports interactive CLI flow, direct command mode, and environment-based token usage.
- Can be used as a Claude Code skill for Figma-to-React workflows.
- Provides REST API mode and configurable output/overwrite behavior.
