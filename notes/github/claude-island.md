# claude-island
## Summary
macOS menu bar app that surfaces Claude Code session events and approval prompts via a Dynamic Island-style notch interface.

## Key Points
- Monitors multiple Claude Code sessions and presents live status updates.
- Lets users approve/deny tool permissions from the notch without terminal context switching.
- Installs hooks into `~/.claude/hooks/` and communicates via Unix socket events.
