# GenAI Toolbox (MCP Toolbox for Databases)

## Summary
An open-source MCP server for databases by Google. It simplifies building Gen AI tools that let agents access database data, handling complexities like connection pooling, authentication, and observability.

## Key Points
- MCP-compatible server for database access by AI agents
- Integrate tools in less than 10 lines of code, reuse across agents/frameworks
- Built-in connection pooling, authentication, and end-to-end observability
- Currently in beta (pre-v1.0)
- Originally "Gen AI Toolbox for Databases", renamed for MCP alignment
