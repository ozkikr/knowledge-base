# rag-chunk
## Summary
rag-chunk is a Python CLI for parsing, chunking, and evaluating Markdown documents for RAG pipelines. It supports multiple chunking strategies, token-aware splitting, and comparative benchmarking to find effective configurations. The project includes metrics and export tooling for systematic experimentation.
## Key Points
- Offers multiple chunking modes including fixed, sliding-window, header-aware, and semantic.
- Supports token-accurate chunking with model-aware tiktoken settings.
- Provides precision/recall/F1-style evaluation workflows for strategy comparison.
- Includes CLI-friendly reporting and JSON/CSV export options.
