# MiroThinker

## Summary
MiroThinker is an open-source deep research agent optimized for research and prediction, achieving 80.8% Avg@8 on the GAIA benchmark. It includes MiroFlow (agent framework, 82.4% GAIA), MiroVerse (147k training samples), and models up to 235B parameters optimized for financial prediction.

## Key Points
- 80.8% Avg@8 on GAIA benchmark
- MiroFlow companion framework scores 82.4% on GAIA
- MiroVerse: 147k open-source training samples on HuggingFace
- Specialized models for financial prediction (MiroThinker-v1.5)
- Online demo with deep research report generation and document upload
