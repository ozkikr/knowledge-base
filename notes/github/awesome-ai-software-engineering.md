# Awesome AI Software Engineering
## Summary
This repository curates tools, frameworks, and practices for AI-powered software engineering. It organizes resources across coding agents, copilots, multi-agent frameworks, SDKs, and spec-driven development workflows. The collection is opinionated toward practical, iterative, and documentation-centered engineering with AI.

## Key Points
- Aggregates ecosystem references for coding agents, copilots, and orchestration frameworks.
- Highlights methodology themes such as specs-before-code and iterative refinement.
- Includes starter guidance and templates for adopting AI-assisted development workflows.
- Functions as a maintained discovery hub for fast-moving AI engineering tooling.
- Links to learning resources and conceptual guides for broader team adoption.
