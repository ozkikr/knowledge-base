# Oracle — Invoke LLMs with Custom Context and Files

## Summary
Oracle bundles your prompt and files so another AI can answer with real context. It supports GPT-5.x, Gemini 3 Pro, Claude Sonnet/Opus, and more — with single or multi-model runs, browser automation, and clipboard copy fallback.

## Key Points
- Bundles files and prompts for context-rich LLM queries
- Multi-model API runs via OpenAI, Gemini, Anthropic
- Browser automation mode (opens ChatGPT/Gemini directly)
- Session management with replay and status commands
- TUI mode for interactive human use
