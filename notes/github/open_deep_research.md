# Open Deep Research

## Summary
Open Deep Research is a configurable, fully open source deep research agent by LangChain. It works across many model providers, search tools, and MCP servers, performing on par with popular deep research agents on the Deep Research Bench leaderboard.

## Key Points
- Fully open source deep research agent built with LangGraph
- Works across multiple model providers and search tools
- Supports MCP servers for extensibility
- Competitive with commercial deep research agents on benchmarks
- Free course available on LangChain Academy
