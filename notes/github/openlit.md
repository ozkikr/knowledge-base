# OpenLIT

## Summary
OpenLIT is an open source AI engineering platform providing OpenTelemetry-native LLM observability, GPU monitoring, guardrails, evaluations, prompt management, and more. It integrates with 50+ LLM providers, vector databases, agent frameworks, and GPUs with just one line of code.

## Key Points
- One-line SDK integration for full-stack AI observability (traces + metrics)
- Analytics dashboard for health, performance, costs, and user interactions
- Prompt Hub for versioning and managing prompts across applications
- Secure API key and secrets management (Vault)
- Follows OpenTelemetry Semantic Conventions for GenAI; uses ClickHouse for storage
