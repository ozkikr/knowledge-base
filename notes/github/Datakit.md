# Datakit
## Summary
Datakit is a browser-based data analysis studio that processes files locally for privacy-preserving workflows. It combines data preview, SQL querying, AI-assisted analysis, and notebook-style scripting in one interface. The platform is positioned for large-file analytics without forcing server-side uploads.

## Key Points
- Handles CSV, Excel, JSON, and Parquet with local browser-side processing.
- Uses DuckDB in-browser for SQL workflows and result exploration.
- Offers AI-assisted query generation/insights with configurable providers.
- Includes notebook capabilities and support for remote data connectors.
