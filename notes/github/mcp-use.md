# mcp-use - Easiest Way to Interact with MCP Servers

## Summary
mcp-use is a framework for building AI agents and clients that interact with MCP (Model Context Protocol) servers. It provides MCP Agents for multi-step reasoning with tools, MCP Clients for direct programmatic access, and an MCP Inspector for debugging.

## Key Points
- MCP Agents: AI agents that use tools and reason across steps
- MCP Clients: Connect any LLM to any MCP server
- MCP Inspector: Web-based debugger for MCP servers
- Available in both Python and TypeScript
- Works with LangChain and multiple LLM providers
