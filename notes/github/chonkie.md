# Chonkie

## Summary
Chonkie is a lightweight, fast Python chunking library for RAG pipelines. It supports multiple chunking strategies, 32+ integrations with vector DBs and tools, 56 languages, and provides a pipeline API for chaining chunking and refinement steps.

## Key Points
- Multiple chunkers: recursive, token, sentence, semantic, and more
- Pipeline API for chaining chunk → refine → embed → ship to vector DB
- 32+ integrations out of the box
- Multilingual support for 56 languages
- Minimal dependencies with optional extras (`chonkie[all]`)
