# difi

## Summary
difi is a terminal-first tool for reviewing and refining Git diffs before pushing changes. It offers a structured tree-plus-diff interface and editor-aware navigation for quick correction loops. The project emphasizes fast startup and keyboard-driven workflows for developers who prefer CLI review.

## Key Points
- Provides focused diff browsing with file-tree context for rapid inspection.
- Integrates with Vim/Neovim workflows and supports direct line jumps for edits.
- Can be installed via Homebrew, Go install, and Arch packages.
- Supports using difi as a native git alias command.
- Pairs with difi.nvim for richer inline editing and visualization.
