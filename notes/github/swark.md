# Swark - Automatic Architecture Diagrams from Code

## Summary
Swark is a VS Code extension that generates architecture diagrams from code automatically using LLMs via GitHub Copilot integration. It outputs Mermaid.js diagrams, supports all programming languages natively through LLM understanding, and requires no additional API keys.

## Key Points
- Free and open source; only requires GitHub Copilot (now free)
- Universal language support via LLM (no per-language deterministic parsing)
- Seamless GitHub Copilot integrationâ€”no extra auth or API keys
- Privacy-first: code shared only with GitHub Copilot
- Generates editable Mermaid.js diagrams for documentation, onboarding, legacy code understanding
