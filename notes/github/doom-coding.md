# Doom Coding

## Summary
A guide for coding on-the-go using a smartphone terminal. The concept of "doom coding" (like doom scrolling but productive) enables terminal-based development from anywhere with an internet connection.

## Key Points
- 5-minute setup guide for mobile terminal coding
- Uses phone as a terminal client to connect to remote development environments
- Claude-assisted research and troubleshooting approach
- Works anywhere with internet connection
- DIY approach to on-the-go development
