# DarwinKit - Native Mac APIs for Go

## Summary
DarwinKit (formerly MacDriver) provides native Apple API bindings for Go. It allows Go developers to build native macOS applications using Apple frameworks like AppKit, CoreGraphics, and more without writing Objective-C or Swift.

## Key Points
- Go bindings for native Apple/macOS APIs
- Previously known as MacDriver
- Enables building native Mac GUI apps in pure Go
- Covers a wide range of Apple frameworks
- Active community with sponsorship support
