# opcode (formerly claudia)

## Summary
opcode is a powerful Tauri 2 desktop GUI application for Claude Code. It provides visual session management, custom agent creation, secure background agents, usage analytics, MCP server management, and timeline/checkpoint features.

## Key Points
- Built with Tauri 2 for a native desktop experience
- Project & session management with visual interface
- Custom CC Agents creation and management
- Usage analytics dashboard for tracking costs and tokens
- MCP server management and CLAUDE.md editing built in
- Repo redirected from getAsterisk/claudia to winfunc/opcode
