# mitto
## Summary
Mitto provides a modern interface for running and managing multiple ACP-compatible coding-agent sessions across desktop, web, and mobile. It is geared toward teams or individuals who want persistent, multi-session workflows and remote continuity. The project combines CLI capabilities with richer UI surfaces, including a native macOS app.

## Key Points
- Supports multiple ACP-compatible agents (e.g., Claude Code, Copilot CLI, Auggie).
- Offers session management with history and resume across devices.
- Includes both web interface and native macOS app for flexible access patterns.
- Adds permission controls, streaming updates, and rich markdown/code rendering.
- Provides straightforward install paths via Homebrew and release artifacts.
