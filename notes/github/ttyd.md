# ttyd

## Summary
ttyd is a simple command-line tool for sharing a terminal over the web. It exposes a terminal session via a web browser using WebSockets, supporting features like SSL, basic authentication, read-only mode, and client limits.

## Key Points
- Share any terminal command over the web via browser
- Supports SSL, basic auth, and auth proxy headers
- Configurable: read-only/writable, max clients, one-shot mode
- IPv6 support and UNIX domain sockets
- Lightweight C implementation with xterm.js frontend
