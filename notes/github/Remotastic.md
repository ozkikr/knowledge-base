# Remotastic
## Summary
Remotastic is a macOS menu bar app that maps Apple TV Siri Remote input to desktop control actions. It enables cursor movement, scrolling, and media/navigation commands from the remote. The app is intended as a lightweight bridge between TV-style hardware and macOS interaction.

## Key Points
- Converts Siri Remote gestures and buttons into pointer and media controls.
- Includes configurable button mappings and cursor/scroll mode toggles.
- Supports multi-monitor setups from a menu bar control surface.
- Depends on accessibility permissions and Bluetooth pairing.
