# SimpleDMS - Document Management for Small Businesses

## Summary
SimpleDMS is an open-source document management system designed for small businesses. It uses a metadata-driven concept with a tag system for categorization and workspaces (spaces) for shared/private access. Also available as SaaS at simpledms.eu and simpledms.ch.

## Key Points
- Metadata-driven: tag system for categorization + workspaces for access control
- Designed for small businesses and families
- Available as both self-hosted open source and SaaS
- Dark mode supported
- Docker Compose setup available
