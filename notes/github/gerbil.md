# Gerbil

## Summary
A desktop application for running Large Language Models locally. Provides a GUI for managing backends, model downloads, and hardware acceleration, powered by KoboldCpp. Supports text and image generation with full offline capability.

## Key Points
- Cross-platform desktop app (Windows, macOS, Linux including Wayland)
- Powered by KoboldCpp/llama.cpp with GPU acceleration (CUDA, ROCm, Vulkan, Metal)
- Built-in image generation (Flux, Chroma, Qwen Image, Z-Image)
- Integrated HuggingFace search for model discovery
- SillyTavern and OpenWebUI integration options
