# CCC - Claude Code Companion

## Summary
CCC bridges Claude Code with Telegram, enabling remote control of coding sessions from your phone. It's fully self-hosted and privacy-first, supporting multi-session management, file transfer, voice messages, and image support.

## Key Points
- 100% self-hosted with no third-party servers
- Remote start and management of Claude Code sessions via Telegram
- Multi-session support with individual Telegram topics
- Voice message transcription via Whisper, image support for Claude analysis
- tmux integration for persistent sessions with seamless handoff between devices
