# OpenCode

## Summary
OpenCode is an open-source AI coding agent available as both a CLI tool and desktop app. It supports multiple platforms and package managers, and provides an AI-powered coding assistant experience similar to commercial alternatives.

## Key Points
- Install via npm, brew, scoop, choco, pacman, nix, or curl
- Desktop app available for macOS (ARM/Intel), Windows, and Linux
- Multi-language README support (15+ languages)
- Open source alternative to commercial coding agents
- Available at opencode.ai
