# Claudex

## Summary
Claudex is a self-hosted, open-source web UI for Claude Code with sandboxed execution environments (Docker local, E2B cloud, Modal cloud). It provides in-browser VS Code, terminal, multi-provider AI support, custom skills, and MCP server integration.

## Key Points
- Multiple sandbox options: Docker (local), E2B (cloud), Modal (cloud)
- Supports Claude Max, OpenRouter, OpenAI, and custom AI providers
- Full IDE experience with VS Code in browser, terminal, and file explorer
- VNC browser control with Playwright MCP for programmatic browser interaction
- Extensible with skills, agents, slash commands, and MCP servers
