# any-llm

## Summary
A Python library by Mozilla AI that provides a single unified interface to communicate with any LLM provider. Switch between OpenAI, Anthropic, Mistral, Ollama, and more without changing code.

## Key Points
- Single `completion()` function works across all providers
- Install provider-specific extras: `pip install 'any-llm-sdk[openai,mistral]'`
- Requires Python 3.11+
- API keys via environment variables or passed directly
- Published on PyPI as `any-llm-sdk`
