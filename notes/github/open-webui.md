# Open WebUI

## Summary
An extensible, feature-rich, self-hosted AI platform designed to operate entirely offline. Supports Ollama and OpenAI-compatible APIs with built-in RAG inference engine, making it a comprehensive AI deployment solution.

## Key Points
- Self-hosted and works fully offline
- Supports multiple LLM runners: Ollama, OpenAI-compatible APIs
- Built-in RAG (Retrieval-Augmented Generation) capabilities
- Extensible plugin architecture
- One of the most popular open-source AI UI projects
