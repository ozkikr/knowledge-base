# MCP Toolbox SDK for Go

## Summary
Go SDK for integrating MCP Toolbox for Databases into GenAI applications. Fetches tool definitions from a running Toolbox instance, provides Go structs, invokes tools, and handles authentication and parameter binding.

## Key Points
- Go SDK for the MCP Toolbox service (centralized tool management)
- Breaking change in v0.6.0: transitioned to multi-module structure
- Loads tools defined in Toolbox as standard Go tools for orchestration frameworks
- Handles communication, auth, and parameter binding automatically
- Part of the broader googleapis/genai-toolbox ecosystem
