# FluidVoice
## Summary
FluidVoice is an open-source macOS voice-to-text dictation app designed for local/offline usage with low-latency transcription. It supports multiple speech backends and optional AI post-processing for rewriting or command-style actions. The project targets privacy-preserving daily dictation workflows with global hotkeys and direct app typing.

## Key Points
- Runs fully local transcription workflows on macOS, including Apple Silicon support.
- Offers command mode and write mode for action execution and text editing.
- Supports several speech engines (Parakeet variants, Apple Speech, Whisper).
- Integrates optional provider-based enhancement while storing keys in Keychain.
- Provides menu bar UX, history, and usage stats.
