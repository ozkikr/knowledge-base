# treemd

## Summary
A TUI/CLI markdown navigator with tree-based structural navigation. Provides a dual-pane interface with an expandable/collapsible heading tree and synchronized content view, plus interactive features like table editing and checkbox toggling.

## Key Points
- Dual-pane TUI: heading tree + synchronized content view
- Interactive table editing with vim keybindings
- jq-like query language for extracting markdown elements
- Supports stdin piping for shell-scripted workflows
- Configurable keybindings and themes
