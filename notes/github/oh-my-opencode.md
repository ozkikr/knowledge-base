# oh-my-opencode
## Summary
oh-my-opencode is an open-source agent harness focused on extending OpenCode with autonomous and specialized coding workflows. It positions itself as a high-agency environment with compatibility layers, curated tools, and multi-agent task delegation. The repository includes installation guidance, community updates, and security warnings about impersonation sites.
## Key Points
- Provides enhanced agent workflows with specialized roles and background execution patterns.
- Includes strong warnings around unofficial downloads and OAuth/ToS considerations.
- Targets advanced coding automation use cases and power-user customization.
- Backed by an active community and frequent iteration.
