# ScreenCoder

## Summary
ScreenCoder is a multimodal AI system that transforms UI screenshots or design mockups into clean, production-ready HTML/CSS code. It uses a modular multi-agent architecture combining visual understanding, layout planning, and adaptive code synthesis.

## Key Points
- Converts screenshots/mockups to editable HTML/CSS with high fidelity
- Modular multi-agent framework with visual understanding and layout planning
- Supports customized modifications after generation
- Includes ScreenBench benchmark with 1000 real-world web screenshots
- Released post-training code (SFT + RL) for alignment
