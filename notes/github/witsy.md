# Witsy

## Summary
Witsy is a BYOK (Bring Your Own Keys) desktop AI assistant and universal MCP client. It supports a wide range of AI providers for chat, image/video creation, TTS, STT, and search. It's one of the first universal MCP clients, allowing MCP servers to work with virtually any LLM.
