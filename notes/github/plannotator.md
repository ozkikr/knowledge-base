# Plannotator

## Summary
Plannotator is an interactive plan review tool for AI coding agents. It provides a visual UI for annotating and refining agent plans, supports team collaboration, and integrates with Claude Code and OpenCode.

## Key Points
- Visual annotation UI for AI agent plans (delete, insert, replace, comment)
- Code review mode for reviewing git diffs with inline annotations
- Image annotation support with pen, arrow, and circle tools
- Auto-save approved plans to Obsidian and Bear Notes
- Integrates as a plugin for Claude Code and OpenCode
