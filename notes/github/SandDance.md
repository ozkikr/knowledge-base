# SandDance

## Summary
Microsoft Research's data visualization tool that uses unit visualizations (one-to-one mapping between data rows and visual marks) with smooth animated transitions. Available as a web app, VS Code extension, Power BI visual, and embeddable JavaScript components.

## Key Points
- Unit visualizations: each data row maps to one visual mark on screen
- Smooth animated transitions maintain context during interaction
- Modular, extensible, and embeddable JavaScript architecture
- Available in Power BI, Azure Data Studio, VS Code, and web
- Created by Microsoft Research VIDA Group
