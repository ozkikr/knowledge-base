# Superego

## Summary
A metacognitive advisor for AI coding assistants that monitors conversations, evaluates the assistant's approach, and provides feedback before finishing. Supports Claude Code (full), OpenAI Codex CLI (alpha), and OpenCode (alpha).

## Key Points
- Evaluates proposed changes before large edits (>20 lines)
- Reviews full conversation before assistant finishes
- Blocks assistant and shows feedback if concerns are found
- Creates feedback loops for course-correction during work
- Installs as Claude Code plugin via marketplace
