# compound-engineering-plugin
## Summary
compound-engineering-plugin is an official Claude Code plugin marketplace/repo centered on compound engineering workflows. It packages planning, execution, review, and learning loops into reusable commands and skills. The project also includes tooling to convert/sync plugins and skills to other agent ecosystems.

## Key Points
- Provides structured workflow commands (plan, work, review, compound).
- Emphasizes iterative improvement by capturing reusable engineering knowledge.
- Supports plugin conversion/sync for multiple coding-agent environments.
- Includes install paths for Claude Code plus experimental multi-tool compatibility.
