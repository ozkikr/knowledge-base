# asyncmachine-go

## Summary
asyncmachine-go is a batteries-included graph control flow library for Go combining aspects of AOP, the actor model, and state machines. It provides a declarative approach to managing complex async workflows with built-in devtools for visualization and debugging.

## Key Points
- Combines AOP, actor model, and state machine paradigms
- Graph-based control flow for complex async workflows in Go
- Includes devtools for state visualization and debugging
- Active development with comprehensive test coverage
- Supports distributed state management via Matrix protocol
