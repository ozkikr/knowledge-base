# gotests

## Summary
A Go test generator that automatically creates table-driven tests from source code. Analyzes function and method signatures to generate comprehensive test scaffolding. Supports generics (Go 1.18+), custom templates, parallel subtests, and has plugins for VS Code, Vim, Emacs, Sublime, and IntelliJ.

## Key Points
- Zero-config: works out of the box with any Go project
- Generates table-driven test structure with proper type handling
- Flexible filtering: specific functions, exported only, or entire packages
- Full Go generics support and recursive directory processing
- Built-in testify template support and custom template system
- Widely adopted with editor plugins across all major editors
