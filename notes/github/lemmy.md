# Lemmy
## Summary
Lemmy is a TypeScript monorepo for building AI applications with unified LLM interfaces and terminal UI components. It provides reusable packages for multi-provider model access, tool handling, and conversation management. The repository also includes example applications such as claude-trace and other agent-focused utilities.
## Key Points
- Core package offers unified interfaces across Anthropic, OpenAI, and Google providers.
- Includes a TUI framework with differential rendering and composable components.
- Ships multiple applications/demos including tracing and chat tooling.
- Structured as a workspace monorepo with shared build, test, and typecheck workflows.
