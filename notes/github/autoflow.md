# AutoFlow

## Summary
AutoFlow is an open-source Graph RAG-based conversational knowledge base tool built on TiDB Vector Storage, LlamaIndex, and DSPy. It provides a Perplexity-style conversational search interface with an advanced website crawler for ingesting documentation sites.

## Key Points
- Uses Graph RAG (knowledge graph + RAG) for enhanced retrieval
- Built on TiDB Serverless Vector Storage for scalable vector search
- Integrates LlamaIndex and DSPy for the RAG pipeline
- Features a built-in website crawler for documentation ingestion
- Deployable via Docker Compose; early stage, moving toward `pip install autoflow-ai`
