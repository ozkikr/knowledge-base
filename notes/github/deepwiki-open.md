# DeepWiki-Open - AI-Powered Wiki Generator for Repos

## Summary
DeepWiki-Open automatically creates interactive wikis for any GitHub, GitLab, or BitBucket repository. It analyzes code structure, generates comprehensive documentation, creates visual Mermaid diagrams, and organizes everything into a navigable wiki.

## Key Points
- Supports GitHub, GitLab, and BitBucket repositories including private repos
- AI-powered code structure analysis with Mermaid diagram generation
- RAG-powered "Ask" feature for chatting with repositories
- DeepResearch mode for multi-turn investigation of complex topics
- Multiple model provider support
