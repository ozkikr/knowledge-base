# CCManager

## Summary
CCManager is a CLI application for managing multiple AI coding assistant sessions (Claude Code, Gemini CLI, Codex CLI, Cursor Agent, etc.) across Git worktrees and projects. It provides real-time session state monitoring without requiring tmux.

## Key Points
- Runs multiple AI assistant sessions in parallel across different Git worktrees
- No tmux dependency â€” completely self-contained terminal UI
- Real-time session state indicators: waiting, busy, idle
- Supports Claude Code, Gemini CLI, Codex CLI, Cursor Agent, Copilot CLI, Cline CLI, OpenCode, Kimi CLI
- Features worktree management, session data copying, configurable shortcuts, and auto-approval (experimental)
