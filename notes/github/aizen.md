# Aizen

## Summary
A macOS workspace for developers who work on multiple branches in parallel. Each worktree gets its own terminal, file browser, web browser, and AI agent session, enabling developers to switch worktrees instead of windows.

## Key Points
- Per-worktree sessions with terminal, files, browser, and AI chat
- GPU-accelerated terminal powered by libghostty
- Supports Claude, Codex, Gemini agents via Agent Client Protocol (ACP)
- Built-in Git operations with diff viewer and syntax highlighting
- Requires macOS 13.5+ on Apple Silicon or Intel
