# WAHA - WhatsApp HTTP API

## Summary
WAHA is a self-hosted WhatsApp HTTP API (REST) that can be set up in under 5 minutes via Docker. It supports three engines: WEBJS (browser-based), NOWEB (WebSocket Node.js), and GOWS (WebSocket Go), providing full WhatsApp messaging capabilities through a simple REST interface.

## Key Points
- Three engine options: WEBJS, NOWEB, GOWS for different use cases
- Docker-based deployment with simple docker pull and run
- QR code authentication flow for WhatsApp sessions
- Swagger API documentation built-in
- Plus version available with additional features
