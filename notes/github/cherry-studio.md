# Cherry Studio

## Summary
Cherry Studio is a desktop AI productivity client supporting multiple LLM providers with smart chat, autonomous agents, and 300+ built-in assistants. Available on Windows, Mac, and Linux under AGPL-3.0.

## Key Points
- Desktop client for Windows, Mac, and Linux
- Unified access to frontier LLMs from multiple providers
- 300+ pre-built AI assistants
- Autonomous agent capabilities
- Active community on Telegram, Discord, and QQ
