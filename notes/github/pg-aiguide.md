# pg-aiguide
## Summary
pg-aiguide provides AI-optimized PostgreSQL guidance via an MCP server and Claude plugin experience. It combines semantic search over Postgres docs with curated best-practice “skills” to improve schema and query generation quality. The project targets better, more modern SQL output from coding assistants.
## Key Points
- Exposes semantic search tools for official PostgreSQL docs and TigerData/Timescale docs.
- Includes reusable best-practice skills for schema design, indexing, constraints, and naming.
- Supports quick installation across Claude Code, Codex, Cursor, VS Code, Gemini CLI, and others.
- Focused on modern Postgres patterns and extension-aware guidance for production usage.
