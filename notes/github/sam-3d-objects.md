# sam-3d-objects
## Summary
Meta research model and codebase for reconstructing 3D object geometry and texture from single images, including occluded/cluttered scenes.

## Key Points
- Publishes code, checkpoints, paper links, demo resources, and notebooks.
- Supports single-object and multi-object generation workflows.
- Can be combined with SAM 3D Body for aligned human/object 3D outputs.
