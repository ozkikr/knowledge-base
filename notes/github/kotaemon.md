# kotaemon
## Summary
Open-source RAG application framework/UI for chatting with documents, aimed at both end users and developers.

## Key Points
- Provides a document QA web UI with support for multiple LLM providers and local models.
- Designed as a customizable framework for building and extending RAG pipelines.
- Emphasizes citations, retrieval quality, and practical multi-user document workflows.
