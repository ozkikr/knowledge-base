# prose
## Summary
OpenProse proposes a programming language model for long-running AI sessions and orchestrated agent workflows. The repository includes language specifications, examples, and integration guidance for agent toolchains.

## Key Points
- Defines a DSL-style approach for composing multi-agent execution flows.
- Includes examples and compiler/spec documentation under skills.
- Targets declarative orchestration of research, writing, and iteration loops.
- Integrates with agent ecosystems like Claude Code and OpenCode.
