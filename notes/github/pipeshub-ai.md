# PipesHub - Workplace AI Platform

## Summary
PipesHub is an open-source, extensible workplace AI platform for enterprise search and workflow automation. It connects to data sources like Google Workspace, Microsoft 365, Slack, Jira, and Confluence, enabling natural language search with citations powered by Knowledge Graphs and Page Ranking.

## Key Points
- Natural language enterprise search across multiple SaaS apps with proper citations
- Real-time or scheduled indexing with flexible deep learning model choices
- Access-driven visibility respecting existing permissions
- No-code interface for creating custom apps and AI agents
- Knowledge Graph and Page Ranking for explainable, relevant results
