# claudechic
## Summary
claudechic is a stylish terminal UI for Claude Code built on Textual. It focuses on improving usability and visual ergonomics for interactive coding-agent workflows in the terminal.

## Key Points
- Provides a TUI wrapper around Claude Code workflows.
- Built with Textual for rich terminal interaction patterns.
- Supports multi-agent and worktree-oriented development setups.
- Early-stage project with active experimentation and iteration.
