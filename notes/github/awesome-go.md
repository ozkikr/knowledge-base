# awesome-go
## Summary
awesome-go is a curated index of Go frameworks, libraries, and developer tools across a wide range of categories. It is community-maintained and designed as a discovery resource rather than a single software package. The repository is one of the central references for Go ecosystem tooling.
## Key Points
- Covers extensive categories including web frameworks, databases, testing, AI, CLI, and DevOps.
- Organized as a structured reference list for quick package discovery.
- Community contribution model with guidelines for additions and maintenance updates.
- Useful starting point for evaluating libraries before implementation.
