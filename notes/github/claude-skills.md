# claude-skills
## Summary
claude-skills is a large curated collection of production-oriented skills, slash commands, and agent bundles for Claude Code. It covers web, cloud, AI integration, and workflow automation use cases with marketplace-style installation. The toolkit focuses on practical acceleration and reduced token waste through context-aware skill loading.
## Key Points
- Provides ~97 skills across categories like Cloudflare, frontend, AI/ML, and tooling.
- Supports both background skills and user-invocable slash commands.
- Includes bundled specialized agents (debugging, review, deployment, migration).
- Documents plugin install flows for marketplace and local development workflows.
