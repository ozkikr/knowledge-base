# RA.Aid

## Summary
A standalone autonomous coding agent built on LangGraph's task execution framework. Handles research, planning, and implementation of multi-step development tasks. Can optionally integrate with aider for specialized code editing.

## Key Points
- Built on LangGraph for agent-based task execution
- Supports research → planning → implementation workflow
- Optional aider integration via `--use-aider` flag for code editing
- Can automatically execute shell commands and make code changes
- `--cowboy-mode` flag skips command approval for full autonomy
