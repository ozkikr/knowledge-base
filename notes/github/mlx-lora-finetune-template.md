# MLX LoRA Fine-tuning Engine

## Summary
A template/engine for fine-tuning LLMs with LoRA using Apple MLX on Mac Silicon. Features a Streamlit web UI with simple and advanced modes, HuggingFace integration, and AI-assisted data preparation using small LLMs.

## Key Points
- LoRA fine-tuning optimized for Apple Silicon (M1/M2/M3) via MLX
- Streamlit UI with Simple (presets) and Advanced (full control) modes
- Data preparation from JSON, raw text, or folders; optional AI-assisted Q&A generation with Qwen3-0.6B
- Checkpoint management and HuggingFace Hub upload/download integration
- Supports multiple output formats: completion, Q&A, knowledge, raw
