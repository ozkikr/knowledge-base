# Marker - Convert PDF to Markdown + JSON

## Summary
Marker converts documents (PDF, images, PPTX, DOCX, XLSX, HTML, EPUB) to markdown, JSON, chunks, and HTML quickly and accurately. It handles tables, equations, code blocks, and images across all languages, with optional LLM-boosted accuracy mode.

## Key Points
- Supports PDF, images, PPTX, DOCX, XLSX, HTML, EPUB in all languages
- Formats tables, equations, inline math, links, references, code blocks
- Hybrid mode with LLM (Gemini/Ollama) for highest accuracy
- Benchmarks favorably against Llamaparse, Mathpix, and other tools
- Works on GPU, CPU, or MPS; ~25 pages/second batch throughput on H100
