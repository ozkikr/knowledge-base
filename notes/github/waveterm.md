# Wave Terminal

## Summary
An open-source, cross-platform terminal that combines traditional terminal features with graphical capabilities like file previews, web browsing, and AI assistance. Runs on MacOS, Linux, and Windows.

## Key Points
- Drag & drop interface for organizing terminal blocks, editors, web browsers, and AI assistants
- Built-in AI assistant (Wave AI) with context-aware terminal understanding
- Rich file preview system for remote files (markdown, images, video, PDFs, CSVs)
- One-click remote connections with full terminal and file system access
- `wsh` command system for managing workspace from CLI
