# Shaper

## Summary
Open source SQL-driven data dashboard tool powered by DuckDB. Visualize and share data using pure SQL queries with a simple Docker-based deployment.

## Key Points
- SQL-only approach to building dashboards
- Powered by DuckDB for fast analytical queries
- Quick start via Docker: `docker run --rm -it -p5454:5454 taleshape/shaper`
- Managed hosting available from Taleshape
