# Hunyuan3D-2

## Summary
High-resolution 3D asset generation using large-scale diffusion models from Tencent. Generates 3D models from text or images with PBR materials. Part of an active series with releases up to Hunyuan3D-2.1 with full open-source training code, and a related HunyuanWorld-1.0 for 3D world generation.

## Key Points
- Text-to-3D and image-to-3D generation with high-resolution output
- PBR (physically-based rendering) material generation support
- Turbo paint model for fast texture generation
- Multiview texture generation pipeline
- Fully open-sourced including training code (v2.1)
- Related projects: HunyuanWorld-1.0, RomanTex, MaterialMVP
