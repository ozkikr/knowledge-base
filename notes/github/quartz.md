# Quartz

## Summary
Quartz is a fast, batteries-included static-site generator that transforms Markdown content into fully functional websites. It's popular for building digital gardens and knowledge bases, with 11.2k+ stars and strong community support.

## Key Points
- Transforms Markdown into fully functional static websites
- Popular for digital gardens and personal knowledge bases
- Batteries-included with sensible defaults
- 11.2k+ GitHub stars, active development
- Supports wikilinks, backlinks, and graph views typical of knowledge management
