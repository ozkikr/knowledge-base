# Potpie

## Summary
Potpie is an open-source platform for creating AI agents specialized in your codebase. It builds a comprehensive knowledge graph of your code to understand complex relationships, enabling automated code analysis, testing, debugging, and feature development through custom or prebuilt agents.

## Key Points
- Prompt-to-Agent: create custom engineering agents from natural language
- Builds a knowledge graph capturing code relationships
- Prebuilt agents for common tasks (debugging, testing, development)
- Available as VS Code extension and API
- Deep code understanding through graph-based analysis
