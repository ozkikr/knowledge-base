# Fizzy

## Summary
Fizzy is an open-source Kanban tracking tool for issues and ideas by 37signals (Basecamp). It can be self-hosted via Docker or deployed with Kamal. Released under the O'Saasy License.

## Key Points
- Kanban board for issues and ideas by 37signals
- Docker deployment or Kamal-based custom deployment
- Open to contributions with a style guide
- O'Saasy License
- Self-hostable alternative to commercial kanban tools
