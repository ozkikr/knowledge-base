# gemini-mcp-server
## Summary
MCP server that connects AI assistants to Gemini CLI capabilities for large-context analysis, web search, and media processing.

## Key Points
- Exposes tools like `gemini`, `web-search`, `analyze-media`, `shell`, and `brainstorm`.
- Targets codebase-scale analysis using Gemini context capacity.
- Installable via npx and integrable with Claude Code/Claude Desktop and other MCP clients.
