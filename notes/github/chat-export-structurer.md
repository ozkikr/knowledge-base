# Chat Export Structurer

## Summary
Converts AI chat exports (ChatGPT, Claude, Grok) into queryable SQLite archives. Local-first tool with full-text search, SHA1-based deduplication, and streaming parser for large files. Part of the MyChatArchive platform.

## Key Points
- Supports ChatGPT, Claude (Anthropic), and Grok exports
- Built-in FTS5 full-text search across all conversations
- SHA1-based deduplication across imports
- Streaming parser handles multi-GB files
- No API keys or cloud services required
