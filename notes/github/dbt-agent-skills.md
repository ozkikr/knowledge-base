# dbt-agent-skills
## Summary
This repository is a curated set of Agent Skills for dbt workflows, aimed at helping coding agents execute dbt tasks more reliably. It packages repeatable guidance for analytics engineering, semantic layer work, troubleshooting, and migration scenarios. The project is designed to be consumed by Claude Code and other skills-compatible AI tools.
## Key Points
- Covers practical dbt workflows such as model authoring, testing, semantic modeling, and job debugging.
- Supports both Claude plugin installation and broader skills CLI workflows for other agent clients.
