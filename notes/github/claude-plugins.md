# claude-plugins
## Summary
This repository is a Claude Code plugin marketplace containing installable plugin packages and command integrations. Its current focus includes DeepWiki-powered repository exploration and Q&A workflows. It serves as a lightweight distribution point for extending Claude Code behavior.
## Key Points
- Adds a plugin marketplace source for Claude Code.
- Includes DeepWiki plugin commands for repo Q&A and architecture exploration.
- Installation is CLI-driven via `claude plugin marketplace add`.
- Structured as plugin definitions and documentation rather than a large codebase.
