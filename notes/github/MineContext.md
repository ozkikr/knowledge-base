# MineContext

## Summary
MineContext is a proactive context-aware AI partner by Volcengine that uses screenshots and content comprehension to build personal context. It's local-first with privacy protection, supports local AI models, and is compared to ChatGPT Pulse and Dayflow.

## Key Points
- Proactive context-aware AI that captures and comprehends screen content
- Local-first architecture with privacy protection and local AI model support
- Frontend built with core tech stack, backend with layered architecture
- Related to OpenViking context database for AI agents
- Targets users wanting a personal AI partner that understands their workflow
