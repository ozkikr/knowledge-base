# ChatGPT Exporter - Export and Share ChatGPT Conversation History

## Summary
ChatGPT Exporter is a GreasyFork/Tampermonkey userscript that adds export functionality to ChatGPT. It supports exporting conversations in multiple formats including Text, HTML, Markdown, and PNG, with options for batch exporting multiple conversations.

## Key Points
- Tampermonkey userscript for exporting ChatGPT conversations
- Supports Text, HTML, Markdown, and PNG export formats
- Batch export of multiple conversations
- Multi-language support (English, French, Indonesian, Korean, Turkish)
- Available on GreasyFork and GitHub
