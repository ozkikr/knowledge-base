# Garage

## Summary
An S3-compatible object store designed for small self-hosted geo-distributed deployments. Built in Rust by Deuxfleurs, it provides lightweight distributed storage that works across multiple geographic locations. GitHub mirror of main repo at git.deuxfleurs.fr.

## Key Points
- S3-compatible API for object storage
- Designed for geo-distributed deployments across multiple sites
- Lightweight and self-hosted, targeting small-to-medium deployments
- Written in Rust for performance and safety
- 2.7k+ GitHub stars, 2,634 commits
