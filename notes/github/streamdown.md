# Streamdown - Streaming Markdown for AI

## Summary
Streamdown is Vercel's drop-in replacement for react-markdown, designed specifically for AI-powered streaming. It handles incomplete or unterminated Markdown blocks gracefully during token-by-token streaming from AI models, with support for GFM, LaTeX math, Mermaid diagrams, and syntax highlighting.

## Key Points
- Drop-in replacement for react-markdown optimized for streaming
- Handles unterminated/incomplete Markdown blocks during streaming
- Supports GFM (tables, task lists), LaTeX via KaTeX, Mermaid diagrams, Shiki syntax highlighting
- Security-first with rehype-harden for safe rendering
- Powers the AI Elements Message component in ai-sdk
