# libtmux

## Summary
A typed Python API for controlling tmux programmatically. Provides object-oriented access to Server, Session, Window, and Pane objects instead of shelling out and parsing tmux output. Powers tmuxp.

## Key Points
- Typed, object-oriented control of tmux state
- Query and traverse live sessions, windows, and panes
- Context managers for automatic cleanup
- pytest plugin for isolated tmux test fixtures
- Requires tmux >= 3.2a, Python >= 3.10
