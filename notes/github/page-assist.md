# Page Assist

## Summary
Page Assist is an open-source browser extension that provides a sidebar and web UI for interacting with local AI models (via Ollama or OpenAI-compatible endpoints) from any webpage. It supports Chrome, Brave, Edge, and Firefox.

## Key Points
- Sidebar that opens on any webpage for in-context AI interaction
- Web UI mode similar to ChatGPT for standalone conversations
- "Chat With Webpage" feature for asking questions about page content
- Supports Chromium-based browsers and Firefox
- Works with Ollama and any OpenAI API-compatible endpoint (LM Studio, llamafile, etc.)
