# kubewall

## Summary
kubewall is an open-source, single-binary Kubernetes dashboard with multi-cluster management and AI integration. It supports multiple AI providers (OpenAI, Claude, Gemini, DeepSeek, Ollama, etc.) for automated troubleshooting and smart recommendations.
