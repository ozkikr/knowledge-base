# Helix - Private GenAI Stack for AI Agents

## Summary
Helix is an enterprise-grade private GenAI platform for building and deploying AI agents. It supports RAG, API calling, vision, and multi-provider LLM support through declarative YAML pipelines, with an intelligent GPU scheduler for efficient resource utilization.

## Key Points
- Deploy AI agents in private data centers or VPCs
- Declarative `helix.yaml` configuration for LLM applications
- Intelligent GPU scheduler packs models efficiently into available memory
- Supports OpenAI, Anthropic, and local models
- Session-based architecture with multi-step reasoning and tool orchestration
