# Claude Agents

## Summary
A collection of custom subagent markdown files for use with Claude Code. Agents can be installed project-wide or globally and cover various development tasks like refactoring, content writing, frontend design, and PRD writing.

## Key Points
- Drop-in `.claude/agents/` markdown files
- Available agents: code-refactorer, content-writer, frontend-designer, prd-writer
- Supports both project-specific and global installation
- 2k+ stars, popular community resource
