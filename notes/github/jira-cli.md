# JiraCLI

## Summary
A feature-rich interactive command line tool for Atlassian Jira. Supports issue search, creation, cloning, linking, transitions, and more. Inspired by GitHub CLI, it works with both Jira Cloud and on-premise installations.

## Key Points
- Interactive CLI for Jira with issue search, creation, and management
- Available via Homebrew, Nix, Docker, and downloadable binaries
- Cross-platform: Linux, macOS, Windows
- Supports both Jira Cloud and on-premise
- Inspired by GitHub CLI
