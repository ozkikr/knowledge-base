# agentic-coding
## Summary
This repository curates practical rules, templates, skills, and workflow advice for day-to-day agentic coding. It blends tool-agnostic patterns with client-specific guidance for Claude Code and Cline. The emphasis is on improving output quality through better instructions, context management, and repeatable operating patterns.
## Key Points
- Provides reusable rule sets and project-level guidance artifacts for coding agents.
- Documents planning-first workflows and context-window management tactics.
- Highlights MCP/tooling strategy tradeoffs and cost/performance considerations.
- Includes implementation tips for reducing token waste and avoiding over-engineering loops.
