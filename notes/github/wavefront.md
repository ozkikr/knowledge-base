# Wavefront

## Summary
An open-source enterprise AI middleware platform for building production-ready AI applications. Alternative to n8n, UnifyApps, and AgentGPT, it connects to any API, database, or LLM to build AI agents, workflows, and automations with enterprise-grade auth and observability.

## Key Points
- Connects to any API, database, file storage, and LLM/SLM
- Built-in authentication (Google Auth, Microsoft AD/Entra), authorization, and observability
- Used for contact center supervision, underwriting, knowledge bases, and voice agents
- Powered by the flo-ai framework (MIT licensed)
- Beta release under AGPL-3.0 license
