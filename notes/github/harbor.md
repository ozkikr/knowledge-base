# Harbor

## Summary
Harbor is a CLI tool that brings up a complete pre-wired LLM stack with hundreds of services using a single command. It uses Docker Compose to orchestrate various AI/ML services for local experimentation and development.

## Key Points
- One command deploys a full LLM infrastructure stack
- Hundreds of composable services available
- Available via npm (`@avcodes/harbor`) and PyPI (`llm-harbor`)
- Docker Compose-based orchestration
- Designed for exploring and experimenting with LLM tools locally
