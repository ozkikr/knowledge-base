# mcp-stockfish

## Summary
An MCP server that bridges AI systems to the Stockfish chess engine via the UCI protocol. Supports concurrent sessions, full UCI command set, and runs in stdio or HTTP mode. Built on mcp-go.

## Key Points
- MCP protocol bridge to Stockfish chess engine
- Supports concurrent sessions with multiple Stockfish instances
- Full UCI command support (position, go, analysis)
- Runs in stdio (default) or HTTP mode
- Docker-ready, built on mark3labs/mcp-go
