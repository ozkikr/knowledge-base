# Boring Semantic Layer (BSL)

## Summary
A lightweight Python semantic layer built on Ibis, supporting any database engine Ibis integrates with (DuckDB, Snowflake, BigQuery, PostgreSQL, etc.). Designed to be MCP-friendly for connecting LLMs to structured data.

## Key Points
- Lightweight: simple `pip install boring-semantic-layer`
- Built on Ibis for multi-database support
- Define semantic tables with dimensions and measures
- MCP-friendly for LLM-to-data connections
- Inspired by Malloy, taking the Python route
