# Bag of Words

## Summary
Bag of Words is an open-source agentic analytics platform that lets you chat, visualize, and run deep analysis on your data with memory, rules management, and observability built in. Supports any LLM (OpenAI, Anthropic, Gemini, Ollama) with any data warehouse (Snowflake, BigQuery, Postgres, Redshift).

## Key Points
- Memory system stores context and preferences down to table/column level
- Rules engine with versioning, approval workflows, and git sync
- Full observability into agent traces, plans, guardrails, and evals
- Dashboard building and sharing with team collaboration
- MCP support for use from Cursor, Claude Desktop, and other AI clients
