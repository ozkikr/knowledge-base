# Crawley - The Unix-way Web Crawler

## Summary
Crawley is a minimalist web crawler built with Unix philosophy in mind, written in Go. It provides a simple CLI interface for crawling websites and outputting results in a composable, pipe-friendly format.

## Key Points
- Follows Unix philosophy: small, composable, pipe-friendly
- Written in Go for performance and easy deployment
- Lightweight CLI tool for web crawling
- 329 stars, actively maintained
