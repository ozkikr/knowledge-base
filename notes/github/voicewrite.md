# VoiceWrite

## Summary
VoiceWrite is a private, on-device voice-to-text app for macOS. It uses Apple's SpeechAnalyzer with no cloud processing, is under 1MB with no bundled models, and works anywhere via a global hotkey.

## Key Points
- Privacy-first: all processing on-device using Apple's SpeechAnalyzer
- Lightweight under 1MB, speech models managed by macOS
- Global hotkey for dictation into any app
- Visual feedback with screen border glow during recording
- Requires macOS 26.0 (Tahoe) or later
