# gopkgview

## Summary
An interactive tool for visualizing Go project dependency graphs via a web-based interface. It uses ELK layout algorithms and React Flow to provide an interactive visualization of `go mod graph` output.

## Key Points
- Interactive web-based dependency graph visualization
- Toggle dependencies by type (stdlib, external, local)
- Focus mode for drilling into specific packages
- Multiple install options: go install, binary, Docker
- Uses ELK layout algorithms and React Flow
