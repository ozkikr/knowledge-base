# GoCVKit

## Summary
GoCVKit is a modular GoCV framework for working with live camera or video streams using configurable pipelines and filters. It provides zero-boilerplate setup, hot-reload configuration, and leak-free resource management.

## Key Points
- TOML-based pipeline configuration with hot reload
- Built-in filters: Grayscale, GaussianBlur, Canny edge detection, etc.
- MJPEG streaming server built in
- Video recording support
- Zero per-frame memory allocations via double-buffer pipeline
