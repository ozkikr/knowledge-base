# Kiln

## Summary
Kiln is a free desktop app for building, evaluating, and optimizing AI systems. It includes evals, RAG, agents, fine-tuning, synthetic data generation, dataset management, MCP support, and comprehensive model support for 100+ models.

## Key Points
- Desktop apps for Windows, macOS, and Linux
- Zero-code fine-tuning for Llama, GPT-4o, and more
- Built-in evals, RAG, agents, and synthetic data generation
- Supports 100+ models via Ollama, OpenAI, OpenRouter, Fireworks, Groq, AWS
- Can train custom reasoning models via distillation
