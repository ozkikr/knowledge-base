# gh-dash - Rich Terminal UI for GitHub

## Summary
gh-dash (DASH) is a terminal UI for GitHub that provides keyboard-driven navigation for PRs and issues. It supports user-defined sections, vim-style hotkeys, custom actions, and full GitHub operations (diff, comment, checkout, push) all controlled via YAML config.

## Key Points
- User-defined, per-repo PR and issue sections with YAML configuration
- Overridable vim-style keyboard hotkeys and custom workflow actions
- Full GitHub operations: diff, comment, checkout, push, update
- Built with Bubbletea, Lipgloss, Glamour, and Cobra (Charm ecosystem)
- Installable as a `gh` CLI extension
