# Byterover Cipher

## Summary
Cipher is an open-source memory layer for coding agents by Byterover. It's compatible with Cursor, Codex, Claude Code, Windsurf, and many other IDEs through MCP, providing persistent memory that travels across IDE switches.

## Key Points
- Dual Memory Layer: System 1 (programming concepts, business logic) and System 2 (reasoning steps)
- MCP integration with all major AI coding IDEs
- Auto-generates coding memories that scale with codebase
- Share coding memories across dev teams in real time
- Zero configuration installation via npm
