# AgenticGoKit

## Summary
An open-source agentic AI framework in Go for building, orchestrating, and deploying intelligent agents. LLM-agnostic with streaming-first API, multi-agent workflow orchestration, MCP tool discovery, and production-grade OpenTelemetry observability.

## Key Points
- Streaming-first v1beta API with multimodal support (images, audio, video)
- Multi-agent workflows: sequential, parallel, DAG, and loop patterns
- Multiple LLM providers: OpenAI, Ollama, Azure, HuggingFace
- Built-in memory and RAG with zero config (swappable with pgvector)
- MCP protocol support for tool discovery and integration
