# jsonrepair

## Summary
A library to repair invalid JSON documents. Handles common issues like missing quotes, single quotes, trailing commas, comments, Python constants, truncated JSON, and more. Supports streaming for infinitely large documents.

## Key Points
- Fixes missing quotes, escape characters, commas, and closing brackets
- Replaces single quotes, special characters, Python constants (None/True/False)
- Strips comments, JSONP notation, MongoDB data types, fenced code blocks
- Streaming API support for large documents
- Available as npm package with ESM, UMD, and CommonJS builds
