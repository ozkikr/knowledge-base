# MCP Agent

## Summary
A composable framework for building effective AI agents using Model Context Protocol (MCP). Implements all patterns from Anthropic's "Building Effective Agents" guide. Handles MCP server lifecycle management and scales from simple agents to durable Temporal-backed workflows.

## Key Points
- Full MCP support with automatic server connection lifecycle management
- Implements all Anthropic "Building Effective Agents" patterns composably
- Durable agents via Temporal integration (pause, resume, recover)
- Simple API: define agents with MCP server lists and instructions
