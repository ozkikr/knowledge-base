# Bruin

## Summary
Bruin is a data pipeline tool that lets you build pipelines with SQL and Python, ingest data from different sources, add quality checks, and build end-to-end data flows. It's a CLI-first tool written in Go with Python support.

## Key Points
- Build data pipelines using SQL and Python
- Data ingestion from multiple sources
- Built-in data quality checks
- End-to-end pipeline orchestration
- CLI-first approach with 6,480+ commits showing active development
