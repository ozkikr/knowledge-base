# Yoink

## Summary
Yoink is a Chrome and Firefox extension that extracts any website's design system (colors, typography, spacing, shadows, components, layouts, animations) into structured YAML that can be fed directly to AI coding assistants like Claude.

## Key Points
- One-click extraction of complete design systems from any website
- Outputs clean YAML format optimized for AI assistant consumption
- 100% private â€” all processing happens locally with zero network requests
- Optional component detection for detailed component-level extraction
- Available for Chrome and Firefox with minimal permissions
