# mcp-ui
## Summary
mcp-ui is an SDK suite for building and rendering interactive user interfaces over the Model Context Protocol. It aligns with the MCP Apps standard and provides server/client libraries across multiple languages. The project targets host-agnostic tool UIs with secure sandboxed rendering patterns.

## Key Points
- Implements MCP Apps UI patterns centered on `_meta.ui.resourceUri` links.
- Provides TypeScript client/server packages plus Python and Ruby server SDKs.
- Supports AppRenderer/UIResourceRenderer models for host integration.
- Includes adapter support for host-specific environments like Apps SDK contexts.
- Positions itself as production-ready for MCP UI delivery.
