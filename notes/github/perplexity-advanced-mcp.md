# perplexity-advanced-mcp
## Summary
perplexity-advanced-mcp is an MCP server package that integrates Perplexity and OpenRouter models for internet-assisted queries. It adds query-type routing, attachment support, and reliability-oriented retry behavior. The tool is aimed at editor-integrated AI workflows that need controlled web research capabilities.
## Key Points
- Supports both Perplexity and OpenRouter providers with configurable model routing.
- Distinguishes simple vs complex query modes for latency/cost trade-offs.
- Accepts file attachments as additional context in advanced query flows.
- Provides CLI-based setup with environment-variable and argument-based configuration.
