# Morphik Core

## Summary
Morphik is an AI-native toolset for visually rich documents and multimodal data. It provides end-to-end tools to ingest, search, transform, and manage unstructured documents including PDFs, images, and videos with techniques like ColPali for visual understanding.

## Key Points
- Uses ColPali for multimodal search that understands visual document content
- Fast and scalable metadata extraction including bounding boxes, labeling, classification
- Integrations with Google Suite, Slack, Confluence
- Addresses failures of traditional RAG with visually rich documents (charts, diagrams, tables)
- Available as hosted service with free tier or self-hosted open source
