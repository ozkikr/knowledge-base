# Dyad

## Summary
Dyad is a local, open-source AI app builder — an alternative to Lovable, v0, Bolt, and Replit. It runs entirely on your machine with your own API keys, offering fast, private, and vendor-lock-in-free AI-powered application development.

## Key Points
- Fully local execution — fast, private, no lock-in
- Bring your own AI API keys
- Cross-platform: Mac and Windows
- Apache 2.0 license (core), FSL 1.1 (pro features)
- Active community on Reddit (r/dyadbuilders)
