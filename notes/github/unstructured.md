# Unstructured - Document ETL for LLMs

## Summary
Unstructured provides open-source components for ingesting and pre-processing images and text documents (PDFs, HTML, Word docs, etc.) for use with LLMs. It offers modular functions and connectors that simplify transforming unstructured data into structured outputs.

## Key Points
- Supports PDFs, HTML, Word, images, and many more formats
- Modular ETL pipeline for LLM data preparation
- Available as Python library or containerized service
- Enterprise Platform available for production workflows
- Includes chunking, embedding, and enrichment capabilities
