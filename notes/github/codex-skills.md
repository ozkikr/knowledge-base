# codex-skills
## Summary
codex-skills packages a frontend-design skill for OpenAI Codex to generate higher-quality UI output. It includes design playbooks, implementation patterns, and starter assets to guide aesthetic and accessible frontend development. The goal is to improve design consistency and production readiness in generated interfaces.
## Key Points
- Provides a reusable skill folder with SKILL.md, references, and starter frontend assets.
- Supports both user-level and repo-level installation for personal or team workflows.
- Encodes multiple visual direction presets and implementation recipes.
- Designed for automatic activation on relevant frontend tasks, with explicit invocation also supported.
