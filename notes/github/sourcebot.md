# Sourcebot

## Summary
Self-hosted tool for codebase understanding with AI-powered Q&A and fast code search. Supports searching and navigating across all repos and branches regardless of hosting platform.

## Key Points
- Ask Sourcebot: natural language questions with inline citations and code snippets
- Code search with regex, filters, boolean logic across all repos/branches
- IDE-level code navigation (goto definition, find references)
- Built-in file explorer with syntax highlighting
- Experimental agents feature including PR review agent
