# Temporal Python SDK

## Summary
The official Python SDK for Temporal, a distributed, scalable, durable orchestration engine for long-running business logic. Features type safety with MyPy generics, multiple activity types (async, threaded, multiprocess), and a custom asyncio event loop for fault-tolerant workflows.

## Key Points
- Type-safe with MyPy generics for workflow/activity parameters
- Supports async, threaded, and multiprocess activities
- Custom asyncio event loop backing distributed fault-tolerant workflows
- Integration with OpenAI Agents SDK now in public preview
- Comprehensive documentation and code samples available
