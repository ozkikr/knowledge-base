# Claude Code History Viewer

## Summary
A cross-platform desktop app (macOS, Windows, Linux) to browse, search, and analyze Claude Code conversation history offline. Reads ~/.claude conversation history and provides analytics, session boards, and real-time monitoring.

## Key Points
- Offline desktop app for browsing Claude Code conversations
- Supports analytics, session boards, and real-time monitoring
- Cross-platform: macOS, Windows, Linux
- Built with Rust (has Rust test CI)
- Multi-language support (English, Korean, Japanese, Chinese)
