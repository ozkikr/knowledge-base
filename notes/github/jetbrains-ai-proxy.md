# jetbrains-ai-proxy
## Summary
jetbrains-ai-proxy bridges JetBrains AI integrations to OpenAI-compatible API endpoints. It is aimed at teams that want flexible backend routing, credential handling, and compatibility with existing model-serving infrastructure.

## Key Points
- Acts as a compatibility proxy between JetBrains AI tooling and OpenAI-style APIs.
- Supports multi-JWT and smart configuration use cases.
- Useful for self-hosted or custom model endpoint deployments.
- Built mainly in Go with deployment-friendly tooling.
