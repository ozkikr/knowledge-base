# Pyrefly - Fast Type Checker and Language Server for Python

## Summary
Pyrefly is a type checker and language server for Python by Facebook/Meta, providing lightning-fast type checking along with IDE features such as code navigation, semantic highlighting, and code completion. Available as CLI tool and extensions for VSCode, Neovim, and Zed.

## Key Points
- Lightning-fast type checking with type inference for variables and return types
- Flow types that understand program control flow to refine static types
- Module-level incrementality with optimized checking and parallelism
- IDE extensions for VSCode, Neovim, Zed with code navigation and completion
- Browser sandbox available for trying online
