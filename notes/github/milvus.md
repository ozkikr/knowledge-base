# Milvus

## Summary
Milvus is a high-performance, cloud-native vector database for scalable approximate nearest neighbor (ANN) search. Written in Go and C++ with hardware acceleration, it supports billions of vectors with real-time streaming updates.

## Key Points
- Fully distributed, K8s-native architecture for horizontal scaling
- Hardware-accelerated (CPU/GPU) vector search
- Supports standalone mode and Milvus Lite (pip install)
- Handles billions of vectors with real-time updates
- Managed version available as Zilliz Cloud
