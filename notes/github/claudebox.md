# ClaudeBox

## Summary
A Docker-based development environment for running Claude Code in fully containerized, reproducible setups. Provides pre-configured development profiles, per-project isolation, network security, and multi-instance support.

## Key Points
- Run Claude Code in isolated Docker containers with pre-configured language profiles
- Per-project isolation: separate images, auth state, history, and configs
- Development profiles for C/C++, Python, Rust, Go, and more
- Network isolation with project-specific firewall allowlists
- Multi-instance support for working on multiple projects simultaneously
