# DocsGPT

## Summary
An open-source private AI platform for building intelligent agents, assistants, and enterprise search. Features an Agent Builder, deep research tools, document analysis across many formats, multi-model support, and rich API connectivity with actionable tools.

## Key Points
- Supports PDF, DOCX, CSV, XLSX, EPUB, MD, HTML, PPTX, images and more
- Ingests from URLs, sitemaps, Reddit, GitHub, and web crawlers
- Multi-model: OpenAI, Google, Anthropic, and local models (Ollama, llama_cpp)
- Pre-built integrations: HTML/React widgets, Discord/Telegram bots
- Enterprise-grade with Kubernetes support and API key management
