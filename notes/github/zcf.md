# zcf
## Summary
ZCF (Zero-Config Code Flow) is a TypeScript CLI project for one-click setup and workflow automation around Claude Code and Codex environments. It emphasizes rapid onboarding, bilingual UX, and configurable presets for developer workflows. The project includes interactive and non-interactive installation paths.
## Key Points
- Provides guided setup/update flows via `npx zcf` commands.
- Targets Claude Code/Codex configuration, workflow scaffolding, and related integrations.
- Includes multilingual documentation and an active community/support channel.
- Designed for minimal manual setup with optional provider presets.
- Maintained as an npm package with docs and changelog support.
