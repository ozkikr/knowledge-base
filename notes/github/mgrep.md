# mgrep
## Summary
mgrep is a semantic grep-style CLI for searching code and documents using natural language. It combines local indexing with semantic retrieval to make repository exploration faster for both humans and coding agents.

## Key Points
- Offers natural-language search over local repositories and files.
- Supports semantic retrieval workflows through CLI-native commands.
- Can run background indexing to keep search context fresh.
- Designed for agent-assisted coding and low-friction developer usage.
