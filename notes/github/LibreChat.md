# LibreChat

## Summary
An enhanced, self-hostable ChatGPT clone supporting multiple AI providers (OpenAI, Anthropic, Google, Azure, AWS Bedrock, etc.). Features agents, MCP support, code interpreter, web search, and secure multi-user authentication.

## Key Points
- Supports virtually all major AI providers and OpenAI-compatible APIs
- Built-in agents with no-code custom assistant builder and marketplace
- Model Context Protocol (MCP) support for tools
- Code interpreter with sandboxed execution in multiple languages
- Web search integration with Jina reranking
