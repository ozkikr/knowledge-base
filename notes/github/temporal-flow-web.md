# Temporal Flow Web

## Summary
A developer tool for visualizing and debugging Temporal workflows through an interactive graph interface. It provides advanced search, live mode for real-time monitoring, and workflow operations like terminate/retry directly from the UI.

## Key Points
- Interactive graph visualization of parent-child workflow relationships
- Advanced search supporting partial names, workflow types, and schedulers
- Live mode for real-time workflow monitoring
- Workflow operations (terminate, retry, start as new) and batch operations
- CMD+K command palette and right-click context menus for quick access
