# Martian Proxy - Programmable HTTP Proxy for Testing

## Summary
Martian is a Go library by Google for building custom HTTP/S proxies designed for testing. It supports request/response modification, header injection, cookie manipulation, traffic verification, and HAR logging.

## Key Points
- Programmable HTTP/S proxy for testing and mocking external services
- Supports header injection, cookie modification, and request/response mutation
- Built-in HAR logging for traffic capture
- Can unwrap encrypted traffic with CA certificate installation
- Cross-platform via Go cross-compilation
