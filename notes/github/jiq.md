# jiq
## Summary
jiq is an interactive jq workbench for exploring JSON with real-time output updates as queries are edited. It combines autocomplete, query history, snippet reuse, and optional AI assistance to accelerate JSON transformation workflows. The interface supports both standard and Vim-style interaction patterns.
## Key Points
- Evaluates jq queries live while editing against file or stdin JSON streams.
- Provides context-aware autocomplete, syntax help, and error overlays.
- Adds snippet management, history search, clipboard support, and result search.
- Offers optional AI-powered query suggestions and fixes via configurable providers.
