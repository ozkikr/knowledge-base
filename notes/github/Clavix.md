# Clavix
## Summary
Clavix is a CLI/tooling project for turning prompt engineering workflows into reusable skills. It supports structured flows such as prompt improvement, PRD generation, planning, implementation, and verification. The repository is positioned as an integration layer across multiple coding-agent environments.

## Key Points
- Provides skill-based workflows for prompt quality, planning, execution, and review.
- Integrates with Agent Skills-compatible tools and multiple coding-agent platforms.
- Uses template-driven outputs and local artifacts to make workflows repeatable.
- Includes commands for setup, update, diagnostics, and system-level integration.
