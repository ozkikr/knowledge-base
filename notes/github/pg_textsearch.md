# pg_textsearch
## Summary
PostgreSQL extension for BM25 relevance-ranked full-text search in Postgres.

## Key Points
- Adds ranked lexical search capabilities with BM25-style scoring.
- Designed for performant top-k retrieval and large-scale Postgres use.
- Targets modern search workloads while staying within native Postgres workflows.
