# kagent

## Summary
kagent is a Kubernetes-native framework for building, deploying, and managing AI agents. It represents agents as Kubernetes custom resources with system prompts, tools, and LLM configurations, supporting multiple providers (OpenAI, Anthropic, Google, Ollama) and MCP tool servers.

## Key Points
- Agents defined as Kubernetes CRDs with system prompts, tools, and LLM configs
- Supports multiple LLM providers including OpenAI, Anthropic, Google Vertex AI, and Ollama
- MCP tool servers for Kubernetes, Istio, Helm, Argo, Prometheus, Grafana, Cilium
- Built-in OpenTelemetry tracing for agent observability
