# npcpy

## Summary
npcpy is a flexible Python agent framework for building AI applications and research with LLMs. It supports local and cloud providers, multi-agent teams, tool calling, multimodal generation, knowledge graphs, and fine-tuning.

## Key Points
- Create agents with personas, directives, and custom tools
- Supports local (Ollama) and cloud LLM providers
- Multi-agent teams with coordination capabilities
- Built-in support for image/audio/video generation and knowledge graphs
- Simple API: NPC class with `get_llm_response()` and tool integration
