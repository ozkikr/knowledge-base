# Opik

## Summary
An open-source platform by Comet for debugging, evaluating, and monitoring LLM applications, RAG systems, and agentic workflows. Provides comprehensive tracing, automated evaluations, prompt optimization, and production-ready dashboards.

## Key Points
- Deep tracing of LLM calls, conversations, and agent activity
- LLM-as-a-judge evaluation and experiment management
- Opik Agent Optimizer for enhancing prompts and agents
- Opik Guardrails for safe and responsible AI practices
- Scalable production monitoring dashboards with online evaluation rules
