# Claw
## Summary
Native macOS wrapper around Claude Code SDK/CLI to provide a desktop app experience for AI-assisted coding.

## Key Points
- Offers plan mode, MCP support, and Xcode-friendly workflows.
- Uses local Claude CLI credentials and supports release/distribution automation files.
- Aims to bring Claude coding capabilities into a more integrated macOS UI.
