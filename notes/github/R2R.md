# R2R - Production-Ready AI Retrieval System

## Summary
R2R is an advanced AI retrieval system supporting Retrieval-Augmented Generation (RAG) with a RESTful API. It offers multimodal content ingestion, hybrid search, knowledge graphs, document management, and a Deep Research API for complex multi-step reasoning queries.

## Key Points
- Production-ready RAG with RESTful API and multimodal content ingestion
- Deep Research API for multi-step reasoning across knowledgebase and internet
- Supports hybrid search and knowledge graphs
- Quick start via pip install or Docker deployment
- Compatible with multiple LLM providers including Anthropic and OpenAI
