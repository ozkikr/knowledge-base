# OpenAI Agents SDK

## Summary
A lightweight yet powerful framework for building multi-agent workflows by OpenAI. Provider-agnostic, supporting OpenAI Responses/Chat Completions APIs and 100+ LLMs, with built-in handoffs, guardrails, sessions, and tracing.

## Key Points
- Core concepts: Agents, Handoffs (control transfer), Guardrails (safety checks), Sessions, Tracing
- Provider-agnostic: works with 100+ LLMs beyond OpenAI
- Automatic conversation history management across agent runs
- Optional voice and Redis session support
- Python 3.10+ required
