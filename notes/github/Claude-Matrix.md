# Claude-Matrix - Tooling System for Claude Code

## Summary
Community plugin that turns Claude Code into a complete development environment with persistent memory, code indexing across 15 languages, automated hooks, scheduled tasks, and code review capabilities. Requires Bun v1.0+ and Claude Code v2.0+.

## Key Points
- Persistent memory with semantic search across sessions
- Code indexing across 15 languages (definitions, callers, exports)
- Automated hooks: context injection, bad package detection, sensitive file warnings
- Dreamer: scheduled tasks for daily reviews, weekly audits, automated commits
- 5-phase code review with blast radius and impact mapping
- macOS and Linux only (Windows via WSL)
