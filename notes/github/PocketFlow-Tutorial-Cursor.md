# PocketFlow Tutorial: Build Cursor with Cursor

## Summary
A meta tutorial project that uses Cursor AI to build an open-source, customizable AI coding agent ("Cursor Agent") that writes, reviews, and refactors code. Built using PocketFlow, a 100-line LLM framework.

## Key Points
- Uses Cursor to build a Cursor-like AI coding agent (meta approach)
- Powered by PocketFlow, a minimal 100-line LLM framework
- Demonstrates "Agentic Coding" paradigm: humans design, agents code
- Includes example SaaS homepage project for the agent to edit
- Intentionally kept simple for learning purposes
