# RD-Agent: AI-Driven Research & Development Automation

## Summary
Microsoft's RD-Agent automates data-driven R&D processes, letting AI drive data-driven AI. It leads as the top-performing ML engineering agent on MLE-bench and supports scenarios including Kaggle competitions, quantitative trading, and general data science tasks.

## Key Points
- Top-performing ML engineering agent on MLE-bench
- Automates data and model R&D processes end-to-end
- RD-Agent-Quant variant for quantitative trading (NeurIPS 2025)
- Supports LiteLLM backend for multiple LLM providers
- Kaggle agent and general data science agent scenarios
