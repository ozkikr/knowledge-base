# Osaurus - AI Edge Infrastructure for macOS

## Summary
Osaurus is an AI edge infrastructure runtime for macOS (Apple Silicon). It runs local or cloud models, shares tools across apps via MCP, and powers AI workflows with a native, always-on runtime. Compatible with OpenAI, Anthropic, Ollama APIs and Apple Foundation Models.

## Key Points
- Native macOS runtime for Apple Silicon with always-on background operation
- Compatible with OpenAI, Anthropic, Ollama, and Apple Foundation Models APIs
- MCP server for sharing tools across applications
- Supports both local and cloud model execution
- Open-source with Apache/MIT licensing
