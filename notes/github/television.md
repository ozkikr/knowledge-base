# Television (tv)

## Summary
Television is a very fast, portable, and hackable fuzzy finder for the terminal written in Rust. It lets you search in real-time through various data sources like files, text, git repositories, environment variables, and Docker containers.

## Key Points
- Fast fuzzy finding across multiple data source "channels" (files, text, git repos, env vars, docker)
- Written in Rust for high performance and portability
- Extensible channel system for different data sources
- Real-time search with instant feedback
- Hackable and customizable
