# Firecrawl

## Summary
Firecrawl is an API service that scrapes, crawls, and extracts structured data from any website, turning it into LLM-ready markdown or structured data. It handles JavaScript rendering, proxies, and dynamic content with industry-leading reliability.

## Key Points
- Outputs clean markdown, structured JSON, screenshots, and HTML for LLM consumption
- Handles proxies, JS rendering, and dynamic content automatically
- Supports actions (click, scroll, input, wait) before extraction
- Batch processing for thousands of URLs asynchronously
- Media parsing for PDFs, DOCX, and images
