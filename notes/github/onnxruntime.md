# ONNX Runtime

## Summary
ONNX Runtime is a cross-platform inference and training machine-learning accelerator from Microsoft. It supports models from PyTorch, TensorFlow/Keras, scikit-learn, LightGBM, XGBoost and more, with hardware acceleration and graph optimizations across different platforms and devices.

## Key Points
- Cross-platform ML inference and training accelerator
- Supports deep learning (PyTorch, TensorFlow) and classical ML (scikit-learn, LightGBM, XGBoost)
- Hardware-accelerated with graph optimizations and transforms
- Training acceleration for transformer models on multi-node NVIDIA GPUs
- Available for multiple languages: Python, C++, C#, Java, JavaScript
