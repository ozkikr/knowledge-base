# kimi-agent-sdk
## Summary
Kimi Agent SDK exposes the Kimi CLI runtime as language-native SDKs for application development. It enables programmatic orchestration of sessions, tool calls, and approval workflows in Go, Node.js, and Python.

## Key Points
- Provides SDKs in multiple languages for integrating Kimi agent workflows.
- Reuses CLI configuration, tools, and MCP integrations.
- Supports streaming responses and interactive approval handling.
- Useful for embedding agent behavior into custom products and automations.
