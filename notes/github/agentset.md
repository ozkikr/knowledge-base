# Agentset

## Summary
Open-source platform for building, evaluating, and shipping production-ready RAG and agentic applications. Provides end-to-end tooling including ingestion, vector indexing, evaluation/benchmarks, chat playground, hosting, and developer-friendly API. Built with Next.js, AI SDK, Prisma, Supabase, and Trigger.dev.

## Key Points
- Turnkey RAG: ingestion, chunking, embeddings, and retrieval out of the box
- Model agnostic: works with any LLM, embeddings, and vector DB
- Built-in multi-tenancy and production hosting with custom domains
- Chat playground with message editing and citations
- API + typed SDKs with OpenAPI spec
- Born from real production experience processing 5M+ documents
