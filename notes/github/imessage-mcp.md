# iMessage MCP

## Summary
A Deno monorepo providing read-only access to the macOS iMessage database via MCP. Enables LLMs to search messages, list chats, retrieve contacts, and correlate with macOS Contacts app data.

## Key Points
- MCP server for LLM integration with iMessage
- Search messages by text, contact, or date range
- List all chats/conversations and contacts
- Integrates with macOS Contacts for handle ID correlation
- Read-only access to ~/Library/Messages/chat.db
