# PII Masker

## Summary
An open-source tool for detecting and masking Personally Identifiable Information using DeBERTa-v3. Provides high-precision detection of names, addresses, phone numbers and more, with a simple Python API for integration into data workflows.

## Key Points
- Powered by DeBERTa-v3 with 1024 token support for longer documents
- Detects multiple PII types: names, addresses, phone numbers, etc.
- Returns both masked text and structured PII dictionary
- Simple Python API for easy integration
- Models available on HuggingFace
