# MCPretentious
## Summary
MCPretentious is an MCP server for controlling terminal sessions, with iTerm2 and tmux backends. It is aimed at robust terminal puppeteering for AI-driven development and operations tasks. The project focuses on low-friction terminal interaction primitives like typing, screen capture, resizing, and mouse events.
## Key Points
- Supports both iTerm2 (WebSocket API) and tmux for cross-platform workflows.
- Exposes terminal lifecycle and interaction tools through MCP.
- Includes optimized screenshot/screen-read features for agent feedback loops.
- Designed for TUI testing and remote shell automation scenarios.
