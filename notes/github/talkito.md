# TalkiTo

## Summary
TalkiTo lets developers interact with AI coding assistants (Claude Code, OpenAI Codex) through speech. Available as CLI tool, web extension, and Python library supporting terminal, Slack, and WhatsApp channels.

## Key Points
- Voice interface for Claude Code and Codex CLI (fully supported)
- Web extension for bolt.new, v0.dev, replit (output only)
- Install via `pip install talkito` or one-liner script
- Supports MCP SSE server mode with Chrome extension
- Requires portaudio for audio capture on macOS
