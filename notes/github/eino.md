# Eino

## Summary
The ultimate LLM/AI application development framework in Go by CloudWeGo. Inspired by LangChain and Google ADK, Eino provides reusable components, an Agent Development Kit, and graph-based composition following Golang conventions.

## Key Points
- Go-native LLM framework with components like ChatModel, Tool, Retriever, ChatTemplate
- Agent Development Kit (ADK) with tool use, multi-agent coordination, human-in-the-loop
- Graph/workflow composition that can run standalone or as agent tools
- Official integrations for OpenAI, Ollama, and more
- Pronounced "aino"
