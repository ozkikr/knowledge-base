# mq - Markdown Query

## Summary
A jq-like command-line tool for processing Markdown files, written in Rust. Enables slicing, filtering, mapping, and transforming structured Markdown content, particularly useful for LLM workflows where Markdown is the primary input format.

## Key Points
- jq-inspired syntax for querying and transforming Markdown documents
- Built in Rust with REPL, debugger, and VSCode/LSP support
- Especially useful for LLM input generation and prompt manipulation
- Extensible with custom functions and external subcommands
- Supports batch processing across multiple Markdown files
