# MLX LM

## Summary
MLX LM is Apple's Python package for generating text and fine-tuning large language models on Apple silicon using the MLX framework. It integrates with Hugging Face Hub and supports thousands of models with simple commands.

## Key Points
- Text generation and fine-tuning (LoRA and full) on Apple silicon
- Integrated with Hugging Face Hub â€” thousands of models available via MLX Community
- Supports quantization and model uploading to Hugging Face
- Distributed inference and fine-tuning with `mx.distributed`
- Installable via pip or conda
