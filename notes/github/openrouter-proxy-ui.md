# OpenRouter Proxy UI

## Summary
A proxy server with a UI for OpenRouter, allowing users to intercept and manage LLM API requests. Built with Python, it provides a visual interface for monitoring and controlling traffic to OpenRouter's model API.

## Key Points
- Provides a UI on top of OpenRouter proxy functionality
- Intercepts and manages LLM API calls
- 82 commits on main branch, actively developed
- Includes test suite and documentation
- Supports multiple AI coding assistant configs (.junie, .roo, .trae)
