# Tock

## Summary
Tock is a command-line time tracking tool written in Go. It stores activity logs as plaintext files and provides an interactive terminal UI (Bubble Tea) for viewing time with calendar views, themes, and iCal export.

## Key Points
- Plaintext file format for activity storage (compatible with Bartib and TimeWarrior)
- Interactive TUI with calendar view built on Bubble Tea
- Supports notes, tags, and customizable color themes
- iCal export and macOS system calendar sync
- Single binary, no dependencies; installable via Homebrew or `go install`
