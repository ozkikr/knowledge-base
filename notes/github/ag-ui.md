# AG-UI - The Agent-User Interaction Protocol

## Summary
AG-UI is an open, lightweight, event-based protocol that standardizes how AI agents connect to user-facing applications. Built by the CopilotKit team, it enables seamless integration between AI agents, real-time user context, and user interfaces.

## Key Points
- Open event-based protocol for agent-to-UI communication
- Standardizes how AI agents connect to frontend applications
- Built for simplicity and flexibility with real-time user context
- Created by the CopilotKit team with active Discord community
- Includes AG-UI Dojo for learning and testing
