# canvas-chat
## Summary
Visual non-linear chat interface where conversations are represented as nodes on an infinite canvas.

## Key Points
- Models discussions as branching/merging graph structures rather than linear threads.
- Designed for exploratory thinking and topic decomposition.
- Provides an interactive UI-first approach to structured conversation workflows.
