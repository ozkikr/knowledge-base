# advanced-imessage-kit
## Summary
advanced-imessage-kit is a TypeScript SDK for programmatic iMessage automation on macOS, including send/read flows and real-time events. It targets advanced messaging use cases like bot workflows, scheduling, reactions, attachments, and conversation management. The API surface is broad and geared toward building robust message-driven applications.
## Key Points
- Provides message send/reply/edit/unsend and reaction APIs.
- Supports chats, contacts, attachments, polls, and event listeners.
- Includes examples for automation patterns and message lifecycle handling.
- Designed for local server-backed integrations in AI/automation systems.
