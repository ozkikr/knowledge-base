# Search Input Query

## Summary
A search query language parser with SQL output support, inspired by Elasticsearch and Tantivy query syntax. Includes a multi-pass recursive descent parser with comprehensive type checking, rich error reporting, and a React component with Monaco editor integration.

## Key Points
- Multi-pass recursive descent parser for search queries
- Generates SQL output with multiple search strategies
- Supports typed field schemas (string, number, date, boolean)
- React component with Monaco editor integration
- Complete test coverage with rich error reporting and position info
