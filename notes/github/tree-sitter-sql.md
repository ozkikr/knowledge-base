# tree-sitter-sql
## Summary
tree-sitter-sql provides a SQL grammar for the tree-sitter parsing ecosystem. This fork emphasizes generated parser artifacts and WASM compatibility improvements. It is used in editor tooling such as SQL language support for Zed.

## Key Points
- Delivers a maintained SQL grammar for tree-sitter consumers.
- Includes generated parser files (`grammar.json`, `parser.c`) in-repo.
- Updates scanner behavior for better WebAssembly compatibility.
- Serves downstream editor integrations for SQL syntax features.
