# Context-Engine

## Summary
Context-Engine is an agentic context compression suite available as an MCP bridge and VS Code extension. It helps manage and compress context for AI coding workflows. Currently migrating to cloud-hosted infrastructure.

## Key Points
- MCP-based context compression for AI agents
- VS Code extension for IDE integration
- Features memory management, CLI tool (ctx), and configurable compression
- Currently migrating to cloud infrastructure (extension temporarily unavailable)
- Available as npm package `@context-engine-bridge/context-engine-mcp-bridge`
