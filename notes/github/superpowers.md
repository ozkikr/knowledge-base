# Superpowers

## Summary
Superpowers is an agentic skills framework and software development methodology for coding agents. It implements a brainstorm → plan → implement workflow with composable skills that trigger automatically, enabling autonomous multi-hour coding sessions with subagent-driven development.

## Key Points
- Complete workflow: brainstorm spec → chunk review → implementation plan → subagent execution
- Emphasizes TDD, YAGNI, and DRY principles
- Installs as a Claude Code plugin via marketplace
- Supports Claude Code, Codex, and OpenCode
- Enables hours of autonomous agent work without deviation
