# MobileBuilder_pub
## Summary
MobileBuilder_pub is a multi-agent web interface for coding CLIs including Claude Code, Cursor Agent, Gemini CLI, and Codex. It emphasizes cross-device usage with PWA installation, live streaming chat, and persistent session history. The project is currently in early alpha and actively solicits testing feedback.
## Key Points
- Unified interface for multiple coding-agent backends.
- Includes PWA support for mobile-friendly installation and scaling.
- Provides real-time response streaming and multi-turn session continuity.
- Supports multi-device synchronization and database-backed session storage.
- Built as a Python web app with front-end assets for responsive UI.
