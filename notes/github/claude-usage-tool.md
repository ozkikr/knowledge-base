# claude-usage-tool
## Summary
claude-usage-tool is a lightweight macOS menu bar app for monitoring Claude subscription and API usage. It helps heavy Claude users keep visibility on limits and credits without repeatedly checking dashboards.
## Key Points
- Displays Claude plan usage and API credit information from the menu bar.
- Built for quick-glance monitoring to avoid surprise limit hits.
- Targets power users who frequently switch between Claude app/API workflows.
- Desktop-focused UX for low-friction usage tracking.
