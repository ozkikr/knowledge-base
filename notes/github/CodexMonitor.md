# CodexMonitor
## Summary
CodexMonitor is a Tauri application for orchestrating multiple Codex agents across local workspaces. It provides a project-centric UI for threads, prompts, approvals, and execution state. The tool also integrates developer workflows like Git diffs, GitHub issues/PRs, and workspace-level controls.

## Key Points
- Manages multiple workspaces, threads, worktrees, and remote backend modes.
- Includes rich composer features like autocomplete, attachments, and model/control toggles.
- Exposes Git and GitHub operations, including branch actions and PR context workflows.
- Offers responsive UI layouts with persistent panel sizing and terminal dock support.
- Includes experimental mobile/iOS pathways with remote daemon access patterns.
