# terminal-ai

## Summary
terminal-ai is a lightweight, unopinionated AI CLI for scripts, pipelines, and automation. It supports multiple providers (OpenAI, Gemini, Claude, Ollama, LiteLLM) and serves as a universal client for MCP, A2A, and other AI protocols.

## Key Points
- Supports OpenAI, Gemini, Claude, Ollama, MSTY, LiteLLM and more
- Designed for shell scripting and pipeline integration
- Universal client for MCP and A2A protocols
- Features: file attachments, image input, clipboard copy, code generation, script execution
- Installed via npm (`@dwmkerr/terminal-ai`)
