# Davia

## Summary
An open-source tool designed for AI coding agents to generate interactive internal documentation. Creates docs with interactive visualizations and editable whiteboards viewable in a Notion-like platform or locally in your IDE.

## Key Points
- AI coding agents generate interactive documentation automatically
- Supports Cursor, GitHub Copilot, Windsurf, Claude Code, Augment
- Interactive visualizations and Excalidraw-style whiteboards
- Local-first with optional remote sync for team collaboration
- Simple CLI: `davia init`, `davia open`, `davia push`
