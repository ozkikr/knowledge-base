# LightRAG: Simple and Fast Retrieval-Augmented Generation

## Summary
LightRAG is a graph-based RAG system published at EMNLP 2025. It combines knowledge graph extraction with retrieval-augmented generation for efficient and accurate information retrieval, supporting reranking, document deletion with KG regeneration, and integration with multiple LLM providers.

## Key Points
- Graph-based RAG with knowledge graph extraction from documents
- Reranker support for boosted mixed-query performance (default mode)
- Document deletion with automatic KG regeneration
- Scalability enhancements for large-scale datasets
- Foundation for RAG-Anything multimodal extension
