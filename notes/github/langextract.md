# LangExtract

## Summary
Google Python library for extracting structured information from unstructured text using LLMs with precise source grounding. Maps every extraction to its exact location in the source text with interactive visualization.

## Key Points
- Precise source grounding: maps extractions to exact text locations with visual highlighting
- Optimized for long documents via chunking, parallel processing, and multi-pass for higher recall
- Generates interactive HTML visualization for reviewing extracted entities
- Flexible LLM support: Gemini, OpenAI, Ollama local models
- Enforces consistent output schema via few-shot examples and controlled generation
