# AionUi

## Summary
AionUi is a free, local, open-source GUI for coworking with AI coding agents including Gemini CLI, Claude Code, Codex, OpenCode, Qwen Code, Goose CLI, and more. It provides a visual graphical interface with multi-model support and local data security.

## Key Points
- Unified GUI for multiple AI coding CLIs (Gemini CLI, Claude Code, Codex, Qwen Code, etc.)
- Cross-platform: macOS, Windows, Linux

- 24/7 cowork mode and OpenClaw-like functionality
- Multi-language support (English, Chinese, Japanese, Korean, Spanish, Portuguese, Turkish)
