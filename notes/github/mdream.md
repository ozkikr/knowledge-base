# mdream

## Summary
The fastest pure JavaScript HTML-to-Markdown converter, optimized for LLM input with ~50% fewer tokens than traditional converters. Zero dependencies, 6kB gzipped, supports streaming, and is 3x faster than Turndown.

## Key Points
- 3x faster than Turndown, converts 1.8MB HTML in ~60ms
- Optimized for LLMs: ~50% fewer tokens in output
- 6kB gzip, zero dependency core
- LangChain-compatible Markdown text splitter for single-pass chunking
- Extensible plugin system; CLI crawler, Docker, GitHub Actions, Vite support
