# STORM - LLM-Powered Knowledge Curation System

## Summary
STORM (Synthesis of Topic Outlines through Retrieval and Multi-perspective Question Asking) is a Stanford research project that automatically researches a topic and generates full-length reports with citations. It also includes Co-STORM for human-AI collaborative knowledge curation.

## Key Points
- Generates comprehensive research reports with citations from web sources
- Co-STORM enables human-AI collaborative knowledge curation (EMNLP 2024)
- Supports multiple search backends: You.com, Bing, custom vector stores
- LiteLLM integration for flexible LLM provider support
- Available as `knowledge-storm` pip package
