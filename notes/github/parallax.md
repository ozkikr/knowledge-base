# Parallax - Distributed Model Serving Framework

## Summary
Parallax by GradientHQ is a distributed model serving framework that lets you build your own AI cluster anywhere. It enables distributed inference across multiple machines for serving large language models.

## Key Points
- Build distributed AI clusters across multiple machines
- Distributed model serving for large language models
- Docker support with dedicated build configurations
- Active development with 400+ commits
- 1.1k+ GitHub stars
