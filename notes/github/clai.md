# clai - Command Line Artificial Intelligence

## Summary
clai is a command-line tool that acts as a local LLM context-feeder. Written in Go, it allows users to pipe and feed context to various LLM providers directly from the terminal, making AI interactions seamless within CLI workflows.

## Key Points
- Local-first CLI tool for interacting with LLMs
- Supports profile-based configuration for different use cases
