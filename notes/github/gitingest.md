# gitingest
## Summary
gitingest converts git repositories or local directories into prompt-friendly text digests for LLM workflows. It provides both CLI and Python interfaces for generating structured repository extracts and statistics. The project also offers web and browser-extension entry points for quick URL-based ingestion.
## Key Points
- Generates digest outputs with repository tree, content extraction, and token-oriented summaries.
- Works on local directories, repository URLs, and selected subpaths.
- Includes private-repo support via GitHub token configuration.
- Offers optional server/self-host setup for hosted or team workflows.
