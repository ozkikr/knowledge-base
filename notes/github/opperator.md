# Opperator - Framework for Building Local AI Agents

## Summary
Opperator is a framework for building and running general AI agents locally from your terminal. It provides a managed runtime with a built-in daemon, code generation from templates, secrets management, and a customizable TUI for control.

## Key Points
- Agents run locally with direct access to files, local AI models, and personal data
- Built-in daemon manages agent runtime and lifecycle
- Code generation from templates with scaffolding
- Managed secrets and API key storage
- Customizable TUI for controlling agents from terminal
