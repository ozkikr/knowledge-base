# agent-slack

## Summary
agent-slack is a Slack automation CLI built for AI agents and tool-driven workflows. It provides compact JSON outputs for message retrieval, thread traversal, search, reactions, and canvas export. The project focuses on low-friction authentication and agent-friendly artifact handling.

## Key Points
- Supports read/write operations for messages, threads, reactions, users, files, and canvases.
- Optimizes outputs for token efficiency by pruning null/empty fields.
- Automatically downloads Slack file attachments and returns local file paths.
- Includes authentication helpers for Slack Desktop, browser, and token-based modes.
- Ships a reusable skill package for coding agents and assistant frameworks.
