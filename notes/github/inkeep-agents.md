# Inkeep Agents - No-Code Visual Builder + TypeScript SDK

## Summary
Inkeep Agents provides both a no-code visual drag-and-drop builder and a TypeScript SDK for creating AI agents, with full 2-way sync between the two. Designed for shipping AI assistants and multi-agent workflows for customer experience, internal copilots, and workflow automation.

## Key Points
- Dual interface: no-code visual canvas and TypeScript SDK with full bidirectional sync
- Sub-agent composition with MCP tool integration
- Use cases: customer support agents, internal copilots, CRM automation, ticket triage
- Technical and non-technical teams can collaborate on same agents
- Open-source agent framework with hosted platform option
