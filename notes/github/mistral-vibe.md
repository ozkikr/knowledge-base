# Mistral Vibe

## Summary
Mistral Vibe is Mistral's open-source CLI coding assistant powered by Mistral's models. It provides a conversational interface to codebases with natural language exploration, modification, and interaction through built-in tools, subagents, and a skills system.

## Key Points
- Open-source CLI coding assistant by Mistral AI
- Built-in agents and subagent task delegation
- Skills system for custom slash commands
- Trust folder system for security management
- Installable via curl one-liner, uv, or pip
