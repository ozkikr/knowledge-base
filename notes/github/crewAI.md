# CrewAI

## Summary
A lean, lightning-fast Python framework for orchestrating role-playing, autonomous AI agents. Built entirely from scratch (independent of LangChain), it enables collaborative intelligence through Crews (autonomous) and Flows (event-driven enterprise architecture).

## Key Points
- Independent framework, not built on LangChain
- CrewAI Crews for autonomous collaborative intelligence
- CrewAI Flows for enterprise event-driven, granular control
- 100,000+ developers certified through community courses
- AMP Suite with tracing, observability, unified control plane
