# MinerU

## Summary
MinerU transforms complex documents like PDFs into LLM-ready markdown/JSON for agentic workflows. It supports multiple hardware acceleration platforms and features cross-page table merging, OCR with automatic EXIF orientation correction, and a simplified installation process.

## Key Points
- Converts PDFs and complex documents to markdown/JSON for LLM consumption
- Supports numerous domestic and international hardware acceleration platforms (Ascend, METAX, etc.)
- Cross-page table merging with improved quality
- Automatic EXIF orientation correction for better OCR accuracy
- Simplified installation via `uv pip install mineru[all]`
