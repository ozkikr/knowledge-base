# sqlit
## Summary
sqlit is a terminal UI database client described as a “lazygit for SQL databases.” It provides fast connection, querying, and exploration workflows across many SQL backends from a unified TUI.
## Key Points
- Supports many providers including PostgreSQL, MySQL, SQLite, SQL Server, and cloud engines.
- Provides TUI-centric query workflow with history and keyboard-friendly controls.
- Designed for rapid inspection/filtering of large result sets in terminal environments.
- Targets developers and operators who prefer command-line database workflows.
