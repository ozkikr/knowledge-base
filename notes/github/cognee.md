# Cognee

## Summary
An open-source knowledge engine that transforms raw data into persistent and dynamic AI memory for agents. Combines vector search, graph databases, and self-improvement to make documents searchable by meaning and connected by relationships.

## Key Points
- Interconnects any data type: conversations, files, images, audio transcriptions
- Replaces traditional DB lookups with unified knowledge engine (graphs + vectors)
- Pythonic data pipelines supporting 30+ data sources
- Highly customizable: user-defined tasks, modular pipelines, built-in search
- Simple API: knowledge creation in ~6 lines of code
