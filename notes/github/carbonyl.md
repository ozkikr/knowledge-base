# Carbonyl

## Summary
Carbonyl is a Chromium-based browser built to run in a terminal. It supports virtually all Web APIs including WebGL, WebGPU, audio/video playback, and animations. It starts in under a second, runs at 60 FPS, idles at 0% CPU, and works over SSH without a window server.

## Key Points
- Full Chromium engine running in terminal with rich rendering
- Supports WebGL, WebGPU, audio, video playback
- 60 FPS, sub-second startup, 0% idle CPU
- Available via Docker, npm, or binary downloads
- Originally started as html2svg project
