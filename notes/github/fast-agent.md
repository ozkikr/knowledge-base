# fast-agent

## Summary
fast-agent enables creating and interacting with sophisticated multimodal agents and workflows using MCP. It's CLI-first with optional TUI, supporting Anthropic, OpenAI, Google providers plus many others via TensorZero, with complete end-to-end tested MCP feature support including Sampling and Elicitations.

## Key Points
- First framework with complete MCP feature support (Sampling + Elicitations)
- Declarative syntax for composing prompts and MCP servers
- Supports structured outputs, PDF, vision, passthrough and playback LLMs
- Recent features: Agent Skills (SKILL.md), MCP-UI, OpenAI Apps SDK (Skybridge), Shell Mode
- Native multi-provider support with comprehensive model coverage
