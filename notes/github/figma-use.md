# figma-use
## Summary
figma-use is a CLI toolkit for controlling Figma with both imperative commands and JSX-based declarative rendering. It enables read/write design automation workflows suitable for AI agents and scripted pipelines. The project includes broad feature coverage for layout, components, exports, diffs, and analysis.
## Key Points
- Supports direct object creation/editing plus JSX rendering via stdin or .figma.tsx files.
- Includes advanced capabilities such as variants, variables/tokens, queries, and linting.
- Offers export/diff flows (including JSX and Storybook-oriented outputs).
- Designed to work through Figma remote debugging without plugin installation.
