# Chatterbox TTS - State-of-the-Art Open-Source TTS

## Summary
Chatterbox is a family of three open-source text-to-speech models by Resemble AI. The flagship Chatterbox-Turbo is a 350M parameter model with paralinguistic tags ([laugh], [cough], etc.), optimized for low-latency voice agents. Also includes a 500M multilingual model supporting 23+ languages.

## Key Points
- Chatterbox-Turbo: 350M params, English, paralinguistic tags, optimized for voice agents
- Chatterbox-Multilingual: 500M params, 23+ languages, zero-shot voice cloning
- Distilled speech-token-to-mel decoder reduces generation from 10 steps to 1
- Native paralinguistic tags for realistic speech ([cough], [laugh], [chuckle])
- Open-source with HuggingFace demos available
