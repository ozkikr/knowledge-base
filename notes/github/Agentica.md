# Agentica

## Summary
Agentica is an open-source AI coding assistant by GenLabs (related to Kilo Code) for planning, building, and fixing code. It merges features from open-source projects while building its own vision, supporting VS Code and JetBrains IDEs.

## Key Points
- Open-source AI coding assistant with 30k+ commits
- Supports VS Code and JetBrains via extensions
- Includes benchmarking suite
- Monorepo with CLI, apps, and packages
- Frequently merges upstream open-source contributions
