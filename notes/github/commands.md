# commands
## Summary
wshobson/commands is a production-oriented command library for Claude Code with both single-purpose tools and orchestration workflows. It aims to standardize common software engineering operations through reusable command primitives. The project is especially useful for teams wanting repeatable multi-step automation patterns.
## Key Points
- Ships dozens of commands split across workflow orchestration and focused utility tools.
- Covers development, testing, security, infrastructure, documentation, and operational tasks.
- Supports prefix-based invocation patterns and alternative flat command setup.
- Includes guidance for extending the command set with consistent naming and architecture.
