# coreml-cli
## Summary
coreml-cli is a native macOS command-line tool for inspecting, running, benchmarking, and compiling Apple Core ML models. It provides a Python/Xcode-free workflow for model operations directly from terminal environments.

## Key Points
- Supports model inspection for inputs, outputs, structure, and metadata.
- Runs inference on image/text/JSON inputs with selectable compute targets.
- Provides batch processing and benchmarking for latency/throughput analysis.
- Compiles .mlmodel assets into optimized .mlmodelc bundles.
- Ships as Homebrew package, release binary, and build-from-source option.
