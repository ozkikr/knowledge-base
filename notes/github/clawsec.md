# clawsec
## Summary
ClawSec is a security skill suite for OpenClaw-family agents focused on hardening agent behavior and artifacts. It combines integrity checks, advisory feeds, audit tooling, and package verification in a single installable suite. The project also automates CVE ingestion and release validation for ongoing security maintenance.
## Key Points
- Provides suite-level install, health checks, updates, and checksum verification.
- Monitors drift/tampering in critical files such as SOUL.md and identity/config artifacts.
- Maintains a live advisory feed combining NVD CVEs and community incident reports.
- Includes offline validators and packaging tools for local skill development.
