# Deeploy

## Summary
Deeploy is a self-hosted, open-source deployment platform positioned as an alternative to Heroku, Vercel, and Netlify. It features a terminal-first TUI client that can manage multiple VPS servers via profiles, with an Apache 2.0 license.

## Key Points
- Terminal-first deployment with a TUI client
- Self-hosted alternative to Heroku/Vercel/Netlify
- One TUI can manage multiple VPS servers via profiles
- Server installed via curl script on VPS, TUI installed locally
- Uses Traefik for routing, Docker Compose for local dev
