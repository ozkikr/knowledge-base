# neatocal
## Summary
NeatoCal is a dependency-free, single-page yearly calendar generator designed for printing and quick customization. It is a JavaScript port of Neatnik's Calendar with expanded URL-driven configuration options. The project supports multiple layouts, localization controls, and optional moon-phase and event data features.
## Key Points
- Generates a full-year printable calendar on one page.
- Supports many URL parameters for layout, language, styling, and calendar behavior.
- Includes optional JSON data-file overlays for custom text and cell coloring.
- Supports drag-and-drop ICS imports for event visualization.
- Provides live demos and preset links for common calendar styles.
