# LazySQL

## Summary
LazySQL is a cross-platform TUI database management tool written in Go, inspired by Lazygit. It provides a terminal-based interface for interacting with SQL databases, designed for developers who live in the terminal.

## Key Points
- Inspired by Lazygit's UX philosophy applied to database management
- Cross-platform TUI built with Go
- Supports multiple database backends
- Currently in alpha stage but used daily by the author in production
- First-time Go project by a JavaScript developer learning a new language
