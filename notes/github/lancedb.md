# LanceDB - Multimodal AI Lakehouse

## Summary
LanceDB is an open-source, developer-friendly embedded retrieval library for multimodal AI. Built on the Lance columnar format, it supports fast vector search over petabytes of multimodal data including text, images, videos, and point clouds.
