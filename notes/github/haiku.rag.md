# Haiku RAG - Opinionated Agentic RAG

## Summary
Haiku RAG is an opinionated agentic RAG framework built on LanceDB, Pydantic AI, and Docling. It features hybrid search, multi-agent research workflows, conversational RAG, and an MCP server, with a local-first embedded database approach.

## Key Points
- Hybrid search with vector + full-text and Reciprocal Rank Fusion
- Multi-agent research workflows: plan, search, evaluate, synthesize
- RLM agent for complex analytical tasks via sandboxed Python execution
- Local-first with embedded LanceDB; also supports S3, GCS, Azure
- MCP server for integration with AI assistants (Claude Desktop, etc.)
