# WikiRAG
## Summary
This repository link currently resolves to the Hermit-AI project, focused on fully offline question answering over ZIM archives (such as Wikipedia dumps). The system is designed for local context extraction and injection without cloud APIs or data upload.
## Key Points
- Emphasizes fully offline operation with no external API dependency.
- Uses ZIM archives as large local knowledge sources.
- Implements a multi-stage retrieval/context pipeline for local models.
- Includes setup scripts for quick local installation.
