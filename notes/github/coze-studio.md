# Coze Studio

## Summary
An all-in-one AI agent development platform with visual tools for creating, debugging, and deploying agents. Derived from the Coze platform serving millions of developers, now open-sourced with its core engine.

## Key Points
- Visual no-code/low-code agent development environment
- Core tech: prompt, RAG, plugin, workflow capabilities
- Backend in Golang, frontend in React + TypeScript, microservices with DDD
- Complete app templates and build frameworks included
- Derived from production platform serving tens of thousands of enterprises
