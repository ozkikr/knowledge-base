# BlackHole
## Summary
BlackHole is a macOS virtual audio loopback driver for routing audio between applications with minimal latency. It is widely used for recording system audio, streaming, and DAW-to-call routing setups. The project supports multiple channel-count builds and installation paths including installers and Homebrew.

## Key Points
- Enables low-latency virtual audio routing across apps on macOS.
- Offers 2/16/64/128/256 channel variants and broad sample-rate support.
- Works on modern macOS systems including Apple Silicon.
- Includes extensive user guides for tools like Zoom, Logic Pro, GarageBand, and Audacity.
- Provides installer/uninstaller flows and developer customization options.
