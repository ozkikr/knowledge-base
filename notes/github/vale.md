# Vale

## Summary
A command-line prose linter that brings code-like linting to writing. Fast, cross-platform, and highly customizable with support for many markup formats and a rich extension system for enforcing editorial style guides.

## Key Points
- Markup-aware: understands HTML, Markdown, AsciiDoc, etc. to avoid false positives
- Highly customizable extension system with YAML-based rules
