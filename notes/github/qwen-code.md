# Qwen Code

## Summary
An open-source AI coding agent for the terminal, optimized for Qwen3-Coder. Supports multiple API protocols (OpenAI, Anthropic, Gemini) and offers a free tier with 1,000 requests/day via Qwen OAuth.

## Key Points
- Multi-protocol support: OpenAI, Anthropic, Gemini-compatible APIs, plus Qwen OAuth free tier
- Both framework and Qwen3-Coder model are open-source and co-evolve
- Rich built-in tools (Skills, SubAgents) for full agentic workflow
- Terminal-first with optional VS Code, Zed, and JetBrains IDE integration
- Claude Code-like experience
