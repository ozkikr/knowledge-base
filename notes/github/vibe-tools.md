# vibe-tools

## Summary
vibe-tools gives AI coding agents (like Cursor) an AI team and advanced skills. It provides commands for browser automation, LLM provider integration, file exploration, and more, extending what AI agents can do within coding environments.

## Key Points
- Provides a suite of commands that extend AI agent capabilities in Cursor and similar tools
- Includes browser automation via Playwright/Stagehand integration
- Supports multiple LLM providers for flexible AI model usage
- Offers file reading, code search, and repository exploration tools
- Designed to make AI coding assistants significantly more capable
