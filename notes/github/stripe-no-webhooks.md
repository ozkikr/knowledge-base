# stripe-no-webhooks
## Summary
stripe-no-webhooks is an opinionated Stripe integration library that codifies plans in application code and streamlines billing setup. It handles webhook plumbing, database syncing, and subscription state management while exposing higher-level APIs for credits, wallets, and usage billing. The project targets Next.js plus PostgreSQL workflows with CLI setup helpers.
## Key Points
- Syncs plan definitions from code to Stripe products/prices.
- Automates webhook handling and database synchronization of billing entities.
- Supports subscriptions, seat billing, credits, wallet balances, and metered usage.
- Includes generators for pricing-page UI and callbacks for lifecycle events.
