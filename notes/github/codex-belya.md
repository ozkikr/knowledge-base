# Codex Belya - Voice Coding Agent for Codex CLI

## Summary
Codex Belya is a voice-first AI coding companion for the Codex CLI, inspired by Iron Man's Jarvis. It enables developers to interact with code through voice, orchestrating a multi-agent environment with specialized assistants for Git automation, project analytics, and custom workflows.

## Key Points
- Voice-first coding: dictate multi-step coding requests in multiple languages
- Full git control via voice (status, diff, add, commit, push, branch management)
- Multi-agent orchestration with specialized function tools per agent
- Accessibility-focused: enables voice-first coding for all developers
- Integrates with Codex CLI and IDE extensions
