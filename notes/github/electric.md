# Electric SQL

## Summary
A read-path sync engine for Postgres that handles partial replication, data delivery, and fan-out. Powers real-time sync for apps like Figma and Linear patterns, syncing data out of Postgres into anything via HTTP API with CDN integration.

## Key Points
- Real-time sync engine for Postgres with partial replication via Shapes
- HTTP API integrates with CDNs for highly-scalable data delivery
- Client libraries for TypeScript and framework integrations (React)
- Requires Postgres with logical replication enabled
- v1.0 released March 2025
