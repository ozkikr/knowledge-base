# Chrome MCP Server

## Summary
A Chrome extension-based MCP server that exposes browser functionality to AI assistants like Claude. Unlike Playwright-based tools, it uses your actual Chrome browser with existing login states and configurations, enabling seamless AI-driven browser automation.

## Key Points
- Works with any LLM/chatbot client — model agnostic
- Uses your real browser with existing sessions and login states
- Fully local — pure local MCP server ensuring privacy
- 20+ tools: screenshots, network monitoring, interactive ops, bookmarks, history
- Built-in vector database for semantic search across tab content
- Includes visual editor for Claude Code & Codex
