# Faker (Ruby)

## Summary
Faker is a Ruby library for generating fake data such as names, addresses, phone numbers, emails, and more. It supports 40+ locales for realistic localized data and is widely used for testing, demos, and database population during development.

## Key Points
- Generates fake names, addresses, emails, passwords, phone numbers, and more
- 40+ locales for realistic localized data
- Extensive generator categories: Internet, Date/Time, Person, Finance, Location, etc.
- Simple API: `Faker::Name.name`, `Faker::Internet.email`, etc.
- One of the most popular Ruby gems for test data
