# CrewAI Gmail Automation

## Summary
A multi-agent system using CrewAI that automates Gmail inbox management. It categorizes emails, applies labels, assigns priorities, generates draft responses, sends Slack notifications for high-priority emails, and performs intelligent cleanup.

## Key Points
- AI-powered email categorization (newsletters, promotions, personal, etc.)
- Priority assignment (HIGH/MEDIUM/LOW) with smart classification rules
- Automated draft response generation for important emails
- Slack notifications for high-priority emails
- Intelligent cleanup with YouTube content protection and trash management
