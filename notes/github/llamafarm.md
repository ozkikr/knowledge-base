# LlamaFarm

## Summary
LlamaFarm is an open-source AI platform that runs entirely on local hardware. It provides enterprise AI capabilities including RAG, custom classifiers, anomaly detection, OCR, and multi-model runtime support — all without cloud dependencies or API costs.

## Key Points
- Complete local/edge AI platform — data never leaves your device
- Supports RAG, custom classifiers (SetFit), anomaly detection, OCR, NER
- Multi-model runtime: Ollama, OpenAI, vLLM, or local GGUF models
- Desktop app available for Mac, Windows, and Linux
- Config-driven via `llamafarm.yaml` with CLI, Web UI, and REST API
