# chess-tui — Play Chess from Your Terminal

## Summary
Chess-tui is a terminal chess game written in Rust using ratatui. It supports local 2-player mode, online multiplayer, UCI engine opponents, Lichess integration, and custom skins — all from the terminal.

## Key Points
- Local 2-player, bot (any UCI engine), and online multiplayer modes
- Full Lichess integration for playing rated games from the terminal
- Built with Rust and ratatui for a responsive TUI experience
- Installable via Homebrew, Cargo, or .deb packages
- Supports helper menus and custom board skins
