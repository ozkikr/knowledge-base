# HumanLayer (CodeLayer)

## Summary
CodeLayer is an open-source IDE for orchestrating AI coding agents, built on Claude Code. It provides battle-tested workflows for solving hard problems in large, complex codebases with features like parallel Claude sessions, advanced context engineering, and keyboard-first workflows.

## Key Points
- Superhuman-style keyboard-first IDE for AI coding agents
- Supports running multiple Claude Code sessions in parallel (MULTICLAUDE)
- Advanced context engineering for complex codebases
- Git worktree support and remote cloud workers
- From the team behind "12 Factor Agents" and context engineering talks at YC
