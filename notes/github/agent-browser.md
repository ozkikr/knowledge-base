# agent-browser

## Summary
A headless browser automation CLI designed for AI agents, built by Vercel Labs. Provides a fast Rust CLI with Node.js fallback, using accessibility tree snapshots with refs for element targeting rather than traditional CSS selectors.

## Key Points
- Ref-based element targeting via accessibility tree snapshots
- Fast Rust CLI with Node.js fallback
- Commands: open, snapshot, click, fill, screenshot, close, and more
- Supports traditional CSS selectors alongside ref-based approach
- Installable via npm or Homebrew
