# Voicebun

## Summary
Voicebun is an open-source platform for building and deploying AI-powered voice agents using natural language descriptions. Built with Next.js, Supabase, LiveKit, and OpenAI, it enables creating voice agents for customer service, education, healthcare, and sales use cases.

## Key Points
- Create voice agents by describing them in plain English
- Real-time low-latency voice interactions via LiveKit
- Built with Next.js 14, TypeScript, Tailwind CSS, Supabase, OpenAI
- #6 Product of the Day on Product Hunt
- Use cases: customer service, education, healthcare, sales
