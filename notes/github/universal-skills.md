# Universal Skills

## Summary
Universal Skills brings Anthropic's Claude Code Skills feature to any AI coding agent that supports MCP. Skills are markdown files containing specialized knowledge that can be dynamically loaded into an agent's context only when needed.

## Key Points
- Implements Claude Code-style skills via MCP for any compatible agent
- Supports Codex, Claude Code, OpenCode, and other MCP-compatible agents
- Skills are markdown files loaded on-demand into agent context
- Reverse-engineered from Claude Code's native skills implementation
- Simple installation via `npx universal-skills mcp`
