## Summary
VoiceMode enables natural voice conversations with Claude Code and other MCP-capable agents. It supports local voice services (Whisper STT, Kokoro TTS) for offline use and includes a remote "operator" agent for headless Claude Code instances.

## Key Points
- Available as Claude Code plugin or standalone Python package
- Works offline with local Whisper STT and Kokoro TTS
- Smart silence detection for natural conversation flow
- Cross-platform: Linux, macOS, Windows (WSL), NixOS
- Includes remote operator agent accessible via iOS app or web interface
