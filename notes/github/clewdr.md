# ClewdR

## Summary
ClewdR is a high-performance Rust reverse proxy for Claude (Claude.ai and Claude Code). It serves OpenAI-compatible endpoints and includes a React admin UI for managing cookies and settings, with minimal resource footprint.

## Key Points
- Proxies Claude.ai and Claude Code with OpenAI-compatible API endpoints
- Single static binary for Linux, macOS, Windows, and Android; Docker available
- Web dashboard for live status and hot config reloads
- Manages Claude browser cookies for authentication
- Works with SillyTavern and other OpenAI-compatible clients
