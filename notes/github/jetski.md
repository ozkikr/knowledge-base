# Jetski (HyprMCP)

## Summary
Jetski is an open-source MCP analytics and authentication platform. It deploys an mcp-gateway in front of your MCP server to handle OAuth 2.1, DCR, real-time logs, client onboarding, and analytics with zero code changes to your existing MCP server.

## Key Points
- Solves MCP server installation/setup, authentication, and observability in one tool
- Supports OAuth 2.1 and Dynamic Client Registration (DCR)
- Provides real-time logs, analytics, and prompt visibility
- Zero code changes required to existing MCP servers
- Part of the broader HyprMCP platform
