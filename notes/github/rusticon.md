# Rusticon

## Summary
Rusticon is a mouse-driven SVG favicon editor that runs in the terminal. It creates and edits SVG images for website favicons at 8x8 or 16x16 pixel resolution with 256 colors, written in Rust.

## Key Points
- Mouse-driven TUI with click, drag, flood fill (Shift+Click), and color picker (Ctrl+Click)
- Supports 8x8 and 16x16 pixel canvases with 256 colors
- Can open existing SVG files or resize/resample regular images to favicon size
- Installable via `cargo install rusticon` or pre-built binaries
