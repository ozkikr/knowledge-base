# AI Code Toolkit

## Summary
A toolkit for enforcing team conventions and practices across multiple AI coding agents. Encodes best practices in centralized YAML configs with hierarchical inheritance, replacing growing AGENTS.md/CLAUDE.md files. Works via MCP with Claude Code, Cursor, Gemini CLI.

## Key Points
- Centralized convention encoding in YAML with hierarchical inheritance
- MCP-based plug-and-play for Claude Code, Cursor, Gemini CLI
- Progressive discovery to avoid context window bloat
- Scaffold definitions, patterns, and rules in templates folder
- AGPL-3.0 licensed
