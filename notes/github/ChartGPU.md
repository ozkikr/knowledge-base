# ChartGPU
## Summary
ChartGPU is a WebGPU-powered TypeScript charting library built for high-throughput interactive visualization. It is optimized for very large datasets while maintaining responsive frame rates. The library includes multiple chart types, annotation tooling, and streaming update patterns for dashboard use cases.
## Key Points
- Designed for high-performance rendering (millions of points) with modern GPU acceleration.
- Supports line/area/bar/scatter/pie/candlestick and density rendering modes.
- Provides annotation systems, zooming, crosshairs, and shared GPU device optimizations.
- Includes examples, React bindings, and API docs for production integration.
