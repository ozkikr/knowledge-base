# ChunkForge

## Summary
ChunkForge is a self-hosted document processing platform for converting PDFs to Markdown with semantic chunking, drag-and-drop editing, rich metadata management, and multi-format export. It supports multiple chunking strategies and AI-powered metadata enrichment.

## Key Points
- PDF to Markdown via LlamaParse, MarkItDown, or Docling
- Multiple chunking strategies: recursive, paragraph, heading, semantic, sentence, token, hierarchical
- Visual drag-and-drop chunk editor with card and overlay views
- Custom metadata schemas with AI (OpenAI) enrichment
- Export to JSON, JSONL, CSV, or Markdown with configurable overlap
