# Claude Context (zilliztech)

## Summary
An MCP plugin by Zilliz that adds semantic code search to Claude Code and other AI coding agents. Uses vector database to index entire codebases, enabling cost-effective retrieval of relevant code context.

## Key Points
- Semantic search across millions of lines of code
- MCP-based integration with Claude Code and other agents
- Uses Zilliz Cloud vector database for storage
- Cost-effective alternative to loading entire directories into context
- Also available as VS Code extension
