# MCP Pointer

## Summary
MCP Pointer lets you point at browser DOM elements and make that context available to agentic coding tools (Claude Code, Cursor, Windsurf) via the Model Context Protocol. It combines a Node.js MCP server with a Chrome extension.

## Key Points
- Option+Click (Alt+Click on Windows) to select any DOM element in the browser
- Captures text content, CSS classes, HTML attributes, positioning, and styling
- Dynamic context control: visible-only text, suppress text, adjustable CSS detail level
- Experimental React component detection via Fiber
- Real-time WebSocket communication between browser and AI tools
