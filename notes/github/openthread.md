# OpenThread

Open-source implementation of the Thread networking protocol, released by Google (used in Nest products).

## Summary

OpenThread is Google's open-source implementation of the Thread networking protocol — a mesh networking standard for IoT/smart home devices. It's OS and platform agnostic with a small memory footprint, supporting both SoC and NCP designs. Thread Certified Component implementing Thread 1.4.0 specification.

## Key Points

- **Thread networking layers**: IPv6, 6LoWPAN, IEEE 802.15.4 with MAC security, Mesh Link Establishment, Mesh Routing
- Supports all device roles + Border Router support
- OS/platform agnostic, narrow platform abstraction layer, small memory footprint
- Highly portable — SoC and NCP designs
- Backed by major companies: Google, Apple, Amazon, Samsung, Qualcomm, Nordic Semi, Silicon Labs, Espressif, etc.
- BSD 3-Clause license
- Documentation at openthread.io
