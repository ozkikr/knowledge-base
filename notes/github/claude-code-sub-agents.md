# claude-code-sub-agents
## Summary
This repository offers a large catalog of specialized Claude Code subagents across frontend, backend, infra, data, QA, security, and product functions. It is designed to improve delegation quality by giving each agent a focused role and operational boundary. The collection also supports orchestrated multi-agent workflows for complex tasks.
## Key Points
- Includes dozens of role-specific agent definitions spanning the full software lifecycle.
- Supports automatic task-to-agent routing plus explicit invocation patterns.
- Documents optional MCP server setup to enhance reasoning, docs lookup, and browser/testing capability.
- Provides an organizer/meta-agent pattern for coordinated multi-agent execution.
