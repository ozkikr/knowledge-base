# Yaak
## Summary
Yaak is an offline-first desktop API client focused on a fast, privacy-first developer experience for working with REST, GraphQL, gRPC, WebSocket, and SSE APIs.

## Key Points
- Built with Tauri, Rust, and React/TypeScript for performance and native desktop delivery.
- Supports importing from Postman, Insomnia, OpenAPI, Swagger, and curl workflows.
- Includes auth options (OAuth2/JWT/Basic), encrypted secrets, and OS keychain integration.
- Provides workspaces, environment variables, filesystem mirroring, and plugin extensibility.

## Related
- API testing and debugging tools
- Postman/Insomnia alternatives
- Local-first developer tooling
