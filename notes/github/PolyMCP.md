## Summary
PolyMCP provides a consistent way to expose tools as MCP servers, connect MCP clients, and run agents that orchestrate those tools. Ships in both Python and TypeScript with an Inspector UI and MCP Apps SDK.

## Key Points
- Create MCP servers from plain functions with `expose_tools_http`
- Supports HTTP, stdio, and in-process transports
- Includes agent framework (PolyAgent, UnifiedPolyAgent) with skills management
- Built-in Inspector for testing MCP servers, tools, and prompts
- Integrates with Ollama, OpenAI, and Anthropic for LLM orchestration
