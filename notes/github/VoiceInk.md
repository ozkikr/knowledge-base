# VoiceInk
## Summary
VoiceInk is a macOS voice-to-text application designed for low-latency transcription with local AI models. It emphasizes privacy by keeping processing offline while offering productivity-oriented interaction patterns. The repo includes build instructions and contribution guidance for open-source development.
## Key Points
- Performs near real-time transcription with privacy-first, local processing.
- Supports global shortcuts, app-aware behavior, and custom dictionary/replacements.
- Offers multiple operating modes for dictation and assistant-style interactions.
- Can be installed through releases/Homebrew or built from source.
