# Vibe Kanban

## Summary
A task management and orchestration tool for AI coding agents like Claude Code, Codex, and Gemini CLI. Enables running multiple coding agents in parallel or sequence, tracking task status, and centralizing MCP configuration.

## Key Points
- Orchestrate multiple coding agents in parallel or sequence
- Switch between different coding agents easily
- Track task status with kanban-style board
- Centralized MCP configuration management
- Supports remote SSH project access
