# Noi

## Summary
Noi is a desktop application (8.7k stars) focused on reducing chaos and improving workflow. It provides multi-window management, session isolation, prompt management for AI chats, a built-in terminal, and a CLI command for integration with tools like Claude Code and Codex.

## Key Points
- Interaction-first workflow with parallel multi-window workspaces
- Session isolation keeps contexts clean and separated
- Local-first: history, prompts, and settings stay on-device
- Built-in terminal and `noi` CLI for integration with AI coding tools
- Multiple themes and visual customization options
