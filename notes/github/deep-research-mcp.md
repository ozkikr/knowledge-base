# Deep Research MCP

## Summary
A Python-based MCP server that integrates research providers (OpenAI Deep Research API, Open Deep Research/smolagents) with Claude Code and other AI assistants. Supports both OpenAI Responses API with web search/code interpreter and Chat Completions API for broad provider compatibility.

## Key Points
- MCP server for integrating deep research capabilities into AI assistants
- Supports OpenAI Deep Research and open-source Open Deep Research stack
- Compatible with Perplexity, Groq, Ollama via Chat Completions API
- Configurable via TOML file with clarification pipeline
- Optional query clarification with triage, clarifier, and instruction builder models
