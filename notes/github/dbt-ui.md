# dbt-ui
## Summary
dbt-ui is a web interface for working with dbt-core projects using a modern frontend and FastAPI backend. It combines editing, lineage visualization, command execution, and Git operations in one workspace. The project is distributed with Docker-first setup to simplify local deployment.
## Key Points
- Provides project browsing and Monaco-based editing for dbt SQL/YML artifacts.
- Includes lineage graph visualization to inspect model dependencies.
- Adds built-in Git workflows like staging, committing, pushing, and branch switching.
- Supports local repository mounts and optional MetaDV-oriented modeling features.
