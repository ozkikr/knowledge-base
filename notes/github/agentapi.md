# AgentAPI

## Summary
AgentAPI by Coder provides an HTTP API to control coding agents including Claude Code, Goose, Aider, Gemini, Amp, Codex, and more. It enables building unified chat interfaces, agent-to-agent control via MCP, and automated workflows like PR review submission.

## Key Points
- HTTP API wrapper for 10+ coding agents (Claude Code, Aider, Goose, Gemini, Codex, etc.)
- Enables building unified chat UIs for multiple agents
- Can be used as MCP server backend for agent-to-agent communication
- Single binary installation, cross-platform
- Useful for CI/CD integration and automated code review
