# claude-code
## Summary
Claude Code is Anthropic's terminal-native agentic coding tool for codebase-aware development workflows. It supports natural-language task execution, code explanation, and git-related operations across local projects. The repository includes installation paths, plugin support, and references to official product documentation.
## Key Points
- Positioned as an agentic coding assistant for terminal and IDE-adjacent workflows.
- Provides official install methods for macOS, Linux, and Windows.
- Includes a plugin ecosystem for extending commands and agent behaviors.
- Offers issue reporting, community support channels, and docs links.
- Documents data usage/privacy safeguards for user feedback and usage telemetry.
