# mcp-server-motherduck
## Summary
This repository provides a local MCP server for DuckDB and MotherDuck-backed workflows. It allows AI assistants to query and interact with analytical data through a local, configurable server.

## Key Points
- Connects assistants to local DuckDB and MotherDuck environments.
- Supports SQL querying and catalog exploration workflows.
- Designed for local/self-hosted deployments with read/write options.
- Useful for analytics-focused assistant integrations.
