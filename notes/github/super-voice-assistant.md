# Super Voice Assistant

## Summary
A macOS voice assistant with global hotkeys for speech-to-text transcription (offline via WhisperKit/Parakeet or cloud via Gemini API), screen recording with visual context transcription, and text-to-speech via Gemini Live. Fast, accurate, and simple.

## Key Points
- Cmd+Opt+Z for local offline transcription (WhisperKit or Parakeet), Cmd+Opt+X for cloud (Gemini)
- Streaming text-to-speech with Gemini Live API (Cmd+Opt+S)
- Screen recording + video transcription using Gemini 2.5 Flash with visual context
- Automatic text pasting at cursor position after transcription
- Requires macOS 14.0+, supports visual disambiguation for ambiguous names
