# cigs - Composable AI Functions for TypeScript

## Summary
TypeScript library for creating chainable AI functions using OpenAI's structured outputs. Call functions with natural language and get type-safe responses back in specified Zod schemas.

## Key Points
- Composable/chainable AI function snippets ("lodash for AI functions")
- Built on OpenAI structured outputs and function calling
- Type-safe with Zod schema definitions for input/output
- Dynamic function responses: call with natural language, specify output schema at runtime
- Inspired by instructor, askmarvin, and fructose but TypeScript-native
