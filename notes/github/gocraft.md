# GoCraft - Go Backend Generator

## Summary
GoCraft is a production-ready Go backend generator that scaffolds modern microservices with preferred frameworks, databases, and features. It generates complete, well-structured Go applications with built-in auth, validation, security, monitoring, and documentation.

## Key Points
- Generates production-ready Go backends in seconds with modular architecture
- Built-in authentication, validation, and security best practices
- Supports gRPC, Docker, Swagger, and monitoring out of the box
- Clean architecture: API routing, middleware, handlers, models separated
- Requires Go 1.23+ and PostgreSQL
