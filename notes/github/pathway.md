# Pathway

## Summary
Pathway is a Python ETL framework for stream processing, real-time analytics, LLM pipelines, and RAG. It features a Python API backed by a scalable Rust engine based on Differential Dataflow, enabling incremental computation with multithreading and distributed processing.

## Key Points
- Python API with Rust engine for high-performance stream processing
- Same code works for batch, streaming, and replay scenarios
- Built-in support for LLM pipelines and RAG use cases
- Incremental computation via Differential Dataflow
- Easy deployment with Docker and Kubernetes
