# pgEdge Postgres MCP Server

## Summary
A PostgreSQL MCP (Model Context Protocol) server with a natural language agent CLI and Web UI. It enables AI agents to interact with PostgreSQL databases through MCP, supporting multiple database configurations, encryption, and authentication.

## Key Points
- MCP server for PostgreSQL with natural language query capabilities
- Includes both CLI chat client (Go) and web client
- Supports Claude Desktop integration
- Multi-database configuration with encryption and authentication
- Docker deployment support; beta software
