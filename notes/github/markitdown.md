# MarkItDown - File to Markdown Converter

## Summary
MarkItDown is a lightweight Python utility by Microsoft for converting various file formats to Markdown for use with LLMs and text analysis pipelines. It preserves document structure (headings, lists, tables, links) and supports PDF, PowerPoint, Word, Excel, images, audio, HTML, and more.

## Key Points
- Converts PDF, Office docs, images (OCR), audio (transcription), HTML, CSV, JSON, XML
- Focuses on preserving structure as Markdown for LLM consumption
- Offers MCP server for integration with Claude Desktop and similar tools
- Available as Python library and CLI
- Comparable to textract but Markdown-focused
