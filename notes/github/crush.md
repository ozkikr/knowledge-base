# Crush

## Summary
A glamorous terminal-based agentic coding tool by Charmbracelet. Connects your tools, code, and workflows to your LLM of choice with a beautiful TUI. Supports multi-model switching, MCP servers, LSP integration, and session-based context management.

## Key Points
- Multi-model support with mid-session LLM switching while preserving context
- LSP-enhanced for additional code context
- Extensible via MCP servers (HTTP, stdio, SSE)
- Cross-platform: macOS, Linux, Windows, Android, FreeBSD, etc.
- Built on the Charm ecosystem powering 25k+ applications
- Session-based with multiple work contexts per project
