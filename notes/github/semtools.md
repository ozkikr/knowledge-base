# SemTools

## Summary
A high-performance CLI tool built in Rust for document processing and semantic search. Provides parse, search, ask, and workspace commands for working with document collections using LlamaParse and multilingual embeddings.

## Key Points
- `semtools parse` for document parsing (PDF, DOCX, etc.) via LlamaParse API
- `semtools search` for local semantic keyword search with multilingual embeddings and cosine similarity
- `semtools ask` provides an AI agent with search/read tools over document collections
- Built in Rust for speed; installable via npm or cargo
- Workspace management for accelerating search over large collections
