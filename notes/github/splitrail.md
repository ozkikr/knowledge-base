# Splitrail

## Summary
Splitrail is a fast, cross-platform, real-time token usage tracker and cost monitor for CLI coding agents including Gemini CLI, Claude Code, Codex CLI, Cline, Roo Code, GitHub Copilot, and more. From the team behind Piebald.

## Key Points
- Tracks token usage and costs across multiple AI coding tools in real-time
- Supports Gemini CLI, Claude Code, Codex CLI, Cline/Roo Code/Kilo Code, GitHub Copilot, OpenCode
- Splitrail Cloud for cross-machine usage aggregation and safe-keeping
- MCP server mode with tools for querying stats, model usage, and cost breakdowns
- VS Code extension available alongside CLI
