# kit
## Summary
kit is a production-oriented code intelligence toolkit for repository mapping, symbol extraction, and code search workflows. It targets AI developer tools and agents through Python APIs, CLI interfaces, and MCP-based integrations. The project also includes utilities for PR review, summarization, and codebase-aware automation.
## Key Points
- Supports file tree, symbol extraction, usages, dependency analysis, and structured search.
- Provides Python SDK plus CLI commands for local and remote repository analysis.
- Includes MCP server support and integrations for assistant-driven development workflows.
- Extends beyond retrieval with PR review/summarization and commit-message utilities.
