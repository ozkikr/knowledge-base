# Dotprompt - Executable GenAI Prompt Templates

## Summary
Dotprompt is Google's executable prompt template file format for Generative AI. It extends Handlebars templating with GenAI-specific features and is designed to be language/provider agnostic, with implementations in JS/TS, Python, Go, Rust, and Java.

## Key Points
- Executable `.prompt` file format extending Handlebars for GenAI
- Language and model-provider agnostic design
- Libraries for JS/TS, Python, Go, Rust, and Java
- IDE extensions for VS Code, JetBrains, and Vim
- Supports structured output schemas and model configuration in frontmatter
