# awesome-claude-code-subagents
## Summary
This repository is a curated collection of Claude Code subagents organized by role and domain. It packages a large catalog of specialized agent definitions spanning core development, infrastructure, quality/security, and data/AI workflows. The project supports plugin-based and script-based installation patterns.
## Key Points
- Curates 100+ specialized subagents for distinct engineering and operational tasks.
- Organizes content into category plugins (core dev, language specialists, infra, QA/security, etc.).
- Supports installation through Claude plugin marketplace, manual copy, and interactive installers.
- Includes meta/orchestration helpers intended to coordinate other installed agents.
- Functions as an "awesome list" style discovery hub for Claude Code agent ecosystems.
