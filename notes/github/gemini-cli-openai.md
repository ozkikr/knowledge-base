# Gemini CLI OpenAI Worker

## Summary
A Cloudflare Workers project that transforms Google's Gemini models into OpenAI-compatible API endpoints. Uses OAuth2 authentication (no API keys needed), leveraging Google's free tier through the Code Assist API.

## Key Points
- Drop-in OpenAI API replacement for Gemini models (2.5 Pro, Flash, Flash Lite)
- Vision support, tool calling, and advanced reasoning/thinking capabilities
- Free tier access through Google's Code Assist API
- Smart token caching with KV storage
- Supports fake and real thinking modes for Gemini 2.5 models
