# Magika - AI-Powered File Type Detection

## Summary
Magika is Google's AI-powered file type detection tool using deep learning for accurate content type identification. It uses a lightweight model (~few MBs) achieving ~99% accuracy across 200+ content types, processing hundreds of billions of samples weekly at Google for Gmail, Drive, and Safe Browsing.

## Key Points
- Custom optimized deep learning model for file content type detection in milliseconds on CPU
- Trained on ~100M samples across 200+ content types (binary and text)
- Available as CLI (Python), web API, and libraries in Python, JS, Go, and Rust
- Used at scale in Google products and integrated with VirusTotal and abuse.ch
- Published at IEEE/ACM ICSE 2025
